<p><b>My top three:</b> </p>

<ul>
    <li><a href="http://www.aw.com/cp/campione-walrath.html"
        target="_parent">The Java Tutorial</a> - This is probably
        the <i>best</i> book to get started with, and continue to
        use to delve into new parts of Java that maybe you
        haven't used yet. Highly recommended. It's also available
        in hypertext format online at <a
        href="http://www.javasoft.com:80/nav/read/Tutorial/index.html"
        target="parent">http://www.javasoft.com:80/nav/read/Tutorial/index.html</a>
    </li>
    <li><a href="http://www.sun.com/books/books/Geary/Geary.html"
        target="_parent">Graphic Java</a> - This is a fantastic
        book focusing on the Java AWT. Another one of my
        favorites. </li>
    <li><a href="http://www.ora.com/www/info/java/" target="_top">O'Reilly
        and Associates &quot;Java in a Nutshell&quot;</a> - The
        K&amp;R of Java. This is a must have reference book for
        Java programmers. </li>
</ul>
