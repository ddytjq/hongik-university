<center><h2>Section 2.1: "Hello, Dave"</h2></center>


<p>
Before diving right into the nitty-gritty of C++ language details,
let's get started with a full-fledged C++ program!  The idea of this
program is to introduce you to the overall structure of a C++ program and give you the flavor of C++.  
</p>

<p>
Here, then, is an entire C++ program:

<pre>
//include this file for cout
#include &lt;iostream.h&gt;

int main() {

  //print out the text string, "Hello, World!"
  cout &lt;&lt; "Hello, World!" &lt;&lt; endl;

  return 0;

}
</pre>
</p>

<p>
To prepare for the next section, create a new text file and
save this code into the file.  
If you are on a <a href="glossary.htm#unix">unix</a> machine,
save the file with the filename <b>hello.C</b> (make sure
it ends with <b>.C</b>, not <b>.c</b>!).
If you are on a <a href="glossary.htm#windows">Windows</a> machine,
save the file with the filename <b>hello.cpp</b>.
</p>

<p>
You're now ready to compile and run your program, covered
on the next page.
</p>
