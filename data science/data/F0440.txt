<h2>
<font color="#000000">Lesson 6 : Working with Variables</font></h2></center>
<b><font color="#000000">6.1&nbsp; Assigning Values to Variables</font></b>
<p><font color="#000000">After declaring various variables using the Dim
statements, we can assign values to those variables.</font>
<br><font color="#000000">The general format of assignment is</font>
<blockquote><font color="#000000">Variable=Expression</font></blockquote>
<font color="#000000">The variable can be a declared variable or&nbsp;
a control property value. The expression could be a mathematical expression,
a number, a string, a boolean value(true or false) and etc. The following
are some examples:</font>
<p><font color="#000000">firstNumber=100</font>
<br><font color="#000000">secondNumber=firstNumber-99</font>
<br><font color="#000000">userName="John Lyan"</font>
<br><font color="#000000">userpass.Text = password</font>
<br><font color="#000000">Label1.Visible = True</font>
<br><font color="#000000">Command1.Visible = false</font>
<br><font color="#000000">Label4.Caption = textbox1.Text</font>
<br><font color="#000000">ThirdNumber = Val(usernum1.Text)</font>
<br><font color="#000000">total = firstNumber + secondNumber+ThirdNumber</font>
