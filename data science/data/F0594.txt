This is the main interface of VB and it provides all the facilities for creating a visual interface, writing the underlying code, debugging the code, and creating an executable file. The most important menu item is the Help option which provides access to the <A HREF="lt1-hlp.htm">online help</A>


<H4>The More Important Option on the Main Menu</H4>

<H5>The View Menu</H5>

<B>The New Procedure Option</B> allows the user to create a new section of code (either a function or a procedure) that can be associated with <B>any</B> event. It is important for the program to think in terms of writing reusable code. The aim is to reduce the code written in each control event to a minimum by calling procedures and functions.
