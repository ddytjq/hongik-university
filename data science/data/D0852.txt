		    <a href="/codecentral/code_profile.jsp?category_ID=4&code_ID=68">
		    MsgWindow</a></b><br>
		    MsgWindow (message window) is basically a Splash Screen to use during program loading to display program information to the user. Its a very small class that will allow you to use a simple method println(String xyz), instead of the standard System.println(). Any String passsed to this method will appear in the Message Window.  The JPanel object has a simple borderlayout, the borders can containcustom images held in the /mwGfx/ folder.
Here are some features of the window :: The graphics for the app are customisable - Text is shown in a textarea on a JPanel - Each message is shown on a new line - Message count and screen resolution info is displayed - There are keyboard commands to clear the textarea and close the window - Timestamp feature places the current time of a message on-screen.

		    A new window class, developed by extending the JWindow (swing component).
The Window will support any java component as its content, and allows developers to cutomise all aspects, including : titleBar text, colors, font, buttons, border size, colors
 window content, size and location.  A number of Icon sets are available for the window in the public depository at www.bjug.net, Its 'skinnable' nature is similar to some of the linux window managers.
  The XenWindow API was developed as a helper class for another open source project at bjug.net.  The result is a flexible API which can be adapted for many applications/applets
