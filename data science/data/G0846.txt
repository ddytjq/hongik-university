<font size="4"><b>CosmoPlayer Installation Notes</b><br></font>
<font size="2"><a href="/wgbh/nova/universe/tour_prep.html#tour1">back to Preparing Yourself and Your Computer for VRML</a></font><br><br>

<b>For PCs</b>

<UL>
<LI>
Although Windows 98 is not listed, we have had no problems running CosmoPlayer
under Windows 98. We have not tried Windows NT or 2000.<br><br>


<LI>
The installation program doesn't recognize many of the later versions of
internet browsers. If your version of Netscape Navigator or Internet Explorer
is not listed as a "supported browser", be sure to check "Other unsupported
browsers". The installation program will then find the newer versions and
allow you to install the plug-ins for those versions.<br><br>


<LI>
Cosmo Software has a support page at <A HREF="http://www.cai.com/cosmo/html/support.htm">http://www.cai.com/cosmo/html/support.htm</A></LI>
</UL>
<br>



<b>For Macs</b>

<UL>
<LI>
??? I haven't installed CosmoPlayer on a Mac.<br><br>

<LI>
Cosmo Software has a support page at <A HREF="http://www.cai.com/cosmo/html/support.htm">http://www.cai.com/cosmo/html/support.htm</A></LI>
</UL>
<br>

<b>For SGIs</b>

<UL>
<LI>
Be sure to download and install CosmoPlayer 2.1. The "Local Universe in 3-D" VRML
world will not work with CosmoPlayer 1.x that comes standard with SGI machines.<br><br>


<LI>
The Cosmoplayer 2.1 plug-in installs itself in /var/netscape/communicator/plugins.
There are some versions of netscape that install themselves in places like
/usr/bin/X11/netscape. Those installations will not find the CosmoPlayer
2.1 plug-in unless your system administrator sets it up.</LI>
</UL>
