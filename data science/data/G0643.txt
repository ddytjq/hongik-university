        <H4>THIS MONTH'S QUIZ</H4></CENTER>

1. A certain board-game uses a number of circular counters, all of the same diameter, with their front faces divided into three equal sectors. Each sector is coloured either red, green or blue, with no two adjacent sectors the same colour. How many distinguishable counters can there be?<p>
2. Another game uses similar counters, with the same colours available, but divided into <b>four</b> equal sectors. How many distinguishable counters can there be?<p>
3. Yet another game has its counters divided into <b>five</b> equal sectors, with the same available colours. How many distinguishable counters can there be in this game?<br>

<hr>
4. (<i>Bonus question. Not part of the quiz, but could earn you a 'Certificate of Distinction'</i>.) How many distinguishable counters can there be with <b>six</b> equal sectors?
