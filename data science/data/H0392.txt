<span class="xlna">How To Chat</span>
<span class="nnv">
<p>To start chatting, enter your user name (any nickname) and click "connect."  Your user name must be a unique chat user name and is valid for your current chat session.
<p>Chat messages are displayed in the chat message area.  To add to the conversation, type in your message in the text box below the messages area.  By default, your message is sent to all users.  To send a message to a particular user, highlight the user from the from the users list on the right. Type the message in the text box, click the "send" button and "Whisper to Member."
<p>If you wish to carry on a private conversation with another user, highlight the user from the users list on the right and select the "Person to Person chat button".  A pop-up box will appear similar to the chat messages area.  Enter in your message in the text box and click "send".  
<p>If someone initiates a private conversation with you, a dialog box will be displayed.  It will identify the message and provide four options from which to choose:
<ul><li>Chat: Selecting this button accepts the private conversation.
<li>Cancel: Does not accept the chat.
<li>Ignore Member: Will not accept the chat, and refuse any correspondence for this person, either in the room or private.
<li>Ignore Member's Domain:  Choose this option if you have already ignored a member once and the member changed nickname to avoid the ignore, and attempting to send another messages.  This will block any messages from this person or from this person's Internet Service Provider (ISP).
</li></ul>
<p>To screen out obnoxious users, highlight the user from the users list on the right, and select the "Ignore User" button.
<p>To end your chat session, select "Quit" button.
