	<B>Command buttons</b>
</font>
</p>


<p align="left">
<font face="tahoma">
<B>Command buttons</b> are very common in Windows applications themselves, you will have trouble finding one that doesn't have some. Command buttons work
simple, you click, they do, but of coarse not w/o some codes which we will be getting into soon... 
</p>

<p align=center>
<img src="http://www.angelfire.com/ny3/visualbasic/images/vbbut.jpg"
</p> 

<p align="left">
<font face="tahoma">
<font color="blue">Command button properties:</font>
</p>
<p align="left">
<font face="tahoma">
<font color="red">
Cancel</font> - Determines if the command button runs its click event code if the user presses the Esc key<Br>
<Br>
<font color="red">
Caption</font> - The caption determines what the button will say (this goes for almost all controls)<Br>
<Br>
<font color="red">
Default</font> - Determines if the button will do something if Enter is pressed, even if another control has focus<Br>
<Br>
<font color="red">
Enabled</font> - Determines if the button is useable or not<Br>
<Br>
<font color="red">
Font</font> - The controls font<Br>
<Br>
<font color="red">
Picture</font> - Determines if the button has a pic in it<Br>
<Br>
<font color="red">
Style</font> - Determines if the button is standard or not, it's best that you always put it as "Graphical"<Br>
<Br>
<font color="red">
TabIndex</font> - What order the button gets focus by pressing the Tab key<Br>
<Br>
<font color="red">
TabStop</font> - Determines if the button gets focused on w/ the tab button<Br>
<Br>
<font color="red">
ToolTipText</font> - Determines the text that will be the tooltip (only in runtime mode)<Br>
<Br>
<font color="red">
Visible</font> - Determines if the control is visible at run time or not<Br>
</p>


<p align=Center>
<font face="tahoma">
<font color="blue">
<img src="http://www.angelfire.com/ny3/visualbasic/images/tip.jpg"> <B>Tip:</b> The <B>Focus</b> is what control is currently being used, or the one that's currently selected.
Add multiple controls to your project, spread them out on the form, run your project and
then on the keyboard press tab and then you will see what the focus is.
</p>

<p align=Center>
<font color="blue">
<font face="tahoma">
<img src="http://www.angelfire.com/ny3/visualbasic/images/tip.jpg"> <B>Tip:</b> The <B>ToolTipText</b> is just a few words you tip the user w/. To give you an example, in your project
add a Command button, go to its properties and then where it says "ToolTipText" write "Hi." 
Now run your project and put the mouse pointer over the Command button...</font>
</p>
</font>
</font>
