<H1>COBE Software Documents</H1>

<H1>
<HR WIDTH="100%"></H1>

<P>Users of the COBE&nbsp;Guest Investigator Software (CGIS)&nbsp;should
be familiar with the following documents:</P>

<UL>
<LI><B>CGIS Software User's Guide</B><I> (version 2.2)</I></LI>

<LI><B>CGIS Software Catalog</B><I> (version 1.0)</I></LI>
</UL>

<P>Equivalent Postscript, ASCII, and WordPerfect versions of these documents
are available in the <A HREF="ftp://rosette.gsfc.nasa.gov/pub/cobe-gi/doc">/pub/cobe-gi/doc
directory</A> on <I>rosette.gsfc.nasa.gov</I>. (There you can also find
a <I>COBE Guest Investigator Handbook</I>, but this is now largely obsolete.)
The directory can be reached by conventional anonymous ftp or in your Web
browser at <A HREF="ftp://rosette.gsfc.nasa.gov/pub/cobe-gi/doc">ftp://rosette.gsfc.nasa.gov/pub/cobe-gi/doc</A>.
</P>

<P>Extensions to the file names indicate the format:<I> .ps</I> for Postscript;
<I>.asc </I>for ASCII; and <I>.wp</I> for WordPerfect. The <I>Software
Catalog</I> appears in a single file (called &quot;csc94121&quot;) of each
type, but the <I>User's Guide</I> is divided into separate files by chapter
or major section. Examples of <I>User's Guide</I> file names are &quot;user2210&quot;
for (version 2.2) Chapter 1.0 and &quot;user22xa&quot; for Appendix A.
</P>

<P>If you have trouble printing the Postscript files, try downloading them
in binary mode and deleting the first line of each file. The WordPerfect
files must be transferred in binary mode.</P>
