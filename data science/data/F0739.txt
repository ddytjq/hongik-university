	<tr><td colspan="2">
 <img src="http://www.vbxtras.com/gifs/3555.gif " align="right" hspace="10" valign="top">
<font size=2>The Amyuni Client/Server PDF Printing Solutions are a series of four products based on the Amyuni PDF Converter and Amyuni PDF Creator products. These Client/Server solutions use either TCP/IP communications or Email between the sender and the receiver to deliver Web server reports in PDF format, archive or print documents generated by clients, print documents to the PDF Converter printer hosted on a server and integrate with the .NET platform. <br><br>

The <b>Web Server PDF Printing Solution</b> allows clients to connect to a web server and request a document or report in PDF format.  The web server uses the PDF Converter or PDF Creator to generate the requested report in PDF format and sends the document back to the client using either email or TCP/IP.  The client can choose to print, view or save the PDF file using either the PDF Creator viewer or Adobe Acrobat Reader.
Using the <b> PDF Document Archiving and Printing Server </b> the client can convert their system files to PDF documents.  The generated PDF file is automatically sent to a file server which will either store the file, print the file to a specific printer or do some other processing with the file. The <b> Application Server </b> allows the client to run an application hosted on a server and print a document to the PDF Converter printer also hosted on the server.  The PDF Converter sends the PDF data back through TCP/IP to the client that will be configured to either print, save or view the PDF file on his/her local drive or printer. And, the <b>.NET PDF Client/Server Tools </b> assists developers in integrating these PDF Functions into applications on the .NET framework.
