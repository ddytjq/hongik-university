<h2>
<b><font color="#990000">Sample 1: A Smart Audio Player</font></b></h2>

<p><br><font color="#000099">Brief Description:</font>
<br>This player enable you to play midi and wave files in your hard drives
or floppy A or CDROM.
<br>It allows you to change drives and search for the particular sound
file that you wish to play.
<br>In this program, you have to add a Combo box, a List Box, a Text Box,
a Drive List Box,
<br>a command button you label as "PLAY", another command button you label
as "STOP", and one
<br>more command button you label as "EXIT". You have to add in MMControl
and make it invisible.
<br>This is programmed in VB4.0, it's working fine. However, I haven't
try it out in VB6.0.

<p><font color="#3333FF">Sub Combo1_Change( )</font>
<br><font color="#3333FF">if ListIndex=0 Then</font>
<br><font color="#3333FF">File1.Pattern=("*.wav")</font>
<br><font color="#3333FF">ElseIf ListIndex=1 Then</font>
<br><font color="#3333FF">File1.Pattern=("*.mid")</font>
<br><font color="#3333FF">Else</font>

<br><font color="#3333FF">End If</font>

<p><font color="#3333FF">Else</font>

<p><font color="#3333FF">End If</font>

<br><font color="#3333FF">MMControl1.Command="Close"</font>
