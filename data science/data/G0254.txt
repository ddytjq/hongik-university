<h1> FIRAS Explanatory Supplement </h1>
<hr>

Version 2.1 (dated 14 July 1995) is now on line.  It contains detailed
descriptions of the FIRAS instrument, its calibration and operation,
and the FIRAS Project Data Sets.
<hr>

<p>
You may download the Explanatory Supplement or use this Web page to browse it.
<a
href="http://nssdca.gsfc.nasa.gov/anon_dir/cobe/project_data_sets/firas/doc/firas_explanatory_supplement.ps">
A PostScript version of the entire document </a> (160 pages;
Appendices excluded) is provided for convenient printing.  
The document is also available by ftp in unprocessed form
(LaTeX manuscript with Encapsulated PostScript figures and assorted ASCII and
PostScript files).  For advice on which files to download, please 
<a
href="http://nssdca.gsfc.nasa.gov/anon_dir/cobe/project_data_sets/firas/doc/areadme.doc">
README.</a> 
Proceed as follows to retrieve the files by ftp:
<PRE>
    % ftp nssdca.gsfc.nasa.gov	  # type this command
      ...			  #  <ftp responds>
    Name: anonymous		  # type "anonymous"
    Password:			  # type your e-mail address
    ftp> cd cobe		  # change to cobe directory
    ftp> cd project_data_sets	  # change to sub-directory
    ftp> cd firas		  # change to sub-directory
    ftp> cd doc			  # change to sub-directory
    ftp> ls			  # list files
    ftp> bin			  # binary mode (required for tar file)
    ftp> get [fname]		  # retrieve file called [fname]
    ftp> quit			  # end session
</PRE>


<p>
To browse individual sections of the Explanatory Supplement, click on
the section names below.  Appendices A - F are manuscripts of journal
or conference proceedings articles in PostScript form.  Appendices G -
J are flat ASCII files.  Appendix K consists of a set of color
PostScript files of which relatively compact JPEG versions are provided
below for quick viewing.
