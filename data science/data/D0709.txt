<h2>
Your first JSP</h2>
JSP simply puts Java inside HTML pages.&nbsp; You can take any existing
HTML page and change its extension to ".jsp" instead of ".html".&nbsp;
In fact, this is the perfect exercise for your first JSP.
<p>Take the HTML file you used in the <a href="Getfamiliar.html">previous
exercise</a>.&nbsp; Change its extension from ".html" to ".jsp".&nbsp;
Now load the new file, with the ".jsp" extension, in your browser.
<p>You will see the same output, but it will take longer!&nbsp; But only
the first time.&nbsp; If you reload it again, it will load normally.
<p>What is happening behind the scenes is that your JSP is being turned
into a Java file, compiled and loaded.&nbsp; This compilation only happens
once, so after the first load, the file doesn't take long to load anymore.&nbsp;
(But everytime you change the JSP file, it will be re-compiled again.)
<p>Of course, it is not very useful to just write HTML pages with a .jsp
extension!&nbsp; We now proceed to see what makes JSP so useful.
