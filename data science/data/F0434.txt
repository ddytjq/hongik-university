The User Interface: the Form Window</H2>
<IMG SRC="images/vb-form.gif">
<P>
A form is a &quot;container&quot; for custom controls. Controls can be &quot;drawn&quot; anywhere within the form. When compiled a form becomes a ordinary window that can be manipulated within the Microsoft Windows environment. It is the interaction between the forms, controls and the underlying code that creates a working window application.
<P>
<I><B>Miscellaneous Information</B></I><BR>
The dots within the form, form a grid that is useful for aligning controls. The size of the grid and whether controls are aligned at all can be control from the Options|Environment.. menu.
<P>
Also like all controls the Property <I>Name</I> is set to a default. In this case Form1.
