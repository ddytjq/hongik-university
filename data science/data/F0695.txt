	<font size=4><div class=cutline>Product Review:</div></font><BR>
	<font size='-1'><h4><i>Script</i>ing More Power and Versatility</h4>
If you do any kind of web development, you quickly become aware that most projects require a variety of different languages and scripting support. Like the majority of us, you may still rely on Notepad for most of your regular script writing and maintenance, and possibly a more flexible text editor for more complex tasks.  This does the job but does not offer the flexibility and ease of use that we get in the powerful code editors in IDEs such as Visual Basic. You lack the productivity features like syntax coloring and code completion that you're used to having in those environments.
<p>
But Primal<i>script</i> brings you a versatile script editor with support for a large variety of scripting languages, including VBScript, JavaScript, Perl, HTML, SQL, DOS batch files, and XML, to name a few. It even supports languages like C++, VB.NET, and C#.
<h4>Thinking Outside the <i>Script</i></h4>
Primal<i>script</i> brings you more than just a powerful editing environment for your scripting needs. For instance, it offers powerful editing, navigation and search capabilities, Windows script host support, server-side web development, source file control integration using MSSCCAPI, and a built-in HTML browser. Primalscript also lets you easily customize its features and interface. You can run scripts in the debugger and listen to debugging messages over TCP/IP. You can format your code and send it to the printer for easy documentation. The Workspace Nexus lets you create projects and manipulate files and offer many productivity features.
<p>
Armed with this powerful tool, you will find your scripting jobs much easier and attain a level of productivity that was never before possible.  Order Primal<i>script</i> today!
<ul>
<li>Versatile scripting environment supporting a wide variety of languages.
<li>Built-in HTML browser and language specific syntax coloring.
<li>Integrated debugging environment.
<li>Powerful productivity and project management features.
</ul></font>
