<p><font face="Arial, Helvetica" size="+1"><b>Simple INI file class</b></font></p>
<p><i>By <a href="/php-bin/authorbio.php?authorid=31&associd=61&assoctype=2">Sam Huggill</a></i></p>
<p><font face="Arial, Helvetica" size="+1"><b>Introduction</b></font></p>

<p>INI Files now seem to have been completely removed
by the registry, but sometimes they are a much better option. This can be becuase of
safety for the registry, and allowing the user to more easily view and change settings.
This simple wrapper class wraps up a few simple INI file functions into a reusable, drop
in class for instant INI file use. It supports the following methods/properties:</p>

<ul>
<li>CreateINI - Creates a new INI file</li>
<li>GetFile - Returns the value specified in the arguements passed</li>
<li>INIFile - Gets/Sets the current INI file path</li>
<li>MakePath - A private function used to create the ini file</li>
<li>WriteFile - Writes a value to the INI file</li>
</ul>

<p>You will probably want to extend its functionality by adding more INI file functions.
Anyway, the class is based around two Win API declares: WritePrivateProfileString and
GetPrivateProfileString. Both require a bit of extra work to make them effective, but
overall they are quite simple to use. A good alternative to the Windows Registry.</p>

<p>To get the class working just use:</font></p>

</font><pre>
<font color="#00007F">Private</font> m_cIni <font color="#00007F">As</font> CIniFile
<font color="#00007F">Private</font> <font color="#00007F">Sub</font> Form_Load()
  <font color="#00007F">Set</font> m_cIni = <font color="#00007F">New</font> CIniFile
<font color="#00007F">End</font> <font color="#00007F">Sub</font>
</pre><FONT SIZE="-1" FACE="verdana, arial, helvetica">

<p>
Add a new class module (CIniFIle) and copy the following
code into it:</p>

</font><pre>
<font color="#00007F">Option</font> <font color="#00007F">Explicit</font>

<font color="#007F00">'// <font color="#00007F">Private</font> member that holds a reference to</font>
<font color="#007F00">'// the path of our ini file</font>
<font color="#00007F">Private</font> strInI <font color="#00007F">As</font> <font color="#00007F">String</font>

<font color="#007F00">'// Win API Declares</font>
<font color="#00007F">Private</font> <font color="#00007F">Declare</font> <font color="#00007F">Function</font> WritePrivateProfileString _
  <font color="#00007F">Lib</font> &quot;kernel32&quot; <font color="#00007F">Alias</font> &quot;WritePrivateProfileStringA&quot; _
 (<font color="#00007F">ByVal</font> lpApplicationName <font color="#00007F">As</font> <font color="#00007F">String</font>, _
  <font color="#00007F">ByVal</font> lpKeyName <font color="#00007F">As</font> <font color="#00007F">Any</font>, _
  <font color="#00007F">ByVal</font> lpString <font color="#00007F">As</font> <font color="#00007F">Any</font>, _
  <font color="#00007F">ByVal</font> lpFileName <font color="#00007F">As</font> <font color="#00007F">String</font>) <font color="#00007F">As</font> <font color="#00007F">Long</font>

<font color="#00007F">Private</font> <font color="#00007F">Declare</font> <font color="#00007F">Function</font> GetPrivateProfileString _
  <font color="#00007F">Lib</font> &quot;kernel32&quot; <font color="#00007F">Alias</font> &quot;GetPrivateProfileStringA&quot; _
 (<font color="#00007F">ByVal</font> lpApplicationName <font color="#00007F">As</font> <font color="#00007F">String</font>, _
  <font color="#00007F">ByVal</font> lpKeyName <font color="#00007F">As</font> <font color="#00007F">Any</font>, _
  <font color="#00007F">ByVal</font> lpDefault <font color="#00007F">As</font> <font color="#00007F">String</font>, _
  <font color="#00007F">ByVal</font> lpReturnedString <font color="#00007F">As</font> <font color="#00007F">String</font>, _
  <font color="#00007F">ByVal</font> nSize <font color="#00007F">As</font> <font color="#00007F">Long</font>, _
  <font color="#00007F">ByVal</font> lpFileName <font color="#00007F">As</font> <font color="#00007F">String</font>) <font color="#00007F">As</font> <font color="#00007F">Long</font>

<font color="#00007F">Private</font> <font color="#00007F">Function</font> MakePath(<font color="#00007F">ByVal</font> strDrv <font color="#00007F">As</font> <font color="#00007F">String</font>, _
  <font color="#00007F">ByVal</font> strDir <font color="#00007F">As</font> <font color="#00007F">String</font>) <font color="#00007F">As</font> <font color="#00007F">String</font>
 
 <font color="#007F00">'// Makes an INI file: Guarantees a sub dir</font>
 <font color="#00007F">Do</font> <font color="#00007F">While</font> <font color="#00007F">Right</font>$(strDrv, 1) = &quot;\&quot;
   strDrv = <font color="#00007F">Left</font>$(strDrv, <font color="#00007F">Len</font>(strDrv) - 1)
 <font color="#00007F">Loop</font>

 <font color="#00007F">Do</font> <font color="#00007F">While</font> <font color="#00007F">Left</font>$(strDir, 1) = &quot;\&quot;
   strDir = <font color="#00007F">Mid</font>$(strDir, 2)
 <font color="#00007F">Loop</font>

 <font color="#007F00">'// <font color="#00007F">Return</font> the path</font>
 MakePath = strDrv &amp; &quot;\&quot; &amp; strDir
<font color="#00007F">End</font> <font color="#00007F">Function</font>

<font color="#00007F">Public</font> <font color="#00007F">Sub</font> CreateIni(strDrv <font color="#00007F">As</font> <font color="#00007F">String</font>, strDir <font color="#00007F">As</font> <font color="#00007F">String</font>)


 <font color="#007F00">'// Make a new ini file</font>
 strInI = MakePath(strDrv, strDir)
<font color="#00007F">End</font> <font color="#00007F">Sub</font>

<font color="#00007F">Public</font> <font color="#00007F">Sub</font> WriteFile(strSection <font color="#00007F">As</font> <font color="#00007F">String</font>, _
 strKey <font color="#00007F">As</font> <font color="#00007F">String</font>, _
 strValue <font color="#00007F">As</font> <font color="#00007F">String</font>)
 
 <font color="#007F00">'// Write to strINI</font>
 WritePrivateProfileString strSection, _
   strKey, strValue, strInI
<font color="#00007F">End</font> <font color="#00007F">Sub</font>

<font color="#00007F">Public</font> <font color="#00007F">Function</font> GetFile(strSection <font color="#00007F">As</font> <font color="#00007F">String</font>, _
  strKey <font color="#00007F">As</font> <font color="#00007F">String</font>) <font color="#00007F">As</font> <font color="#00007F">String</font>

  <font color="#00007F">Dim</font> strTmp <font color="#00007F">As</font> <font color="#00007F">String</font>
  <font color="#00007F">Dim</font> lngRet <font color="#00007F">As</font> <font color="#00007F">String</font>

  strTmp = <font color="#00007F">String</font>$(100, 0)
  lngRet = GetPrivateProfileString(strSection, _
    strKey, &quot;&quot;, strTmp, _
    <font color="#00007F">Len</font>(strTmp), strInI)
    
  GetFile = strTmp
 
<font color="#00007F">End</font> <font color="#00007F">Function</font>

<font color="#00007F">Public</font> <font color="#00007F">Property</font> <font color="#00007F">Let</font> INIFile(<font color="#00007F">ByVal</font> New_IniPath <font color="#00007F">As</font> <font color="#00007F">String</font>)

 <font color="#007F00">'// Sets the new ini path</font>
 strInI = New_IniPath
 
<font color="#00007F">End</font> <font color="#00007F">Property</font>

<font color="#00007F">Public</font> <font color="#00007F">Property</font> <font color="#00007F">Get</font> INIFile() <font color="#00007F">As</font> <font color="#00007F">String</font>

 <font color="#007F00">'// Returns the current ini path</font>
 INIFile = strInI
 
<font color="#00007F">End</font> <font color="#00007F">Property</font>
</pre><FONT SIZE="-1" FACE="verdana, arial, helvetica">
