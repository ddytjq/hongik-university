<h2>
Creating an Entity Bean:&nbsp; Blizzard Step 5</h2>
This brings us to the last page.&nbsp; Note that the names start off as
"PropsTable" and "propsTable" for the package name, and are edited here
to "Props" and "props".
<p><img SRC="EntityWiz5.gif">
<p>Change the bean and package names to "Props" and "props", enter a directory
name, such as "C:\Ejbtut" where you want the entity bean source files to
be placed, and click "Build".&nbsp; You should see output similar to what
is shown below, in your Java console window.
<pre><b>The following entity bean source files have been created:
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Remote Interface: C:\Ejbtut\Props.java
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Home Interface: C:\Ejbtut\PropsHome.java
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Bean Class: C:\Ejbtut\PropsBean.java
The ejb-jar file has been created in:
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; C:\Ejbtut\META-INF\ejb-jar.xml
The commands to build and compile the entity bean are in:
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; C:\Ejbtut\build.cmd
[Press ENTER to continue]</b>
