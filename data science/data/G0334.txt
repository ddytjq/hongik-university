<H1>FIRAS Explanatory Supplement </H1>

<P>
<HR>Version 4 (dated 30 December 1997) is now on line. It contains detailed
descriptions of the FIRAS instrument, its calibration and operation, and
the Pass 4 FIRAS Data Sets. 
<HR></P>

<P>You may download the Explanatory Supplement or use this Web page to
browse it. <A HREF="http://nssdcftp.gsfc.nasa.gov/spacecraft_data/cobe/firas/doc/FES_V4.PS">
A PostScript version of the entire document </A> (137 pages; Appendices
excluded) is provided for convenient printing. The document is also available
by ftp in unprocessed form (LaTeX manuscript with Encapsulated PostScript
figures and assorted ASCII and PostScript files). For advice on which files
to download, please <A HREF="http://nssdcftp.gsfc.nasa.gov/spacecraft_data/cobe/firas/doc/AREADME.DOC">
README.</A> Proceed as follows to retrieve the files by ftp: </P>

<PRE>    % ftp nssdcftp.gsfc.nasa.gov     # type this command
      ...                            #  <ftp responds>
    Name: anonymous                  # type &quot;anonymous&quot;
    Password:                        # type your e-mail address
    ftp&gt; cd spacecraft_data/cobe     # change to cobe directory
    ftp&gt; cd firas                    # change to sub-directory
    ftp&gt; cd doc                      # change to sub-directory
    ftp&gt; ls                          # list files
    ftp&gt; bin                         # binary mode (required for tar file)
    ftp&gt; get [fname]                 # retrieve file called [fname]
    ftp&gt; quit                        # end session
</PRE>

<P>To browse individual sections of the Explanatory Supplement, click on
the section names below. Appendices A - F are manuscripts of journal or
conference proceedings articles in PostScript form. Appendices G - J are
flat ASCII files. </P>
