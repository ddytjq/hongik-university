&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Functions make life easier. A lot easier. I
promise you that if you make use of functions a lot in your scripts and end up
using that function a lot, you will save yourself quite a bit of time! For
those of you who know other languages, a function is the same thing as a
method.<br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;In simple terms, a function is basically a
group of commands that have a name associated with them. Functions are useful
because it is much easier to make a function, then execute it many times, than
to retype a bunch of commands over and over. If you don't quite understand
right now, don't panic, by the end of this lesson you will! Here is the basic
form for a function.<br><br>
function functionname(){<br>
&nbsp;&nbsp;&nbsp;commands;<br>
}<br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;First things first. The open brace is the
beginning of the function, and the closed brace is the end of the function.
Replace functionname with whatever you want
the name of your function to be. A relatively short name is good, because it reduces
typing and room for screwing up! Take out "commands;" and put your own commands
in! Now to execute your function, simply put "functionname();" (without the
quotes) after your function ends!<br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;But maybe you don't want to execute your
function right when the document loads! Maybe you want to execute it when a
user clicks on a certain link! Here's some HTML code that would let you do just
that!<br><br><xmp>
<a href="javascript:functionname();">Click Here To Execute Your Function!</a>
</xmp>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Now you know how to make a function
execute when you click on a link............but I know you want to see it in
action! I created a function named itworked as an example, and here it is:<br><br>
<xmp>
function itworked(){
    alert("The function just executed");
}
</xmp>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;I put those lines in between the
script tags, of course. Now I will have it be executed when you click on it.
Here is the code I used:<br><br>
<xmp>
<a href="javascript:itworked();">Click Here To Execute itworked</a>
</xmp>and here is the link!<br><br>
<a href="javascript:itworked();">Click Here To Execute itworked</a>
<br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Or maybe I want itworked to execute when your
mouse simply goes over a link? Here is the code I would use!<br><br><xmp>
<a href="" onMouseOver="javascript:itworked();">Put your mouse over this!</a>
</xmp>and here is the link!
<br><br>

<a href=""
onMouseOver="javascript:itworked();">Put your mouse over this!</a>
<br><br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;And we haven't even gotten to the
best part! To increase the versatility, you can use arguments (also known as
parameters) with your functions. When you create your function, your arguments
go in between the parentheses, and are seperated by commas. If you only have
one argument, you don't need any parentheses.<br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;I think that the best thing I can do to
teach you about using arguments is to give you an example and explain it. I
created a function named mult and used two arguments. Here is the
function:<br><br>
function mult(x,y){<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;alert(x*y);<br>
}<br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;The arguments in mult are x and y. It allows me to specify two numbers for x and y each time I want to use the function in my HTML document. If I used 4 for x, and 3 for y, a small window would pop up saying 12, because 3 times 4 equals 12.<br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Now I will use mult so you can se it in
action. The following line of HTML code is the code which makes the link right
below it.<br><br>
<xmp>
<a href="javascript:mult(5,6);">Click here to execute mult!</a>
</xmp>
<a href="javascript:mult(5,6);">Click here to execute mult!</a><br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;There we go! That explains the basics of
using and writing functions in JavaScript! Hopefully that will help you to
understand the other JavaScript tutorials more!











<script language="javascript">
function itworked(){
	alert("The function just executed");
}
function mult(x,y){
	alert(x*y);
}
</script>
