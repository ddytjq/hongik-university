Young</a></TD></TR><TR><TD id=Review>.NET is a vast topic and a large challenge to developers of all levels of experience. This book is an very good starting point for understanding the new technologies that now form the core of future development with Microsoft tools.
<br><br>
Chappell provides a comprehensive introduction to .NET as a whole. His perspective is both architectural and functional, and he explains how the technologies contained within .NET relate to each other. While writing for a technical audience, the author’s emphasis is on general concepts. This is a difficult balance to achieve but Chappell succeeds admirably. He writes clearly and concisely. The book is well organized, with key ideas highlighted in marginal notes and the author’s commentary and opinions presented in sidebars. As a result, the main exposition of each chapter flows smoothly. 
<br><br>
<br>
Key chapters include:
<br>
<ul>
<li>
Web services
</li>
<li>
The Common Language Runtime
</li>
<li>
.NET languages, including C# and VB.NET
</li>
<li>
The .NET Framework class library
</li>
<li>
ADO.NET
</li>
<li>
ASP.NET
</li>
<li>
.NET My Services
</ul>

As you begin developing with .NET you will quickly go beyond what this book covers in some areas. But you may well return to it to refresh your knowledge of those parts of .NET that you have not been using.
<br><br>
As an introduction to the topic, Understanding .NET would be more useful if it included an annotated bibliography (“To learn more …”). Also, the book is not a tutorial; a better subtitle might be “An Analysis and Detailed Introduction.”
<br><br>
These small points aside, this is an excellent introduction to .NET.</TD></TR></Table>
