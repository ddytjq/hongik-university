<h3><font color="#000000">Links</font></h3>

<a href="http://www.a1vbcode.com/" target="_blank">A1VBCode - VB Source Code Site:</a>
Provides hundreds free Visual Basic source code snippets and applications in all categories. Submit your own code and win monthly VB prizes.<p>
<a href="http://www.cgvb.com/" target="_blank">Carl & Gary's Visual Basic Home Page:</a>
One of the first and most popular Visual Basic sites on the web. Contains an exhaustive
list of hundreds of Visual Basic links in many categories.<p>
<a href="http://www.vbwire.com/" target="_blank">VB Wire:</a>
A Visual Basic news source. This site provides Visual Basic news which is updated daily. The
site also provides a weekly VB newsletter which is delivered to your inbox with all of the past weeks VB news.<p>
<a href="http://www.appdev.com/" target="_blank">AppDev Training:</a>
Developer Training for Visual Basic, ASP, XML, SQL Server, VBA, Microsoft Access, Visual InterDev, and FoxPro through CD&#8217;s, videos, classes and licensing.<p>
<a href="http://visualbasic.ittoolbox.com/" target="_blank">ITtoolbox Portal for Visual Basic:</a>
Content, community, and service for Visual Basic professionals. Providing technical discussion, job postings, an integrated directory, news, and much more.<p>
<a href="http://www.vbchat.com/" target="_blank">VB Chat:</a>
A place where you can go and chat live about Visual Basic with other Visual Basic
programmers around the world.<p>
<a href="http://www.justVBjobs.com/" target="_blank">Just VB Jobs:</a>
This is an excellent new job site. It contains a huge database of nothing but VB jobs. It also contains numerous other VB resources which you'll find very helpful.<p>
<a href="http://www.aivosto.com/vb.html" target="_blank">VBShop - The Visual Basic Shop:</a>
Tools and tips for VB programmers. Optimization, documentation, add-ins.<p>

<a href="http://www.programmersheaven.com/" target="_blank">Programmers Heaven - Where programmers go:</a>
The Internet's most complete source of free downloadable programming files,
source codes, utilities, Visual Basic, C/C++, JAVA, and other tools for programmers
and developers. All files and links are organized in an easy-to-find format.<p>
<a href="http://www.visualbasicjobs.com/" target="_blank">Visual Basic Jobs:</a>
Another excellent VB job site which you can use to search through a huge list of Visual Basic jobs across the U.S.<p>
<a href="http://ProgrammingLinks.cjb.net/" target="_blank">Programming Links:</a>
A huge collection of categorized programming links. <p>
<a href="http://www.programmingtutorials.com/" target="_blank">ProgrammingTutorials.com:</a>
Excellent site offering programming tutorials in many categories.<p>
<a href="http://www.vbcode.com/" target="_blank">VB Code:</a>
This very popular site contains many VB code samples and snippets which you can search
through.<p>

<a href="http://www.vb-web-directory.com/" target="_blank">VB Web Directory:</a>
Contains a huge index of Visual Basic Resources. Contains links, books, VB forums, job bank,
and much more.<p>
<a href="http://www.codeguru.com/vb/" target="_blank">CodeGuru:</a>
An information resource for Visual Basic programmers. Contains information on all aspects
of Visual Basic.<p>

<a href="http://www.vbaccelerator.com/" target="_blank">VB Accelerator:</a>
An excellent general Visual Basic site providing up to date VB news, tips, and many source code samples.<p>


<a href="http://www.visualbasicexpert.com" target="_blank">Visual Basic Expert:</a>
This site is for serious VB developers, huge amounts of professional code for download, help forum, developer resources, book reviews, online training and more..<p>

<a href="http://www.developersdomain.com/vb/index.html" target="_blank">Developers Domain: </a>This site contains free code downloads, message boards, links, and much more.<p>
<a href="http://www.vb-bookmark.com/" target="_blank">Visual Basic Bookmark:</a> A comprehensive directory of programming resources and development information for Visual Basic programmers, Database developers & web designers.<p>
<a href="http://www.codearchive.com/" target="_blank">CodeArchive.com- The Source Code Site:</a> The largest amount of Visual Basic source code on the Internet. Other features include a chat room and message board. You can get 50 MB for your
own web site on CodeArchive.com .<p>
<a href="http://www.planet-source-code.com/vb/default.asp?lngWId=1&blnFrm=true" target="_blank">Planet Source Code:</a> Contains thousands of lines of source code which you can copy and paste directly into your own applications. Some of the code is in downloadable zip files.
