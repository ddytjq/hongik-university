<P>
While supplies last, VBxtras is pleased to offer a FREE copy of IP*Works! Scripting Edition (a $95 value) to everyone who places an order of at least $250 (excluding shipping and handling charges). When placing an order, be sure to ask your sales rep for your FREE copy! 
</P>

IP*Works! contains simple high-level programmable components and powerful TCP/IP programming tools. All controls have simple, yet powerful, interfaces that shield developers from the complexity of TCP/IP programming while giving them access to a wide spectrum of Winsock features. Corporate developers will find the features they need to quickly enable their applications to participate in a TCP/IP network without steep learning curves. Internet developers will find fast components with small footprints that provide excellent performance over dialup links. IP*Works! runs on top of the industry standard Winsock API and implements standard protocols specified in Internet RFCs.
<p>
<b>IP*Works! version 5</b> is /n software's most ambitious release ever, with scores of new features, 12 new components (SOAP, XML, WebUpload, WebForm, Ping, Traceroute, TFTP, HTMLMailer, etc.), updated documentation, an expanded list of sample applications, and a fresh new look and feel!  The new ASP Edition is built entirely with C++ and ATL for blazing performance, and it includes unique new security features allowing for fine-grained administrative control of the capabilities of the included controls.  IP*Works! and IP*Works! SSL V5 are now available in 15 editions: ASP, Visual Basic, ActiveX, Scripting, C++, Delphi, C++ Builder, and Java.

And the new .NET Edition consists of fully-managed .NET components.  <h4>New Components</h4>  Here's a run down of what's new in IP*Works! version 5.0:  <li>The Simple Object Access Protocol (SOAP) component allows two machines to communicate using IP*Works!'s HTTP protocol and an XML-based message format, providing a platform-independent method of exchanging data. <li>XMLp is a SAX-2 specification-compliant XML parser. It parses your XML data and raises events to which you can respond whenever it encounters an error in the XML document. <li>The WebForm and WebUpload components let you send information to a web server without having to use a web browser. WebForm uses the HTTP Post method to post data - similar to what happens when you fill out a form online and click on the submit button. WebUpload allows you to upload files to a web server for storage. <li>The Ping and TraceRoute components help you determine if a particular server is running, displaying a list of servers your traffic takes as it goes from point A to point B. <li>The Trivial FTP (TFTP) control transfers files efficiently - just specify the server, the transfer type (such as upload or download), and filename. Network routers such as Cisco use TFTP to upload updates. <li>The NetClock component requests the current time from one of the atomic clocks and can optionally set your system clock. I keep my computer clocks updated with it. Too bad it can't set my VCR! <li>The RCP control is a file-copying program.  For years UNIX users have copied files from one box to another using the Remote Copy Protocol; now Windows users can access this program. <li>Ever wondered who registered that domain name, or if a particular domain name was available? The WhoIs component queries the specified name server and returns information regarding the domain queried. <li>The FileMailer and HTMLMailer components easily send out email with attachments or in HTML format.  HTMLMailer works like IP*Works!'s SMTP component, except it sends mail formatted as HTML instead of plain text.  <h4>Untangle the Web</h4>  IP*Works! has all you need for high-powered, just-plain-awesome Internet development.
