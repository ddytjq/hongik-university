<SPAN Class="smbl">
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Choices Choices Choices. That is what the
Choice GUI component does for you. If you are familiar with Visual Basic, the
Choice in Java is the same thing as the combo box in VB. For those of you that
arent, a Choice is a box that drops down and gives you choices when click on
it. For those of you that are still unsure, check out the applet below!<br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Okay! So now you know what a Choice is! How
should you use it? Actually, it is very easy to use, there is only ONE
constructor:<br><br><i>
Choice objectnamehere=new Choice();</i><br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Not hard at all. That is how you make a
Choice. The Choice has some methods that you NEED to use, or else the Choice
won't do any good. Here are the most important methods:<br><br><i>
objectnamehere.add("Text here");<br>
objectnamehere.remove("Text here");<br>
objectnamehere.getSelectedIndex();<br>
objectnamehere.removeAll();<br>
objectnamehere.select("Text here");<br>
objectnamehere.getSelectedItem();<br><br></i>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;The first method adds a line to the
dropdown part of the Choice. To use it, type in the name of the Choice you wish
to add a line to (must already be declared), where it says objectnamehere
above, then copy the rest up until the parentheses. In the parentheses, put in
quotes what you want the line to say.<br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;The second method works the same way
as the first, except it removes a line from the dropdown. The only thing you
have to worry about is making sure that you put the exact line of text that you
want to take out.<br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;The third method is used if you are
handling events and you want to know what part of the Choice is selected. That
is what getSelectedIndex does. It returns an integer. If it returns 0, then the
first line is selected. If it returns 1, then the second line is selected. And
so on.<br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;The fourth method removes all of the lines
of the Choice. Very simple. The fifth method selects a line that contains the
text in between the quotes (if there is one!)<br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;The fifth method is finds out what the item that is selected says. For example, if I had the line Java selected, doing getSelectedItem() would give me the String whose value is Java.
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Okay! Here is the applet! Enjoy!<br><br>
<center><applet code=choice.class width=300 height=100></applet></center>

<br><br>

<pre><small>

//the next lines imports the Applet class, so that we can have the
//program we are about to make extend Applet.
import java.applet.Applet;
//the next line imports the GUI objects
import java.awt.*;
//the next line signals the beginning of the class (program)
public class choice extends Applet{
    //the next line makes a Choice called main
    Choice main=new Choice();
    //the init method executes when the Applet is initialized
    public void init(){
        //the next line adds the Choice named main to the Applet
        add(main);
        //the next line adds Hello to the Choice
        main.add("Hello");
        //the next line adds Goodbye to the Choice
        main.add("Goodbye");
        //the next line adds Ciao to the Choice
        main.add("Ciao");
		//the next line adds Java to the Choice
        main.add("Java");
    }
}
