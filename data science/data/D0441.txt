               THIS WEEKS ARTICLE: Fonts
------------------------------------------------------------

	If you have started to work with the Graphics class
and its different methods, you have probably noticed that
only get one font (unless you know how to use fonts already).
After a while, looking at the same font over and over could
get really annoying. The purpose of this article is to teach
you how to make your own Fonts to make your applets a little
more attractive. 

	Basically to create your own font, you need to create
an instance of the Font class. Use the basic format below
to create your Font:

Font fontname=new Font("Serif",Font.STYLE,size);

	Replace the word fontname in the line above with the
name that you want the Font you create to have. Serif is a
basic style of a font. You could replace Serif with Courier,
Helvetica, Arial, Geneva, or whatever you might choose. 

	Font.STYLE dictates how the font looks. For example, if
you want to make your font bold, you could put "Font.BOLD" in
the place of "Font.STYLE" in the basic format I gave you.
"Font.ITALIC" in place of  "Font.STYLE" would make the font
italicized. If you want your font to be plain, with no bold,
italics, or anything like that, use "Font.PLAIN" instead of
"Font.STYLE".

	Replace "size" in the basic format with an integer that
is the size you want your Font to be. Common sizes are 10, 12,
14, and 24. You can use whatever number you want though!

	Once you have the basic look of your Font, you can play
around with the values discussed above until you get it exactly
how you want it. Here is how you set the font of your Graphics
object to the Font you created:

g.setFont(fontname);

	This is assuming, of course, the your Graphics object is
named g. If it isn't, just replace g with the name of your 
Graphics object. Also replace fontname with the name of your
font.

	Once you have created your Font and have set the Font
of your Graphics object,  you can use the drawString method
and the Font you created will now appear on the screen!
