<!--Part body--><td colspan=4 bgcolor=#CBD1E9>Yes! As part of the process of preparing the source code for release under an Open Source license, SciTech Software and a select group of external developers will create and distribute a binary patch upgrade release (11.0c) on the Open Watcom web site. This binary patch will be available free of charge to existing Watcom C/C++ and Fortran customers and will include all updates and bug fixes developed by Sybase since the last commercial update of the product (11.0b).
