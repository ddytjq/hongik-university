<H2>Getting Help</H2>

<H3>The VB Beginner's Tutorial</H3>

<I>Learning Microsoft Visual Basic Tutorial</I><BR>
Under the Help Menus Item there is an option for <I>Learning Microsoft Visaul Basic</I>. This provides a tutorial that describes the Visual Basic programming environment and provide an explanation of some of the basic concepts. Thoughout these web pages references are made to to this tutorial at relevent points.
 
<H3>The Help File</H3>
The Most important tool for learning VB is the huge Help file that comes with the program. By using the <B>search facility</B> it is possible to find  a description for all the VB commands &amp; functions aswell as all the properties &amp; events for the basic custom controls.
<P>
The help file also provides a glossary of terms, and refernces for all the events, properties, objects &amp; trappable errors that exist. There is a great deal of useful information within this help file which can be acceessed easily using the inbuilt search engine. Once a beginner has gained the knowledge &amp; confidence to start using VB the help system provides a quick and easy method for learning as he/she goes along.
