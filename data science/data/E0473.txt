<h2>FTP Sites</h2>

<p><strong>Hint:</strong> The <a href="zip-picker.html">Zip Picker</a>
page will help you select the basic files you need and choose a nearby
FTP site for you.</p>

<p>The primary distribution site for djgpp is <b>SimTel.Net</b>, which
is a large collection of software for various systems.  The primary
site is <A HREF="http://www.simtel.net/">the SimTel.NET
archive</A></p>

<p>Once you find the closest FTP site, the things you want to read
first are <a
href="ftp://ftp.simtel.net/pub/simtelnet/gnu/djgpp/v2/readme.1st">v2/readme.1st</a> and <a href="v2faq/">the DJGPP FAQ</a>.</p>

<p>Here is a list of additional SimTel mirrors to try (from <a
href="ftp://ftp.simtel.net/pub/simtelnet/MIRRORS.TXT">SimTel's
mirror list</a>): </p>

<h2>CD-ROM Distributors</h2>

<p><a
href="http://www.burnadisc.com/cgi-bin/WebObjects/Burnadisc.woa/wa/instantSTCheckout?assets=104&discName=Simtel+Selects+GNU+2001">The
Simtel Select DJGPP CD-ROM</a> is custom made from the latest DJGPP distribution
when you order it.  This is the best way to get DJGPP on CD-ROM.</p>

<p>You can purchase djgpp from the Free Software Foundation. I think
they provide only CD-ROM, and include many GNU programs besides djgpp
itself.  Their address is: </p>
