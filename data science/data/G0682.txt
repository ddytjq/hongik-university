<H1>DIRBE Explanatory Supplement </H1>

<P>Version 2.3 (dated 14 January 1998) of the DIRBE Explanatory Supplement
is now on line. It contains detailed descriptions of the DIRBE instrument,
its calibration and operation, and the DIRBE data products. 
<HR>All of the files described below may be obtained by ftp as follows:
</P>

<PRE>    % ftp nssdcftp.gsfc.nasa.gov      # type this command
      ...                             #  <ftp responds>
    Name: anonymous                   # type &quot;anonymous&quot;
    Password:                         # type your e-mail address
    ftp&gt; cd spacecraft_data/cobe      # change to cobe directory
    ftp&gt; cd dirbe                     # change to sub-directory
    ftp&gt; cd doc                       # change to sub-directory
    ftp&gt; ls                           # list files
    ftp&gt; bin                          # binary mode (not for .doc files)
    ftp&gt; get [fname]                  # retrieve file called [fname]
    ftp&gt; quit                         # end session
</PRE>

<P>Alternatively, you may wish to retrieve the LaTeX manuscript and Encapsulated
PostScript versions of the figures. The relevant files come in a single
package and can be found in the <A HREF="http://nssdcftp.gsfc.nasa.gov/spacecraft_data/cobe/dirbe/doc/">DIRBE&nbsp;documents
directory</A> on the COBE&nbsp;ftp server. UNIX users should download the
compressed tar file and then rename, &quot;uncompress,&quot; and unpack
it as follows: </P>

<PRE>    % mv des_v2_3_tar.Z des_v2_3.tar.Z
    % uncompress des_v2_3.tar
    % tar xvf des_v2_3.tar</PRE>

<P>VMS users should obtain the VMS BACKUP file and use the command </P>

<PRE>    $ backup des_v2_3.bck/save</PRE>

<P>to unpack it. Use the following commands to process the document: </P>

<PRE>    &gt; latex des_v2.3
    &gt; dvips des_v2</PRE>

<P><A HREF="http://space.gsfc.nasa.gov/astro/cobe/dirbe_image.html">Color
versions</A> of some of the figures are available.
