<p><font face="Arial, Helvetica" size="+1"><b>Make a file association</b></font></p>
<p><i>By <a href="/php-bin/authorbio.php?authorid=31&associd=60&assoctype=2">Sam Huggill</a></i></p>
<p><font face="Arial, Helvetica" size="+1"><b>Introduction</b></font></p>

<p>Many programs these days come with their own file
format, so why shouldn't yours? By accessing the correct parts of the registry you can
make a link to your own file type, and execute your program when the file is double
clicked on. This topic is far from complete in that you would want to interpret the
command line parameters sent to your program about the file. Check out <a
href="http://vbaccelerator.com">http://vbaccelerator.com</a> for an article on command
line params. Anyway, back to the code. This sample code creates a link to .xxx files and
calls them MyApp.Document. It then loads up c:\LongPathname\MyApp.Exe Change these values
as appropiate.</p>

<p>Add a new form with a command button (cmdTest).
Copy the following code into the form:</p>

</font><pre>
<font color="#00007F">Option</font> <font color="#00007F">Explicit</font>

<font color="#007F00">'// Windows Registry Messages</font>
<font color="#00007F">Private</font> <font color="#00007F">Const</font> REG_SZ <font color="#00007F">As</font> <font color="#00007F">Long</font> = 1
<font color="#00007F">Private</font> <font color="#00007F">Const</font> REG_DWORD <font color="#00007F">As</font> <font color="#00007F">Long</font> = 4
<font color="#00007F">Private</font> <font color="#00007F">Const</font> HKEY_CLASSES_ROOT = &amp;H80000000
<font color="#00007F">Private</font> <font color="#00007F">Const</font> HKEY_CURRENT_USER = &amp;H80000001
<font color="#00007F">Private</font> <font color="#00007F">Const</font> HKEY_LOCAL_MACHINE = &amp;H80000002
<font color="#00007F">Private</font> <font color="#00007F">Const</font> HKEY_USERS = &amp;H80000003

<font color="#007F00">'// Windows <font color="#00007F">Error</font> Messages</font>

<font color="#00007F">Private</font> <font color="#00007F">Const</font> ERROR_BADDB = 1
<font color="#00007F">Private</font> <font color="#00007F">Const</font> ERROR_BADKEY = 2
<font color="#00007F">Private</font> <font color="#00007F">Const</font> ERROR_CANTOPEN = 3
<font color="#00007F">Private</font> <font color="#00007F">Const</font> ERROR_CANTREAD = 4
<font color="#00007F">Private</font> <font color="#00007F">Const</font> ERROR_CANTWRITE = 5
<font color="#00007F">Private</font> <font color="#00007F">Const</font> ERROR_OUTOFMEMORY = 6
<font color="#00007F">Private</font> <font color="#00007F">Const</font> ERROR_INVALID_PARAMETER = 7
<font color="#00007F">Private</font> <font color="#00007F">Const</font> ERROR_ACCESS_DENIED = 8
<font color="#00007F">Private</font> <font color="#00007F">Const</font> ERROR_INVALID_PARAMETERS = 87
<font color="#00007F">Private</font> <font color="#00007F">Const</font> ERROR_NO_MORE_ITEMS = 259

<font color="#007F00">'// Windows Security Messages</font>
<font color="#00007F">Private</font> <font color="#00007F">Const</font> KEY_ALL_ACCESS = &amp;H3F
<font color="#00007F">Private</font> <font color="#00007F">Const</font> REG_OPTION_NON_VOLATILE = 0

<font color="#007F00">'// Windows Registry API calls</font>
<font color="#00007F">Private</font> <font color="#00007F">Declare</font> <font color="#00007F">Function</font> RegCloseKey <font color="#00007F">Lib</font> &quot;advapi32.dll&quot; _
 (<font color="#00007F">ByVal</font> hKey <font color="#00007F">As</font> <font color="#00007F">Long</font>) <font color="#00007F">As</font> <font color="#00007F">Long</font>

<font color="#00007F">Private</font> <font color="#00007F">Declare</font> <font color="#00007F">Function</font> RegCreateKeyEx _
  <font color="#00007F">Lib</font> &quot;advapi32.dll&quot; <font color="#00007F">Alias</font> &quot;RegCreateKeyExA&quot; _
 (<font color="#00007F">ByVal</font> hKey <font color="#00007F">As</font> <font color="#00007F">Long</font>, _
  <font color="#00007F">ByVal</font> lpSubKey <font color="#00007F">As</font> <font color="#00007F">String</font>, _
  <font color="#00007F">ByVal</font> Reserved <font color="#00007F">As</font> <font color="#00007F">Long</font>, _
  <font color="#00007F">ByVal</font> lpClass <font color="#00007F">As</font> <font color="#00007F">String</font>, _
  <font color="#00007F">ByVal</font> dwOptions <font color="#00007F">As</font> <font color="#00007F">Long</font>, _
  <font color="#00007F">ByVal</font> samDesired <font color="#00007F">As</font> <font color="#00007F">Long</font>, _
  <font color="#00007F">ByVal</font> lpSecurityAttributes <font color="#00007F">As</font> <font color="#00007F">Long</font>, _
  phkResult <font color="#00007F">As</font> <font color="#00007F">Long</font>, _
  lpdwDisposition <font color="#00007F">As</font> <font color="#00007F">Long</font>) <font color="#00007F">As</font> <font color="#00007F">Long</font>

<font color="#00007F">Private</font> <font color="#00007F">Declare</font> <font color="#00007F">Function</font> RegOpenKeyEx _
  <font color="#00007F">Lib</font> &quot;advapi32.dll&quot; <font color="#00007F">Alias</font> &quot;RegOpenKeyExA&quot; _
 (<font color="#00007F">ByVal</font> hKey <font color="#00007F">As</font> <font color="#00007F">Long</font>, _
  <font color="#00007F">ByVal</font> lpSubKey <font color="#00007F">As</font> <font color="#00007F">String</font>, _
  <font color="#00007F">ByVal</font> ulOptions <font color="#00007F">As</font> <font color="#00007F">Long</font>, _
  <font color="#00007F">ByVal</font> samDesired <font color="#00007F">As</font> <font color="#00007F">Long</font>, _
  phkResult <font color="#00007F">As</font> <font color="#00007F">Long</font>) <font color="#00007F">As</font> <font color="#00007F">Long</font>

<font color="#00007F">Private</font> <font color="#00007F">Declare</font> <font color="#00007F">Function</font> RegSetValueExString _
  <font color="#00007F">Lib</font> &quot;advapi32.dll&quot; <font color="#00007F">Alias</font> &quot;RegSetValueExA&quot; _
 (<font color="#00007F">ByVal</font> hKey <font color="#00007F">As</font> <font color="#00007F">Long</font>, _
  <font color="#00007F">ByVal</font> lpValueName <font color="#00007F">As</font> <font color="#00007F">String</font>, _
  <font color="#00007F">ByVal</font> Reserved <font color="#00007F">As</font> <font color="#00007F">Long</font>, _
  <font color="#00007F">ByVal</font> dwType <font color="#00007F">As</font> <font color="#00007F">Long</font>, _
  <font color="#00007F">ByVal</font> lpValue <font color="#00007F">As</font> <font color="#00007F">String</font>, _
  <font color="#00007F">ByVal</font> cbData <font color="#00007F">As</font> <font color="#00007F">Long</font>) <font color="#00007F">As</font> <font color="#00007F">Long</font>

<font color="#00007F">Private</font> <font color="#00007F">Declare</font> <font color="#00007F">Function</font> RegSetValueExLong _
  <font color="#00007F">Lib</font> &quot;advapi32.dll&quot; <font color="#00007F">Alias</font> &quot;RegSetValueExA&quot; _
 (<font color="#00007F">ByVal</font> hKey <font color="#00007F">As</font> <font color="#00007F">Long</font>, _
  <font color="#00007F">ByVal</font> lpValueName <font color="#00007F">As</font> <font color="#00007F">String</font>, _
  <font color="#00007F">ByVal</font> Reserved <font color="#00007F">As</font> <font color="#00007F">Long</font>, _
  <font color="#00007F">ByVal</font> dwType <font color="#00007F">As</font> <font color="#00007F">Long</font>, _
  lpValue <font color="#00007F">As</font> <font color="#00007F">Long</font>, _
  <font color="#00007F">ByVal</font> cbData <font color="#00007F">As</font> <font color="#00007F">Long</font>) <font color="#00007F">As</font> <font color="#00007F">Long</font>


<font color="#00007F">Private</font> <font color="#00007F">Sub</font> CreateAssociation()

  <font color="#00007F">Dim</font> sPath <font color="#00007F">As</font> <font color="#00007F">String</font>

  <font color="#007F00">'// File Associations begin with a listing</font>
  <font color="#007F00">'// of the default extension under HKEY_CLASSES_ROOT.</font>
  <font color="#007F00">'// So the first step is to create that</font>
  <font color="#007F00">'// root extension item</font>
  CreateNewKey &quot;.xxx&quot;, HKEY_CLASSES_ROOT

  <font color="#007F00">'// <font color="#00007F">To</font> the extension just added, add a</font>
  <font color="#007F00">'// subitem where the registry will look for</font>
  <font color="#007F00">'// commands relating to the .xxx extension</font>
  <font color="#007F00">'// (&quot;MyApp.Document&quot;). Its type is <font color="#00007F">String</font> (REG_SZ)</font>
  SetKeyValue &quot;.xxx&quot;, &quot;&quot;, &quot;MyApp.Document&quot;, REG_SZ

  <font color="#007F00">'// Create the 'MyApp.Document' item under</font>
  <font color="#007F00">'// HKEY_CLASSES_ROOT. This is where you'll put</font>
  <font color="#007F00">'// the command line to execute or other shell</font>
  <font color="#007F00">'// statements necessary.</font>
  CreateNewKey &quot;MyApp.Document\shell\open\command&quot;, HKEY_CLASSES_ROOT

  <font color="#007F00">'// <font color="#00007F">Set</font> its default item to &quot;MyApp Document&quot;.</font>
  <font color="#007F00">'// This is what is displayed in Explorer against</font>
  <font color="#007F00">'// for files with a xxx extension. Its type is</font>
  <font color="#007F00">'// <font color="#00007F">String</font> (REG_SZ)</font>
  SetKeyValue &quot;MyApp.Document&quot;, &quot;&quot;, &quot;MyApp Document&quot;, REG_SZ

  <font color="#007F00">'// Finally, add the path to myapp.exe</font>
  <font color="#007F00">'// Remember to add %1 as the final command</font>
  <font color="#007F00">'// parameter to assure the app opens the passed</font>
  <font color="#007F00">'// command line item.</font>
  <font color="#007F00">'// (results in '&quot;c:\LongPathname\Myapp.exe %1&quot;)</font>
  <font color="#007F00">'// Again, its type is string.</font>
  sPath = &quot;c:\LongPathname\Myapp.exe %1&quot;
  SetKeyValue &quot;MyApp.Document\shell\open\command&quot;, &quot;&quot;, sPath, REG_SZ

  <font color="#007F00">'// All done</font>
  MsgBox &quot;The file association has been made!&quot;

<font color="#00007F">End</font> <font color="#00007F">Sub</font>

<font color="#00007F">Private</font> <font color="#00007F">Function</font> SetValueEx(<font color="#00007F">ByVal</font> hKey <font color="#00007F">As</font> <font color="#00007F">Long</font>, _
 sValueName <font color="#00007F">As</font> <font color="#00007F">String</font>, lType <font color="#00007F">As</font> <font color="#00007F">Long</font>, _
 vValue <font color="#00007F">As</font> <font color="#00007F">Variant</font>) <font color="#00007F">As</font> <font color="#00007F">Long</font>

  <font color="#00007F">Dim</font> nValue <font color="#00007F">As</font> <font color="#00007F">Long</font>
  <font color="#00007F">Dim</font> sValue <font color="#00007F">As</font> <font color="#00007F">String</font>

  <font color="#00007F">Select</font> <font color="#00007F">Case</font> lType
    <font color="#00007F">Case</font> REG_SZ
      sValue = vValue &amp; <font color="#00007F">Chr</font>$(0)
      SetValueEx = RegSetValueExString(hKey, _
        sValueName, 0&amp;, lType, sValue, <font color="#00007F">Len</font>(sValue))

    <font color="#00007F">Case</font> REG_DWORD
      nValue = vValue
      SetValueEx = RegSetValueExLong(hKey, sValueName, _
        0&amp;, lType, nValue, 4)

  <font color="#00007F">End</font> <font color="#00007F">Select</font>
   
<font color="#00007F">End</font> <font color="#00007F">Function</font>


<font color="#00007F">Private</font> <font color="#00007F">Sub</font> CreateNewKey(sNewKeyName <font color="#00007F">As</font> <font color="#00007F">String</font>, _
  lPredefinedKey <font color="#00007F">As</font> <font color="#00007F">Long</font>)

  <font color="#007F00">'// handle to the new key</font>
  <font color="#00007F">Dim</font> hKey <font color="#00007F">As</font> <font color="#00007F">Long</font>
  
  <font color="#007F00">'// result of the RegCreateKeyEx function</font>
  <font color="#00007F">Dim</font> r <font color="#00007F">As</font> <font color="#00007F">Long</font>
   
  r = RegCreateKeyEx(lPredefinedKey, sNewKeyName, 0&amp;, _
    vbNullString, REG_OPTION_NON_VOLATILE, KEY_ALL_ACCESS, 0&amp;, hKey, r)

  <font color="#00007F">Call</font> RegCloseKey(hKey)

<font color="#00007F">End</font> <font color="#00007F">Sub</font>


<font color="#00007F">Private</font> <font color="#00007F">Sub</font> SetKeyValue(sKeyName <font color="#00007F">As</font> <font color="#00007F">String</font>, sValueName <font color="#00007F">As</font> <font color="#00007F">String</font>, _
vValueSetting <font color="#00007F">As</font> <font color="#00007F">Variant</font>, lValueType <font color="#00007F">As</font> <font color="#00007F">Long</font>)

  <font color="#007F00">'// result of the SetValueEx function</font>
  <font color="#00007F">Dim</font> r <font color="#00007F">As</font> <font color="#00007F">Long</font>
   
  <font color="#007F00">'// handle of opened key</font>
  <font color="#00007F">Dim</font> hKey <font color="#00007F">As</font> <font color="#00007F">Long</font>
   
  <font color="#007F00">'// open the specified key</font>
  r = RegOpenKeyEx(HKEY_CLASSES_ROOT, sKeyName, 0, _
    KEY_ALL_ACCESS, hKey)

  r = SetValueEx(hKey, sValueName, lValueType, vValueSetting)

  <font color="#00007F">Call</font> RegCloseKey(hKey)

<font color="#00007F">End</font> <font color="#00007F">Sub</font>
