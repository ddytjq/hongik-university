<HR align=left><BR>
<A name=1></A><FONT face="Arial, Helvetica, sans-serif"><FONT 
color=#0000ff>1. </FONT>A Checklist&nbsp;</FONT>

<IMG SRC="../../figures/getStarted/check.gif" WIDTH="31" HEIGHT="35 " ALT=""></H2>

To write your first program, you will need: 
<OL>
<LI>
<B>A development environment for the Java platform.</B>
You can
<a target="other" href="http://developer.apple.com/java/download.html"><font color="#009bbb">download</font></a><a href="http://developer.apple.com/java/download.html"><img src="../../images/otherIcon.gif" width=11 height=11 border=0 align="ABSMIDDLE" alt="(outside of the tutorial)"></a>the Macintosh Runtime Environment for Java Software Development Kit
(MRJ SDK) from Apple's website.
</li>

<LI>
<B>A runtime environment for the same version of the Java platform.</B>
You can
<a target="other" href="http://developer.apple.com/java/download.html"><font color="#009bbb">download</font></a><a href="http://developer.apple.com/java/download.html"><img src="../../images/otherIcon.gif" width=11 height=11 border=0 align="ABSMIDDLE" alt="(outside of the tutorial)"></a>the Macintosh Runtime Enviroment for Java (MRJ)
from Apple's website.
</li>

<LI>
<B>Stuffit Expander 5.5</B> to open these files. 
You can
<a target="other" href="http://www.aladdinsys.com/expander/expander_mac_login.html"><font color="#009bbb">download</font></a><a href="http://www.aladdinsys.com/expander/expander_mac_login.html"><img src="../../images/otherIcon.gif" width=11 height=11 border=0 align="ABSMIDDLE" alt="(outside of the tutorial)"></a>it for free from Aladdin Systems' website.

</li>

<LI><B>A text editor.</B> In this example, we'll use SimpleText, the 
simple editor included with the Mac OS platforms. To find SimpleText, 
from the <FONT color=#990099>File</FONT> menu
select <FONT color=#990099>Find</FONT>,
type in "SimpleText",
and then hit the <FONT color=#990099>Find</FONT> button. 
You might find more then one SimpleText application;
any one of them should work fine.
You can easily adapt these instructions if you use a different text editor.
</LI>
</OL>

These four items are all you need to write your first Java program.

<TR bgColor=#ccffff vAlign=top>
<TD bgColor=#dbe3f7 vAlign=top><B><FONT 
	    face="Arial, Helvetica, sans-serif"><FONT color=#990000>Why 
	    Bytecodes are Cool</FONT></FONT></B> 
<P><FONT size=-1>So, you've heard that with the Java 
programming language, you can "write once, run anywhere." 
This means that when you compile your program, you don't 
generate instructions for one specific platform. Instead, 
you generate Java bytecodes, which are instructions for the 
Java Virtual Machine (Java VM). If your platform--whether 
it's Win32, UNIX, Mac OS, or an Internet browser--has the 
Java VM, it can understand those 
bytecodes.</FONT>
</P>
</TD>
</TR>
</TABLE>
</TD>

<TD rowSpan=2 vAlign=top>&nbsp;</TD>
</TR>

<TR>
<TD vAlign=top>Your first program, <CODE>HelloWorldApp</CODE>, 
      will simply display the greeting "Hello world!". To create this 
      program, you will:&nbsp; 
<UL>
<LI><B>Create a Java <FONT color=#990099>source file</FONT>.</B> 
	A source file contains text, written in the Java programming 
	language, that you and other programmers can understand. You can 
	use any text editor to create and edit source files.
<BR>&nbsp; 
<LI><B>Compile the source file into a <FONT 
	color=#990099>bytecode file</FONT>.</B> The Java 
	<I>compiler</I>, <CODE>javac</CODE>, takes your source file and 
	translates its text into instructions that the <I>Java Virtual 
	Machine </I>(Java VM) can understand. The compiler puts these 
	instructions into a bytecode file.
<BR>&nbsp; 
<LI><B>Run the program contained in the bytecode file.</B> The 
	Java VM is implemented by a Java <I>interpreter</I>, 
	<CODE>java</CODE>. This interpreter takes your bytecode file and 
	carries out the instructions by translating them into 
	instructions that your computer can understand.
</UL>

<p><center><IMG SRC="../../figures/getStarted/helloWorld.gif " WIDTH="465               " HEIGHT="218               " ALIGN="BOTTOM" NATURALSIZEFLAG="3" ALT=""></center></p>
</TD>
</TR>

<H2>
<A name="2a"></A> <FONT face="Arial, Helvetica, sans-serif"><FONT 
color=#ff0000>a.</FONT> Create a Java Source File.</FONT></H2>

<BLOCKQUOTE>
<P>You have two options:</P>
<UL>
<LI>You can save the file
<CODE>
<a target="source" href="../application/example/HelloWorldApp.java"><font color="#bb000f"><code>HelloWorldApp.java</code></font></a><a href="../application/example/HelloWorldApp.java"><img src="../../images/sourceIcon.gif" width=11 height=11 border=0 align="ABSMIDDLE" alt="(in a .java source file)"></a></CODE>
on your computer and avoid a lot of typing. Then, you can go straight to <A 
    href="#2b">step b</A>.<BR><BR>
</LI>
<LI>Or, you can follow these (longer) instructions: </LI>
</UL>
<P><B><FONT color=#ff0000>1.</FONT></B> Start SimpleText. In a new 
  document, type in the following code:</P>
<TABLE border=1 cellPadding=2 cellSpacing=2 width=500>
<TR> <TD width="100%"><PRE>
/** 
 * The HelloWorldApp class implements an application that
 * simply displays "Hello World!" to the standard output.
 */
class HelloWorldApp {
    public static void main(String[] args) {
        System.out.println("Hello World!"); //Display the string.
    }
}
</PRE> </TD> </TR>
</TABLE>

<BLOCKQUOTE>&nbsp; 
<TABLE border=1 cellPadding=10 cellSpacing=0 width=450>
<TR bgColor=#ffffcc>
<TD bgColor=#ffffdb>
<TABLE border=0 cellPadding=0 cellSpacing=0 width=402>
<TR>
<TD width="77%">
&nbsp;<B><FONT face="Arial, Helvetica, sans-serif"><FONT 
		color=#990000>Be Careful When You 
	      Type</FONT></FONT></B>
</TD>
<TD width="13%">
<IMG SRC="../../figures/getStarted/typeA.gif" WIDTH="30" HEIGHT="30" ALT=""></td>

<td WIDTH="10%">
<IMG SRC="../../figures/getStarted/typea2.gif" WIDTH="30" HEIGHT="30" ALT=""></td>

</TR>
</TABLE>
<P><FONT size=-1>Type all code, commands, and file names 
	  exactly as shown. The Java compiler and interpreter are 
	  <B>case-sensitive</B>, so you must capitalize 
	  consistently.</FONT></P>

<CENTER>
<P><CODE>HelloWorldApp
<IMG SRC="../../figures/getStarted/notequal.gif" WIDTH="20" HEIGHT="19" ALT="">&nbsp;helloworldapp</CODE></P></CENTER>
</TD>
</TR>
</TABLE>
</BLOCKQUOTE>
<B><FONT color=#ff0000>2.</FONT> </B>Save this code to a file. From the menu 
  bar, select <FONT color=#990099>File</FONT> &gt; <FONT 
  color=#990099>Save As</FONT>. In the <FONT color=#990099>Save 
  As</FONT> dialog box:&nbsp;</BLOCKQUOTE>
<UL>
<UL>
<LI>Specify the folder where you'll save your file. In this example, 
the folder is <CODE>MRJ SDK 2.2</CODE>.&nbsp; 
<LI>In the <FONT color=#990099>Save this document as:</FONT> text 
box, type <CODE>HelloWorldApp.java</CODE>. </LI>
</UL>
</UL>

<BLOCKQUOTE>When you're finished, the dialog box should look like this: 

<p><center><IMG SRC="../../figures/getStarted/SimpleText1.gif" WIDTH="357" HEIGHT="251" ALIGN="BOTTOM" NATURALSIZEFLAG="3" ALT=""></center></p>
Now click <FONT color=#990099>Save</FONT>, and exit SimpleText. 
</BLOCKQUOTE>

<H2>
<A name=2b></A><FONT face="Arial, Helvetica, sans-serif"><FONT 
color=#ff0000>b. </FONT>Compile the Source File.</FONT>
</H2>

<BLOCKQUOTE>
<P>Open the folder <FONT color=#990099>MRJ SDK 2.2</FONT> and it should look something like this:  
</P>

<p><center><IMG SRC="../../figures/getStarted/Finder1.gif" WIDTH="355" HEIGHT="184" ALIGN="BOTTOM" NATURALSIZEFLAG="3" ALT=""></center></p>
<P>Inside there is a folder called <FONT color=#990099>Tools</FONT>. In this folder, there is a 
  folder called <FONT color=#990099>JDK Tools</FONT>.
<BR>Open it, and you will see a program called <FONT color=#990099>javac</font>.</P>

<p><center><IMG SRC="../../figures/getStarted/javacicon-mac.gif" WIDTH="36" HEIGHT="51" ALIGN="BOTTOM" NATURALSIZEFLAG="3" ALT=""></center></p>
<P>Now drag and drop your <code>HelloWorldApp.java</code> onto the <code>Javac</code> 
application. <code>Javac</code> will now open and should show you:
</P>

<p><center><IMG SRC="../../figures/getStarted/javac1.gif" WIDTH="615" HEIGHT="269" ALIGN="BOTTOM" NATURALSIZEFLAG="3" ALT=""></center></p>
<P>The text area <FONT color=#990099>Source Files</FONT> shows the 
absolute path to the <code>.java</code> file we just created. Now there's  
nothing left to do except click the <FONT color=#990099>Do 
Javac</FONT> button to compile your code.</P>

<P>If a message like the one
below is returned without error messages, congratulations. You have successfully compiled your program.</P>

<p><center><a href="../../figures/getStarted/javac2.gif" target="_top"><IMG SRC="../../figures/getStarted/javac2.gif" WIDTH="615" HEIGHT="146" ALIGN="BOTTOM" NATURALSIZEFLAG="3" ALT=""></a><br><em>This figure has been reduced to fit on the page. <br> Click the image to view it at its natural size.</em></center></p>

<TABLE border=1 cellPadding=10 cellSpacing=0 height=250 width=541>
<TR bgColor=#ffcccc>
<TD bgColor=#ffdddd height=250><B><FONT 
	  face="Arial, Helvetica, sans-serif"><FONT color=#804000><FONT 
	  size=+1>Error Explanation</FONT></FONT></FONT></B> 
<P><B>If you drag and drop your <code>.java</code> file on top of the 
	  <code>javac</code> program and the file is only moved on top of the <code>javac</code>
	  application.</B> 
<P>When you tried to compile and the only thing that happens 
	  is that your <CODE>.java</CODE> file is copied/moved on top of your <code>javac</code> 
	  application you need to rebuild your desktop. 
<P>To rebuild, you have to restart you computer and press 
	  and hold both the <FONT color=#990099>Apple</FONT> and 
	  <FONT color=#990099>Alt</FONT> key until you get a 
	  dialog asking if you really want to rebuild your 
	  desktop. 
	  <P>Answer "Yes." When your desktop is finished rebuilding you

of the JBindery icon.

<hr></blockquote>

You should now see the following dialog box:</P>

<p><center><IMG SRC="../../figures/getStarted/JBindery1.gif" WIDTH="499" HEIGHT="357" ALIGN="BOTTOM" NATURALSIZEFLAG="3" ALT=""></center></p>
<P>
Click the <FONT color=#990099>Run</font> button. If you now get:</P>

<p><center><IMG SRC="../../figures/getStarted/output-mac.gif" WIDTH="227" HEIGHT="151" ALIGN="BOTTOM" NATURALSIZEFLAG="3" ALT=""></center></p>

</BLOCKQUOTE>

<H2><A name=3></A><FONT face="Arial, Helvetica, sans-serif"><FONT 
color=#990099>3.</FONT><FONT color=#009900> </FONT>Creating Your First 
Applet</FONT>
</H2>

<P><CODE>HelloWorldApp</CODE> is an example of a Java 
<I>application</I>, a standalone program. Now you will create a Java 
<I>applet </I>called <CODE>HelloWorld</CODE>, which also displays the 
greeting "Hello world!". Unlike <CODE>HelloWorldApp</CODE>, however, the 
applet runs in a Java-enabled Web browser such as HotJava, Netscape 
Navigator, or Microsoft Internet Explorer.</P>

<P>To create this applet, you'll perform the basic steps as before: 
create a Java source file; compile the source file; and run the 
program.</P>
<HR align=left width="70%">

<H2><FONT color=#990099 face="Arial, Helvetica, sans-serif"><A 
name=3a></A>a. </FONT>Create a Java Source File.
</H2>
<BLOCKQUOTE>
<P>Again, you have two options:</P>
<UL>
<LI>You can save the files 
<a target="source" href="../applet/example/HelloWorld.java"><font color="#bb000f"><code>HelloWorld.java</code></font></a><a href="../applet/example/HelloWorld.java"><img src="../../images/sourceIcon.gif" width=11 height=11 border=0 align="ABSMIDDLE" alt="(in a .java source file)"></a>and
<a target="source" href="../applet/example/Hello.html"><font color="#bb000f"><code>Hello.html</code></font></a><a href="../applet/example/Hello.html"><img src="../../images/sourceIcon.gif" width=11 height=11 border=0 align="ABSMIDDLE" alt="(in a .java source file)"></a>on your computer and avoid a lot of typing. Then, you can go 
straight to <A href="#3b">step b</A>.
<BR>
<BR>
<LI>Or, you can follow these instructions: </LI>
</UL>
<P>
<B><FONT color=#990099>1.</FONT></B><FONT color=#990099> 
</FONT>Start SimpleText. Type the following code into a new document: 
</P>
<TABLE border=1 cellPadding=2 cellSpacing=2 width=500>
<tr> <td width="100%"><pre>
import java.applet.Applet;
import java.awt.Graphics;

public class HelloWorld extends Applet {
    public void paint(Graphics g) {
        g.drawString("Hello world!", 50, 25);
    }
}
</pre></td></tr>                           
</TABLE>

<P>Save this code to a file called <CODE>HelloWorld.java</CODE>.</P>

<P><B><FONT color=#990099>2.</FONT> </B>You also need an HTML file to 
  accompany your applet. Type the following code into a new SimpleText 
  document:</P>

<TABLE border=1 cellPadding=2 cellSpacing=2 width=500>
<TR>
<TD width="100%"><PRE><CODE>&lt;HTML&gt;
&lt;HEAD&gt;
&lt;TITLE&gt;A Simple Program&lt;/TITLE&gt;
&lt;/HEAD&gt;
&lt;BODY&gt;
Here is the output of my program:
&lt;APPLET CODE="HelloWorld.class" WIDTH=150 HEIGHT=25&gt;
&lt;/APPLET&gt;
&lt;/BODY&gt;
&lt;/HTML&gt;
</CODE></PRE></TD>
</TR>
</TABLE>

<P>Save this code to a file called <CODE>Hello.html</CODE>.</P>

<P>Make sure that your files HelloWorld.java and Hello.html are in the 
  same folder.</P>
</BLOCKQUOTE>

<HR align=left width="70%">

<H2><FONT face="Arial, Helvetica, sans-serif"><FONT color=#990099><A 
name=3b></A><FONT face="Arial, Helvetica, sans-serif"><FONT 
color=#990099></FONT></FONT>b. </FONT>Compile the Source 
File.</FONT>
</H2>
<BLOCKQUOTE>
<P>Compile the <FONT size=-2>HelloWorld.java</FONT> source file using 
  javac as before.</P>
<P>The compiler should generate a Java bytecode file, 
  <CODE>HelloWorld.class</CODE>. </P></BLOCKQUOTE>

<HR align=left width="70%">

<H2><FONT face="Arial, Helvetica, sans-serif"><FONT color=#990099><A 
name=3c></A><FONT face="Arial, Helvetica, sans-serif"><FONT 
color=#990099></FONT></FONT>c. </FONT>Run the Program.</FONT>
</H2>
<BLOCKQUOTE>
<P>Although you can view your applets using a Web browser, you may 
  find it easier to test your applets using the simple Applet Runner 
  application that comes with the Java<SUP><FONT size=-2>TM</FONT></SUP> 
  Platform. To view the HelloWorld applet using Applet Runner, Open the 
  folder <FONT color=#990099>Apple Applet Runner</FONT> in the MRJ SDK 
  2.2 folder. There should be an application called Apple Applet 
  Runner.</P>

<p><center><IMG SRC="../../figures/getStarted/viewericon-mac.gif" WIDTH="106" HEIGHT="48" ALIGN="BOTTOM" NATURALSIZEFLAG="3" ALT=""></center></p>
<P>Drag and drop your Hello.html file on this application.</P>
<P>Now you should see:</P>

<p><center><IMG SRC="../../figures/getStarted/AppleAppletRunner1.gif" WIDTH="175" HEIGHT="78" ALIGN="BOTTOM" NATURALSIZEFLAG="3" ALT=""></center></p>
<P>Congratulations! Your applet works.</P>
</BLOCKQUOTE>
