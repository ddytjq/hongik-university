<h2>
Adding an ODBC Data Source</h2>
ODBC data sources can be added on Windows machines via the <tt>Control
Panel</tt>.&nbsp; Double click the <tt>ODBC Data Sources</tt> or the <tt>ODBC-32</tt>
icon (on Windows/2000, this is available after clicking on the "Administrative"
icon first), which will bring up a property sheet.&nbsp; Select the "<tt>System
DSN</tt>" tab and click the "<tt>Add...</tt>" button.&nbsp; Select the
driver for the database you want to add, and fill in the database information.&nbsp;
<p>If you have downloaded sample <tt>Props.mdb</tt> file, select "Microsoft
Access Driver" and click "<tt>Finish</tt>".&nbsp; Then add a data source
name (e.g. "<tt>PropsOdbc</tt>") and click the "<tt>Select...</tt>" button
to select the <tt>Props.mdb</tt> file.&nbsp;
