<h2>
Setting up an ODBC Connection on a Windows machine</h2>
Open up the Control Panel and look for an icon named "ODBC Data Sources"
or "ODBC-32" or just "Data Sources"..&nbsp; If you are on a Windows/2000
machine, this icon will be inside "Administrative Services" folder in the
Control Panel.
<p>Now select the "System DSN" tab, and click "Add".&nbsp; Select the Microsoft
Access driver and click the "<tt>Finish</tt>" button.&nbsp; You will now
be taken to the Microsoft Access specific page.&nbsp; Specify the data
source name as "jspsql" and click the "Select.." button to specify the
<tt>jspsqlsample.mdb</tt> file you have downloaded.  Add in a description
if you prefer, and click OK.
