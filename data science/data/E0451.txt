                        <p> <FONT FACE="Verdana, Arial, Helvetica, sans-serif" size="2" COLOR="#000000"> 
                          <b>What is Open Watcom?</b></FONT> <br>
                          <FONT FACE="Verdana, Arial, Helvetica, sans-serif" size="2"> 
                          Open Watcom is an effort to take the commercial Watcom 
                          compilers, and turn them into an Open Source Project. 
                          When Sybase realised that the Watcom C/C++ and Fortran 
                          compilers were no longer commercially viable, they searched 
                          around to find a solution for their existing customers. 
                          SciTech Software, a long time user of Watcom compiler 
                          products contacted Sybase about turning the compilers 
                          into an Open Source project, and hence the Open Watcom 
                          Project was born.</FONT> 
                        <FONT face="Verdana, Arial, Helvetica, sans-serif" size="2"> 
                        <br>
                        The Open Watcom products are the first and only commercial 
                        compilers to be open sourced and, with more than a million 
                        lines of source code, the Watcom products represent one 
                        of the largest pools of commercial source code <I>of any 
                        type</I> ever released under an Open Source license. Sybase 
                        is releasing the code under an Open Source license that 
                        complies with the Open Source Definition, published by 
                        the Open Source Initiative (<A 
              href="http://www.opensource.org/" target="_blank">http://www.opensource.org</A>). 
                        <P> Initially the development on the Open Watcom Project 
                          will be closed, as SciTech and a team of selected external 
                          developers work to prepare the source code for release 
                          under an Open Source license. During this time, the 
                          Open Watcom team will create and distribute a binary 
                          patch upgrade release (11.0c) on this web site. This 
                          binary patch will be available free of charge to existing 
                          Watcom C/C++ and Fortran customers and will include 
                          all updates and bug fixes developed by Sybase since 
                          the last commercial update of the product (11.0b). </P>
                        <P> Following the binary patch release, the complete source 
                          to the Open Watcom products will be released on the 
                          web site, so that any developer will be able to use 
                          and modify the code. </P>
                        <P><font face="Verdana, Arial, Helvetica, sans-serif" size="2" color="#000000"><b>Open 
                          Watcom News</b></font><br>
                          For the latest press release news for the Open Watcom 
                          Project, check the <a href="/news/press_content.html" target="_blank">Open 
                          Watcom Press Room</a>. Members of the media can also 
                          find contact information for Open Watcom Media Relations 
                          as well as what other publications have to say about 
                          the Open Watcom Project. </P>
                        </FONT> 
                        <p> <font face="Verdana, Arial, Helvetica, sans-serif" size="2" color="#000000"> 
                          <b>Contacting SciTech</b></font><br>

                          for the Open Watcom project. The <a href="http://www.scitechsoft.com/about/contact.html" target="_blank"> 
                          contact information page</a> contains addresses, phone 
                          numbers and email addresses for SciTech Software departments. 
                          </font><br>
