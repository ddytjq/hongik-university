<h2>
Lesson 11: Introduction to VB Functions- Part II</h2></center>
<b>11.1 Creating Your Own Functions</b>
<p>The general format of a function is as follows:
<p>Public&nbsp; Function <font color="#3333FF">functionName </font><font color="#000000">(Arg
As dataType,..........) As dataType</font>
<p><font color="#000000">or</font>
<p>Private&nbsp; Function <font color="#3333FF">functionName </font><font color="#000000">(Arg
As dataType,..........) As dataType</font>
<p><font color="#000000">* Public indicates that the function is applicable
to the whole program and</font>
<br><font color="#000000">&nbsp;&nbsp; Private indicates that the function
is only applicable to a certain module or procedure.</font>
<br>&nbsp;
<p>Example 11.1
<p>In this example, a user can calculate future value of a certain amount
of money he has today based on the interest rate and the number of years
from now(supposing&nbsp; he will invest this amount of money somewhere).
The calculation is based on the compound interest rate.
<p><img SRC="fv.gif" height=274 width=320>
<br>&nbsp;
<p>Public Function FV(PV As Variant, i As Variant, n As Variant) As Variant
<br>'Formula to calculate Future Value(FV)
<br>'PV denotes Present Value
<br>FV = PV * (1 + i / 100) ^ n
<p>End Function
<p>Private Sub compute_Click()
<br>'This procedure will calculate Future Value
<br>Dim FutureVal As Variant
<br>Dim PresentVal As Variant
<br>Dim interest As Variant
<br>Dim period As Variant
<br>PresentVal = PV.Text
<br>interest = rate.Text
<br>period = years.Text
<p>FutureVal = FV(PresentVal, interest, period)
<br>MsgBox ("The Future Value is " &amp; FutureVal)
<br>End Sub
<p>Example 11.2
<p>The following program will automatically compute examination grades
based on the marks that a student obtained.
<p><img SRC="grade.gif" height=242 width=318>
<p>Public Function grade(mark As Variant) As String
<br>Select Case mark
<br>Case Is >= 80
<br>grade = "A"
<br>Case Is >= 70
<br>grade = "B"
<br>Case Is >= 60
<br>grade = "C"
<br>Case Is >= 50
<br>grade = "D"
<br>Case Is >= 40
<br>grade = "E"
<br>Case Else
<br>grade = "F"
<br>End Select
<p>End Function
<p>Private Sub compute_Click()
<p>grading.Caption = grade(mark)
<br>&nbsp;
<p>End Sub
<p>Private Sub End_Click()
<br>End
<p>End Sub
