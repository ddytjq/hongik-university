	<tr><td colspan="2">&nbsp;</td></tr>
	<tr><td colspan="2">
 <img src="http://www.vbxtras.com/gifs/vom.gif" align="right" hspace="10" valign="top">
<font size=2>Visual UML is an affordable, easy-to-use, yet powerful and full-featured object-modeling tool that supports all OMG 1.3 UML (Unified Modeling Language) diagram types: Use Case, Package, Class, Object, State, Collaboration, Activity, Deployment, Component and Sequence diagrams.<BR><BR>  
Visual UML Developer Edition adds several integrated, bi-directional language interfaces that support the generation and reverse-engineering of Visual Basic, C# and Java code. Plus, generation of C++ code. Plus database schema generation and reverse-engineering of several database types. Visual UML Developer Edition can also run as an add-in to the Visual Basic IDE (for generating and reverse-engineering Visual Basic projects) and the Visual Studio.NET IDE (for generating and reverse-engineering C# projects.)<BR><BR>
Visual UML Plus Edition adds integrated support for Visual Basic for Applications (VBA) 6.3.<BR><BR>
Visual UML for Visual FoxPro adds integrated round-trip code generation and reverse-engineering of Visual FoxPro projects.
</font><br><br>

			<td><div style="font:8pt"><b>Which Visual UML is Right for Me?</b><br><br><ul>
<li><b>Visual UML v2.9.2 Standard Edition</b> is the entry-level product.  It implements the Unified Modeling Language (UML) 1.3 through an integrated model/data dictionary and comprehensive support for Class, Object, Package, Use Case, Component, Deployment, Collaboration, Sequence, Activity, and State diagrams.</li><br><br>
<li><b>Visual UML v2.9.2 Developer</b> includes all the features of the Standard Edition plus it adds several integrated, bi-directional language interfaces that support the generation and reverse-engineering of Visual Basic, C# and Java code. Plus, generation of C++ code. Plus database schema generation and reverse-engineering of several database types. Visual UML Developer Edition can also run as an add-in to the Visual Basic IDE (for generating and reverse-engineering Visual Basic projects) and the Visual Studio.NET IDE (for generating and reverse-engineering C# projects).

</li><br><br>
<li><b>Visual UML v2.9.2 Plus Edition</b> contains the same features as the Standard Edition plus integrated support for Visual Basic for Applications (VBA) 6.3.</li><br><br>
<li><b>Visual UML v2.9.2 Plus Developer</b> includes all of the features of the Plus Edition plus it adds several integrated, bi-directional language interfaces that support the generation and reverse-engineering of Visual Basic, C# and Java code. Plus, generation of C++ code. Plus database schema generation and reverse-engineering of several database types. Visual UML Developer Edition can also run as an add-in to the Visual Basic IDE (for generating and reverse-engineering Visual Basic projects) and the Visual Studio.NET IDE (for generating and reverse-engineering C# projects).

</li><br><br>
<li><b>Visual UML v2.9.2 Standard for Visual FoxPro</b> contains the same features as the Standard Edition plus a bi-directional interface to Visual FoxPro.  This edition supports round-trip code generation and reverse-engineering of Visual FoxPro projects.</li><br><br>
<li><b>Visual UML v2.9.2 Plus for Visual FoxPro</b> contains the same features as the Plus Edition plus a bi-directional interface to Visual FoxPro.  This edition supports round-trip code generation and reverse-engineering of Visual FoxPro projects.</li><br><br>
