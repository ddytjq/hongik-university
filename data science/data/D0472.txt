<p><font face="Arial" size="2">This applet is a simple image animation which plays a
series of GIF files at a preset speed specified by the HTML author as a parameter. The
HTML author can also specify background color. The animation is stopped when the applet
container is clicked, and restarts when it is clicked again. </font></p>

<p><font face="Arial" size="2">This is what the mark-up looks like to implement the
animator applet in your HTML page (Note: this assumes that<i>animator_andyhosk.class</i>
is in the same directory as it's parent HTML page). The parameters, by the way, correspond
to the instance that is running on this page right now: </font></p>

<form>
  <div align="center"><center><p><textarea cols="50" name="applettagesource" rows="10">&lt;APPLET code=animator_andyhosk.class id=animator_andyhosk width=260 height=60&gt; 
&lt;PARAM name=number_of_images value=18&gt;
&lt;PARAM name=imagebase value=&quot;images/img00&quot;&gt; 
&lt;PARAM name=time_interval value=50&gt;
&lt;PARAM name=backgroundcolor value=&quot;white&quot;&gt; 
&lt;B&gt;Your browser does not support Java&lt;/B&gt; 
&lt;/APPLET&gt;
</textarea></p>
  </center></div>
</form>

<table border="1" cellpadding="1" cellspacing="0">
  <tr>
    <td><font face="Arial" size="2"><b>Parameter</b> </font></td>
    <td><font face="Arial" size="2"><b>Possible Values</b> </font></td>
  </tr>
  <tr>
    <td><font face="Arial" size="2">number_of_images </font></td>
    <td><font face="Arial" size="2">The number of images in the animation. </font></td>
  </tr>
  <tr>
    <td><font face="Arial" size="2">imagebase </font></td>
    <td><font face="Arial" size="2">A string representing the relative path and base file
    names of the image files. &nbsp;For example, if you have 18 images, their files names are <b>img0001.gif
    </b>through <b>img0018.gif</b>, and they are located in a sub-directory called
    &quot;images&quot;, then the imagebase parameter is &quot;images/img00&quot;. &nbsp;It is
    important to note a few things: </font><ul>
      <li><font face="Arial" size="2">The imagebase path is relative to the applet's codebase. </font></li>
      <li><font face="Arial" size="2">The images must be .GIF files. </font></li>
      <li><font face="Arial" size="2">The animator applet can support up to 99 images. </font></li>
    </ul>
    </td>
  </tr>
  <tr>
    <td><font face="Arial" size="2">time_interval </font></td>
    <td><font face="Arial" size="2">Time delay between frames, in milliseconds; for all
    intents and purposes, represents the speed of the animation. </font></td>
  </tr>
  <tr>
    <td><font face="Arial" size="2">backgroundcolor </font></td>
    <td><font face="Arial" size="2">white, black, red, blue, green, yellow, <i>or</i> an
    integer representing the RGB value <b>(See Note Below)</b>. </font></td>
  </tr>
</table>

<h2><font face="Arial" size="2">Background Color Attributes </font></h2>

<p><font face="Arial" size="2">Most people will find the six basic colors sufficient.
However, you can set the background color of the applet to any one of the 16+ million
colors in the 24 bit RGB (Red-Green-Blue) spectrum. But first, you must calculate the
value of your custom color, using <a href="../rgb/">this calculator</a>. </font></p>
