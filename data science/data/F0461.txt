<H2><A HREF="customgis.htm"><IMG SRC="icons/msvb.gif" ALT="Visual Basic Icon" ALIGN=middle BORDER=0></A>The General Programming Process in VB</H2>
There are six main steps in creating a Visual Basic Application. These are;
<P>
<OL COMPACT>
<LI>Create a visual interface
    <UL COMPACT>
    <LI>create a form (VB term for a window)
    <LI>add <A HREF="lt1-22.htm">controls</A> to fulfil the requirements of the program
    <LI>edit the <A HREF="lt1-23.htm">properties</A> of each control
    </UL>
<LI>Write the underlying code
    <UL COMPACT>
    <LI>write additional functions and procedures to control the program and the controls
    <LI>associate this code with <A HREF="lt1-24.htm">control events</A>
    </UL>
<LI>Debug the program
<LI>Create an Executable File
<LI>Test the program in different environments
<LI>Create a Setup Disk
</OL>
