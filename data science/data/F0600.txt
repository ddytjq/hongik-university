<B>Menus</b>
</font>
</p>

<p align="left">
<font face="tahoma">
<font size="2">
<B>Menus</b> are very common in applications these days and are very useful, not to mention they organize a program and make it look professional. Making menus is very easy, and they work like so, you click they do. Pretty much like command buttons.
</p>

<p align=center>
<img src="http://www.angelfire.com/ny3/visualbasic/images/vbmenu.jpg"
</p> 

<p align="left">
<font face="tahoma">
In order to start on making a menu, run VB, right click over a form in VB's development environment and click menu editor.

<p align=left>
<font face="tahoma">
<font size="2">
Here is what you need to know about the menu editor:
</p>

<p align=left>
<font face="tahoma">
<font size="2">
<font color="red">
Checked</font> - This shows if the menu item has a check next to it. (Checks will not check and uncheck @ run time if the user clicks on them several times, this has to be done in code)<Br>
<Br>
<font color="red">
HelpContextID</font> - Matches a help file description if you have help files in your program<Br>
<Br>
<font color="red">
Index</font> - This distinguishes menu control arrays<Br>
<Br>
<font color="red">
Shortcut</font> - A keypress that accesses a menu item (for example if you have Quit as one of your menu items you can have the shortcut of F1, and instead of the user having to go and actually click the menu item, they can just press F1 instead)<Br>
<Br>
<font color="red">
Window List</font> - Determines if the menu item applies to an MDI (multiple document interface, such as Microsoft Word)<Br>
</p>

<p align=center>
<font face="tahoma">
<font size="2">
<font color="blue">
<img src="http://www.angelfire.com/ny3/visualbasic/images/tip.jpg"> <B>Tip:</b> <B>Control arrays</b> are controls that all have the same name (example, you have 3 text boxes that have the name txt1)
</font>
</p>

<p align=left>
<font face="tahoma">
<font size="2">
Lets make a quick menu (well, sorta quick...).<Br>
<Br>
<B>1</b>) Open menu editor in VB<Br>
<B>2</b>) In the Caption write "&File" (the & underlines the letter coming after it, so we can access it w/ the keyboard)<Br>
<B>3</b>) In the Name write "mnuFile"<Br>
<B>4</b>) Click Next and then click the right arrow button<Br>
<B>5</b>) In Caption write "Example"<Br>
<B>6</b>) In Name write "mnuExample"<Br>
<B>7</b>) Click Next<Br>
<B>8</b>) In Caption write "-"<Br>
<B>9</b>) In Name write "mnuSeperator"<Br>
<B>10</b>) Click Next<Br>
<B>11</b>) In Caption write "Example2"<Br>
<B>12</b>) In Name write "mnuExample2"<Br>
<B>13</b>) Click Next and then click the right arrow button<Br>
<B>14</b>) In Caption write "Example3"<Br>
<B>15</b>) In Name write "mnuExample3"<Br>
<B>16</b>) Click Next and then click the left arrow button<Br>
<B>17</b>) In the Caption write "Example4"<Br>
<B>18</b>) In the Name write "mnuExample4"<Br>
<B>19</b>) Click OK<Br>
<B>20</b>) Run it<Br>
</p>

<p align=left>
<font face="tahoma">
<font size="2">
OK so it was a lot of work, but now you know almost everything you need to know to master Menus. in order to put menus to work, click the menu item (in VB's dev. environment) and that's where the codes are going to go.
