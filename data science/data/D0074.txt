<H4>Notes on this Web Page Background:</H4>
It is an <a href="../3d/index.html#anaglyph">anaglyph</a>
that I drew using Java.  View it with red-blue 3D glasses. It is an experiment with
complimentary colors. Red has been replaced with cyan. Blue has been replaced with yellow.
<p>
This background shows 2 layers of a honeycomb (a solid tesselation)
composed of <a href="http://home.teleport.com/~tpgettys/platonic.shtml">octahedra and tetrahedra</a>.
This is the only "quasi-regular" honeycomb (see H.S.M. Coxeter "Regular Polytopes" section 4.7
<a href="../references.html#Coxeter01">(Coxeter01)</a>).
For other honeycomb backgrounds, see <a href="../mark/index.html">Mark's Page</a>
and my <a href="../mark/rhdodec.html">Rhombic Dodecahedron Page</a>.
