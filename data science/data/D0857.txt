<h1>
Online Resources for Java Programmers
</h1>

<p>

<p>

The Web offers many resources
to help you use the Java platform.
This section lists some that we use ourselves,
or that we've heard good things about.

<p>

<blockquote><hr><strong>Caveat:</strong>&nbsp;A Web site that's good
one year might stagnate the next,
and it can be difficult to tell 
unless you know the site well.
Please
<a href="../forms/sendusmail.html" target="_top">tell us</a>
if any of these Web sites is obsolete,
or if you have a URL for an excellent resource not mentioned here.
<hr></blockquote>

<p>

We list three kinds of resources:

<dt> <a href="#general">General Programming Resources</a>
<dd> Web sites that provide information useful to most developers.

<dt> <a href="#subset">Specialized Programming Resources</a>
<dd> Sources of more specialized programming information.
     Examples include assorted tutorials,
     The Swing Connection,
     and a Web site for Macintosh programmers.

<dt> <a href="#nonprogram">Other Resources</a>
<dd> Other stuff, such as the latest industry news.
</dl>

Don't forget that the 
<a href="../index.html">Tutorial</a> has
many examples,
and that you can

Another source of resources is the
<a target="other" href="http://java.sun.com/nav/developer/resources/"><font color="#009bbb">Additional Resources</font></a><a href="http://java.sun.com/nav/developer/resources/"><img src="../images/otherIcon.gif" width=11 height=11 border=0 align="ABSMIDDLE" alt="(outside of the tutorial)"></a> page on java.sun.com.
