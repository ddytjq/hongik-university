<H2><A NAME="soap"></A> <A NAME="s8">8. SOAPX4 and PhpXMLP </A></H2>

<P>SOAP is an XML-based protocol for messaging and RPC-style communication between two processes.
The World Wide Web Consortium's XML Protocol (XP) Working Group is working on a SOAP 
standard, which will be called XP.
<P>The SOAPX4 (Simple Object Access Protocol) implementation for PHP is at
<A HREF="http://dietrich.ganx4.com/soapx4">http://dietrich.ganx4.com/soapx4</A>.
<P>Alternate SOAP implementation is at 
<A HREF="http://sourceforge.net/projects/phpxmlp">http://sourceforge.net/projects/phpxmlp</A>.
<P>

<H2><A NAME="xmlengine"></A> <A NAME="ss8.1">8.1 XML/XSLT Publishing Engine</A>
</H2>

<P>Krysalis is the XML/XSLT publishing engine,
inspired from Cocoon but adapted to the PHP technology.
<P>InterAKT has developed Krysalis to achieve what the next generation of
web services have to offer -  the complete separation of the information,
application logic and presentation layer.
<P>Krysalis is an open-source PHP development platform, based on the
XML/XSLT core. It reused most of PHAkt code to create the Krysalis taglib library.
<P>Major features are:
<UL>
<LI>Dynamic XML support.</LI>
<LI>Dynamic PDF generation.</LI>
<LI>Sitemap description with regular expressions.</LI>
<LI>PXP page caching.</LI>
<LI>Multiple transformation pipelines.</LI>
<LI>Multiple connection types to various databases (ADODB).</LI>
</UL>
<P>Please visit the Krysalis site for more information
<A HREF="http://www.interakt.ro/products/Krysalis">http://www.interakt.ro/products/Krysalis</A>.
