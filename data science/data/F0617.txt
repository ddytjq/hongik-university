<P>

  The links in the following text display images that demonstrate the techniques in this book. 
Some browsers make these files appear grainy. If yours does, try viewing the .GIF files stored 
in your Web cache directory.
  <P>

  <TABLE WIDTH="100%" CELLPADDING="0" CELLSPACING="0">
    <TR><TD BGCOLOR="#7FCFFF" ALIGN="Center"><B>Part One: Working with Windows</B></TD></TR>
  </TABLE>
  <P>

  <DL>
<DT><B>1. Visual Basics</B></DT>
<DD>
This chapter covers Visual Basic graphics fundamentals such as
coordinate systems, handling Paint events, graphical controls,
creating scrolled windows, and basic colors.
</DD>

<DT><B>2. Using the API</B></DT>
<DD>
This chapter explains how to use the Windows API in 
Visual Basic programs. It also describes many API functions that are
useful in graphics programs.
</DD>

<DT><B>3. Advanced Color</B></DT>
<DD>
Chapter 3 describes more advanced color topics including different
color models, device-independent and device-dependent bitmaps,
and color palettes. The more bitmap-oriented parts book later
assume you are using higher color modes such as 24-bit color.
This chapter describes color palettes for completeness.
</DD>

<DT><B>4. Advanced Text</B></DT>
<DD>
Chapter 4 covers advanced text operations such as finding text metrics
to learn exactly how big a font is, and how to use the CreateFont API
function to build stretched, compressed, and 
<A HREF="vbgp/rotated_text.gif">rotated text</A>.
</DD>

<DT><B>5. Printing</B></DT>
<DD>
This chapter explains how to use the Printer object to produce
high-resolution printouts. It explains how to print, display a print
preview, handle multi-page documents, and scale objects to fit
the printer nicely. It also includes code that lets you easily
print a form much as PrintForm does, only in high resolution.
</DD>
</DL>
  <P>

  <TABLE WIDTH="100%" CELLPADDING="0" CELLSPACING="0">
    <TR><TD BGCOLOR="#7FCFFF" ALIGN="Center"><B>Part Two: Image Processing</B></TD></TR>
  </TABLE>
  <P>

  <DL>
<DT><B>6. Point Processes</B></DT>
<DD>
This chapter explains image processing techniques including that
manipulate an image one pixel at a time. These methods include image
<A HREF="vbgp/point_process1.gif">averaging</A>, 
adjusting an image's brightness, contrast, or color balance, and
<A HREF="vbgp/hugecat.jpg">compositing</A>.
</DD>

<DT><B>7. Area Processes</B></DT>
<DD>
Chapter covers image processing techniques thaht manipulate areas
in an image. These methods include spatial filtering to smooth,
sharpen, or 
<A HREF="vbgp/point_process2.gif">emboss</A> an image.
This chapter also describe
<A HREF="vbgp/rotate.jpg">rotation</A> and
<A HREF="vbgp/warp.jpg">shape-disorting</A>
image transformations.
</DD>
</DL>
  <P>

  <TABLE WIDTH="100%" CELLPADDING="0" CELLSPACING="0">
    <TR><TD BGCOLOR="#7FCFFF" ALIGN="Center"><B>Part Three: Animation</B></TD></TR>
  </TABLE>
  <P>

  <DL>
<DT><B>8. Bitmap Animation</B></DT>
<DD>
Chapter 8 explains the fundamentals of bitmap animation. It shows how
to use several different methods for presenting  images quickly
enough to create smooth animation.
</DD>

<DT><B>9. Advanced Animation</B></DT>
<DD>
Chapter 9 tackles more advanced animation topics. It describes 
different ways you can control an animation such as scripting,
simulation, and sprites. It also describes specialized animation
techniques such as tweening and
<A HREF="vbgp/morph.jpg">morphing</A>.
</DD>
</DL>
  <P>

  <TABLE WIDTH="100%" CELLPADDING="0" CELLSPACING="0">
    <TR><TD BGCOLOR="#7FCFFF" ALIGN="Center"><B>Part Four: Two-Dimensional Graphics</B></TD></TR>
  </TABLE>
  <P>

  <DL>
<DT><B>10. Fractals and Tilings</B></DT>
<DD>
As its title implies, this chapter describes fractals and tilings
including
<A HREF="vbgp/random_tree.gif">random trees</A>,
animated 
<A HREF="vbgp/snowflake.gif">snowflakes</A>,
<A HREF="vbgp/hilbert_curve.gif">Hilbert curves</A>,
<A HREF="vbgp/gasket.gif">gaskets</A>,
<A HREF="vbgp/chaos.gif">the chaos game</A>,
the Mandelbrot set (<A HREF="vbgp/mandelbrot_set8.gif">image 1</A>,
<A HREF="vbgp/mandelbrot_set6.gif">image 2</A>,
<A HREF="vbgp/mandelbrot_set4.gif">image 3</A>), and
Julia sets (<A HREF="vbgp/julia_set5.gif">image 1</A>,
<A HREF="vbgp/julia_set8.gif">image 2</A>,
<A HREF="vbgp/julia_set4.gif">image 3</A>).
Using these programs you can build intricate images with remarkably little code.
</DD>

<DT><B>11. Drawing Curves</B></DT>
<DD>
Chapter 11 shows how you can draw several useful kinds of curves.
It explains how to draw parametric curves such as
<A HREF="vbgp/bowditch_curve.gif">Bowditch curves</A> and
<A HREF="vbgp/cycloid.gif">cycloids</A>. It also shows
how to fit a curve to data using Hermite, least squares, and 
<A HREF="vbgp/bezier_curve.gif">Bezier curves</A>.
</DD>

<DT><B>12. Two-Dimensional Transformations</B></DT>
<DD>
This chapter explains a simple way to represent two-dimensional
transformations that allow you to perform important operations such
as scaling, translating, and rotating objects in a uniform way.
Chapter 12 also describes
<A HREF="vbgp/shape_distorting.gif">shape-distorting transformations</A>.
</DD>
</DL>
  <P>

  <TABLE WIDTH="100%" CELLPADDING="0" CELLSPACING="0">
    <TR><TD BGCOLOR="#7FCFFF" ALIGN="Center"><B>Part Five: Three-Dimensional Graphics</B></TD></TR>
  </TABLE>
  <P>

  <DL>
<DT><B>13. Three-Dimensional Transformations</B></DT>
<DD>
This chapter extends the two-dimensional transformations described 
in Chapter 12 into three dimensions. It explains how to display 
simple
<A HREF="vbgp/three_dimensional.gif">three-dimensional objects</A>.
</DD>


<DT><B>14. Surfaces</B></DT>
<DD>
Chapter 14 explains several different ways to generate and display
surfaces including
<A HREF="vbgp/fractal_surface.gif">fractal surfaces</A>,
<A HREF="vbgp/parametric_surface.gif">parametric surfaces</A>, and
<A HREF="vbgp/surface_of_transformation.gif">surfaces of transformation</A>.
By adding a little color, you can make surprisingly realistic
<A HREF="vbgp/valley.jpg">landscapes</A>.
</DD>


<DT><B>15. Hidden Surface Removal</B></DT>
<DD>
Three-dimensional images are easier to understand if hidden
surfaces are not displayed. Chapter 15 shows how to remove hidden 
surfaces from both
<A HREF="vbgp/hidden_surface_simple.gif">simple</A> and
<A HREF="vbgp/hidden_surface_complex.gif">complex</A> three-dimensional scenes.
</DD>


<DT><B>16. Shading Models</B></DT>
<DD>
Shading models add extra realism to three-dimensional images.
This chapter builds a complex shading model to produce
<A HREF="vbgp/shaded.gif">more realistic pictures</A>.
It also explains the 
<A HREF="vbgp/gouraud.jpg">Gouraud</A> and
<A HREF="vbgp/phong.jpg">Phong</A>
methods for smooth shading.
</DD>


<DT><B>17. Ray Tracing</B></DT>
<DD>
Ray tracing allows a program to draw reflective and transparent
objects. Chapter 17 develops an advanced ray tracing system
to display fascinating images.

<UL>
<LI><A HREF="vbgp/frosty.jpg">Frosty</A> (one of my favorites)
<LI><A HREF="vbgp/big_blue_marble.jpg">Big blue marble</A>
<LI><A HREF="vbgp/graphics_evolution.jpg">Graphics evolution</A>
<LI><A HREF="vbgp/ray_spheres1.jpg">Two overlapping</A>
<LI><A HREF="vbgp/ray_spheres2.jpg">Four overlapping</A>
<LI><A HREF="vbgp/ray_spheres3.jpg">Big and small</A>
<LI><A HREF="vbgp/ray_spheres4.jpg">Six orbiting</A>
<LI><A HREF="vbgp/ray_spheres5.jpg">Six overlapping</A>
</UL>

This chapter covers:

<UL>
<LI><A HREF="vbgp/Fig17-16.gif">Multiple light sources</A>.

<LI>A new <A HREF="vbgp/Fig17-21.jpg">checkerboard</A> object.

<LI><A HREF="vbgp/Fig17-31.gif">Texture mapping</A>.

<LI><A HREF="vbgp/Fig17-32.jpg">Environment mapping</A>.

<LI><A HREF="vbgp/Fig17-33.jpg">Image mapping to determine transparency</A>.

<LI>A script-driven
<A HREF="vbgp/Fig17-11.jpg">object editor</A>
that lets you preview scenes using wireframe or shading models.
</UL>
</DD>
</DL>
  <P>

  <TABLE WIDTH="100%" CELLPADDING="0" CELLSPACING="0">
    <TR><TD BGCOLOR="#7FCFFF" ALIGN="Center"><B>Part Six: Beyond Three Dimensions</B></TD></TR>
  </TABLE>
  <P>

  <DL>
<DT><B>18. Higher-Dimensional Transformations</B></DT>
<DD>
Chapter 18 extends the transformations described in Chapters 12 and 13
into higher dimensions. This allows a program to display
<A HREF="vbgp/four_d.gif">four-dimensional objects</A>,
and even objects of higher dimensions. This chapter explains how to 
use animation to make higher-dimensional images understandable.
</DD>


<DT><B>19. Mathematical Tools</B></DT>
<DD>
This chapter describes some useful mathematical formulas that
calculate geometric values such as the distance between two points,
between a point and a line, between two lines, and between a
point and a plane. Chapter 19 reviews matrix and vector operations,
and shows how to derive the locations of the vertices in the 
Platonic solids (tetrahedron, cube, octahedron, dodecahedron,
and icosahedron).
</DD>
</DL>
  <P>
