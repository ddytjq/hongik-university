<p><strong>Story:</strong></p>
<p><p>If you are looking for a good book on developing XML-enabled database applications, you should check out <A HREF="http://www.wrox.com/Books/Book_Details.asp?isbn=1861003587">Professional XML Databases</A>.</p><p align="center"><a href="http://www.wrox.com/Books/Book_Details.asp?isbn=1861003587" target="_top"><img src="/images/3587.gif" alt="Professional XML Databases" border="0"></a></p><p>This is an outstanding book. Of course, I'm biased, because I'm one of the authors. I wrote chapter 16, <i>JDBC</i>.  In this chapter, I walk through how to develop an XML Gateway Architecture for JDBC. I also discuss how to use XML to create distributed JDBC applications using the <i>javax.sql.Rowset</i> interface and the WebRowSet class.  Check it out, and let me know what you think.</p></p>
