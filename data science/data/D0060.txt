<p><FONT size=2><font color="#0000ff" face="Arial">Hoskinson's Rotating Ad Banner Applet allows you to display 
multiple advertising banners, in GIF, JPEG, or text format, each with their own 
hyperlink. &nbsp;With the Rotating Ad Banner Applet, you can:</font><font face="Arial">
</font></FONT>

<ol>
  <li><font face="Arial" size=2>Display 
  multiple ad banners, in either GIF, JPEG, or text format.
    </font>
  <li><font face="Arial" size=2>Assign 
  each ad banner it's own hyperlink and browser frame target.
    </font>
  <li><font face="Arial" size=2>Specify 
  the time interval at which banners are displayed. </font>
  <li><font face="Arial" size=2>Specify a 
  custom background, foreground, and link color for the applet container. </font>
  <li><font face="Arial" size=2>Specify a 
  custom intro message (displayed while the applet is retrieving images). 
 </font>
  <li><font face="Arial" size=2>Specify a 
  custom font for your intro message and text banners. </font>
  <li><font face="Arial" size=2>Specify an 
  optional 3D border size. </font>
  <li><font face="Arial" size=2>Package 
  your image files with the Applet byte code files into a Java Cabinet file, if 
  desired, to speed download for your Microsoft Internet Explorer 3.0x users. 
  </font></li>
</ol>

Banner applet in your HTML page (Note: this assumes that <i>adBanner_andyhosk.class </i>is in the same directory as it's parent HTML page). The parameters, by the 
way, correspond to the instance that is running on this page right now:

<h2><font face="Arial" size=2>Background and Foreground Color Attributes </font></h2>

you can set the background and foreground color of the applet to any one of the 16+
million colors in the 24 bit RGB (Red-Green-Blue) spectrum. But first, you must calculate
the value of your custom color. To facilitate this, please use this <A
 href="/java/RGB" >calculator</A>. </font></p>
