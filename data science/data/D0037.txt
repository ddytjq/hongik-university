      <p class="body"><span class="title">Create Compelling Java Applications 
        for Smart Devices?</span> <br>
      </p>
      <p class="body">Start by downloading free development tools here at devicetop.com. 
        Depending on your project, choose from the Espial Suite for TV or the 
        Espial Suite for Devices. </p>
      <p class="body">Not sure how to get started? Take the time to go through 
        the tutorials. Learn about each GUI component, RTKs, and 2D navigation 
        in the Espresso tutorial. Learn about building OSGi compliant applications 
        in the DeviceTop tutorial. </p>
      <p class="body">Another great resource to learn about Java basics and building 
        apps for iTV, is the <a href="http://www.javadevices.org/dtcourse">Java/iTV 
        Course</a>. Get up-to-speed by reading the lectures and labs at your own 
        pace. </p>
      <br>
      <p class="body"><span class="title11px">ESPIAL SUITE FOR TV, Community Edition</span><br>
      </p>
      <p class="body">The Espial Suite for TV, Community Edition is a 100% Java 
        application environment that sits independently from the operating system 
        and JVM on the upper layer of a STB software stack allowing developers 
        to build graphically rich applications that run in a TV set-top box environment. 
        It includes Espial DeviceTop, an operating environment, and Espial Espresso, 
        a lightweight GUI toolkit. <br>
        <br>
        <span class="title"><a href="/site/download?ID=espial_tv_ce.exe">DOWNLOAD 
        WINDOWS</a><br>
        <span class="title"><a href="/site/download?ID=espial_tv_ce.zip">DOWNLOAD

        <a href="/dt/tutorials/Espresso40Tutorial.zip">Espial Espresso Tutorial

        <a href="http://www.devicetop.com/site/forum/open_forum/forum-12">DeviceTop 
        Discussion Forum</a><br>
        <a href="http://www.devicetop.com/site/forum/open_forum/forum-8">Espresso 
        Discussion Forum</a></p>
      <p class="body"> </p>
      <p class="body"><span class="title11px">iTV Development Platform</span> 
      </p>
      <p class="body">This iTV development platform supports a subset of the DVB-MHP 
        specification which allows developers to test iTV applications, xlets,<br>
        in a device scenario. We encourage you to download this MHP Application 
        Simulator if you are building iTV applications.</p>
      <p class="body"><a href="/dt/tutorials/DVB-MHP-RefImpl.ZIP">DOWNLOAD</a><br>
        <a href="http://www.javadevices.org/dtcourse" target="_blank">Learn how 
        to build iTV apps in the Java/iTV Course</a>!</p>
      <p class="body"> </p>
      <p class="title11px">ESPIAL SUITE FOR DEVICES, Community Edition</p>
      <p class="body">The Espial Suite for Devices, Community Edition is a 100% 
        Java application environment that sits independently from the operating 
        system and JVM on the upper layer of an internet-enabled device stack 
        allowing developers to build graphically rich applications that run on 
        a smart Internet device. It includes Espial DeviceTop, an operating environment, 
        and Espial Espresso, a lightweight GUI development toolkit.</p>
      <p class="title"><a href="/site/download?ID=espial_device_ce.exe">DOWNLOAD 
        WINDOWS</a><br>
        <a href="/site/download?ID=espial_device_ce.zip">DOWNLOAD PURE JAVA</a></p>
      <p class="body"><a href="/dt/tutorials/DTopTutorial_Devices.zip">Espial 
        DeviceTop Tutorial for Devices</a><br>
        <a href="/DT/tutorials/Espresso353Tutorial.zip">Espial Espresso Tutorial 
        for Devices</a><br>
        <a href="http://www.devicetop.com/site/forum/open_forum/forum-12">DeviceTop 
        Discussion Forum</a> <br>
        <a href="http://www.devicetop.com/site/forum/open_forum/forum-8">Espresso 
        Discussion Forum</a></p>
      <p class="body"> </p>
      <p class="body"><span class="title11px">Espial Espresso</span></p>
      <p class="body">Espial Espresso, a 100% Java, lightweight GUI development 
        toolkit, simplifies the creation of customized, graphically rich, Java-based 
        applications and services for resource-constrained environments. Espresso 
        3.54 is specifically for devices and Espresso 4.01 is specifically for 
        iTV.</p>
      <p class="body"><a href="/site/download?ID=espresso_354.zip">Download Espial 
        Espresso 3.54 for Devices</a><br>
        <a href="dt/tutorials/Espresso354Tutorial.zip">Tutorial on Espial Espresso 
        3.54 for Devices</a></p>
      <p class="body"><a href="/dt/espial/espresso_TV_eval402.zip">Download Espial 
        Espresso 4.02 for TV</a><br>
        <a href="/dt/tutorials/Espresso40Tutorial.zip">Tutorial on Espial Espresso 
        4.01 for TV</a></p>
      <p class="body"> </p>
      <p class="body"><span class="title11px">Espial Escape</span></p>
      <p class="body">Espial Escape 4.8 is the first 100% Java browser SDK 
        to offer a scalable configuration matching the memory requirements of 
        a wide range of resource-constrained devices. With support for key Internet 
        standards (including HTTP, HTTPS, SSL, HTML4.0, XML, XSLT, JavaScript 
        and CSS), multimedia, and iTV features like fit-to-screen-width, minimum 
        font sizes, zoom controls and pointerless navigation, Escape provides 
        a full-featured browsing experience with a small core footprint. </p>
      <p class="body">We have built a custom UI for this version of Espial Escape, 
        which is to be used specifically for the Smart Device Developer Contest 
        3 and for people developing web content for TV. </p>

        for Java, release 3.0 software is an integrated development environment 
        (IDE) based on Java 2 Platform, Enterprise Edition and the open source 
        NetBeans Tools Platform. Written in the Java programming language, 
        it provides a superb environment for building and deploying web services. 
      </p>
      <p class="body">The Community Edition software is offered at NO CHARGE and 
        includes a complete and highly-integrated set of tools for teams of developers
