The mutant is transformed with a library of the targetorganism. Transformants whose mutant phenotype is corrected contain thetarget gene.</li><li>When complementation is not feasible, tagging mutagenesis can be used.</li><li><a NAME="TDNA"></a>Seeds of <i>Arabidopsis</i> are transformed at highefficiency with <a href="MG4373.html">T-DNA</a> using <i>Agrobacteria</i>.Resulting transformants are germinated and plantlets screened for mutantphenotypes. <a href="#Facts">Top</a></li><li><a NAME="wt"></a>DNAs such as <a href="../MGW3/MG322.html">transposableelements</a> and T-DNA that integrate into the chromosomal DNA can createmutations by interruption of the gene in which they insert. <a href="#Facts">Top</a></li><li>Further genetic manipulation may be needed after the initial tagging eventto assure that the tag is indeed associated with the mutant phenotype.<a href="#Facts">Top</a></li><li><a NAME="Tn"></a>When a transposable element transposes, it often integratesitself in genes, causing mutations that can be recognized by phenotype.<a href="#Facts">Top</a></li><li><a NAME="TagLib"></a>A genomic library is made from DNA of the tagged mutantorganism. <a href="#Facts">Top</a></li><li><a NAME="ScreenT"></a>The genomic library of the tagged mutant organismis screened by <a href="MG423.html">hybridization</a> with the taggingagent as <a href="MG425.html">probe</a>. <a href="#Facts">Top</a></li><li><a NAME="PlasRes"></a>DNA of the tagged mutant organisms is digested witha <a href="MG421.html">restriction enzyme</a> that does not cleave withinthe tagging agent. The fragments are circularized by ligation at diluteconcentrations and used to transform <i>E. coli</i>. <a href="#Facts">Top</a></li><li><a NAME="InvPCR"></a>Oligonucleotides complementary to the ends of thetagging agent and with their 3'-ends pointing away from the agent are usedto <a href="MG426.html">amplify</a> the sequences flanking the taggingagent.</li><li><a NAME="ScreenH"></a>Positive clones obtained from a rescued plasmid,inverse PCR or a tagged genomic library can be used as nucleic acid hybridizationprobes to screen a <a href="MG416.html">library</a> of genomic DNA to identifyclones containing the wild type gene. <a href="#Facts">Top</a></li><li><a NAME="Test"></a>Positively reacting genomic library clones need to befurther examined to :&nbsp;<img SRC="nav4.gif" USEMAP="#nav4B1F889C9" height=71 width=69 align=RIGHT>&nbsp;<map NAME="nav4B1F889C9"><area COORDS="2,56,25,71" SHAPE="rect" HREF="MG413.html"><area COORDS="4,29,26,50" SHAPE="rect" HREF="MG49.html"><area COORDS="24,3,44,27" SHAPE="rect" HREF="MG4.html"><area COORDS="43,29,58,56" SHAPE="rect" HREF="MG42.html"><area COORDS="26,30,40,56" SHAPE="rect" HREF="MG41.html"><area COORDS="42,59,66,69" SHAPE="rect" HREF="MG415.html"></map></li><ul><li>Identify where on the clone the gene is located.</li><li>Prove that the clone actually contains the gene.</li></ul></ul><h3>Further Information</h3><ul><li>In a variant of tagging mutagenesis, a population of tags is used to generatethe mutants.&nbsp; Each tag differs from the others in a pair of 20meroligonucleotides that serve as bar codes.&nbsp; The population of mutantsis subjected to a selective pressure.&nbsp; The bar codes in the survivorsare compared with those in the initial population.&nbsp; Any missing barcodes must derive from a mutation in a gene required to deal with the selectivepressure.&nbsp; The presence of the bar code allows its isolation fromthe initial population (<a href="http://www.ncbi.nlm.nih.gov:80/entrez/query.fcgi?cmd=Retrieve&db=PubMed&list_uids=11701889&dopt=Abstract">ref</a>).</li></ul><center><a NAME="bottom"></a><a href="MG413.html">Last</a> | <a href="MG41.html">Overview</a>| <a href="#Facts">Top</a> | <a href="MG415.html">Next</a></center><p><br><p><hr>This is page 414 of <a href="../MG01.html">Molecular Genetics</a> by<a href="http://biochem.okstate.edu/faculty/~melcher/UM.html">Ulrich Melcher</a>,&copy;1997, 1998, 2002<p><hr><address>E-mail inquiries to <a href="mailto:umelcher@biochem.okstate.edu">U.
