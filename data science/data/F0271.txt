<h2>
<u>Sample Program: Using If....Then....Else</u></h2>

<p><br><img SRC="math1e.gif" height=285 width=444>
<p>In this program, you have to draw three text boxes, three label boxes,
one image box
<br>and four command buttons.Change the of the first text box to x (text
box starting from the left), second text box to y and third text box to
z.&nbsp; Change the caption of label1 to "<b>+</b>"and the caption of label2
to "<b>=</b>".Clear the caption of label3 and change its name to display,
its purpose is to show that whether the answer is wrong or correct. Now,
change the caption of the four command buttons to<b> Start, Next, OK</b>
and <b>Stop, </b>you may also change their names following their captions
respectively.
<br>You also need to draw a line and insert an image in the image box as
shown in my program.
<br>Now let's look at the codes:
<p><font color="#000099">Private Sub</font> Stop_Click()
<p>End
<br><font color="#000099">End Sub</font>
<p><font color="#000099">Private Sub</font> Text1_Change()
<p><font color="#000099">End Sub</font>
<p><font color="#000099">Sub</font> Start_Click()
<br><font color="#009900">' To get any random integers from 0 to 100</font>
<br>Randomize Timer
<br>firstNum = Int(Rnd * 100) + 1
<br>secondNum = Int(Rnd * 100) + 1
<br>x.Text = Str$(firstNum)
<br>y.Text = Str$(secondNum)
<p><font color="#000099">End Sub</font>
<p><font color="#000099">Private Sub</font> Picture1_Click ( )
<p><font color="#000099">End Sub</font>
<p><font color="#000099">Private Sub</font> OK_Click( )
<p><font color="#000099">If</font> z.Text = x.Text + y.Text&nbsp; <font color="#000099">Then</font>
<br>display.Caption = "Correct"
<br>Image2.Visible = "true"
<br>Line1.Visible = "true"
<br><font color="#000099">Else</font>
<br>display.Caption = "Wrong"
<br>Image2.Visible = "false"
<br>Line1.Visible = "false"
<br>End If
<p><font color="#000099">End Sub</font>
<p><font color="#000099">Private Sub</font> Next_Click()
<br>z.Text = ""
<br>display.Caption = ""
<br>Start_Click
<p><font color="#000099">End Sub</font>
<p>if you would like the user to press the Enter key after typing the answer,
write an event procedure for the keyPress event as follows:
<p><font color="#000099">Private Sub</font> z_KeyPress(KeyAscii As Integer)
<p>If (KeyAscii = 13) Then
<br>OK_Click
<br>End If
<p><font color="#000099">End Sub</font>
<p>where <font color="#CC33CC">KeyAscii=13 </font>denotes the <font color="#993366">Enter</font>
key.
<p>After starting&nbsp; the program, the user (can be a teacher, parent
or a child himself) have to click on the Start button. Two numbers will
be presented in the first two text box. The user have to key in the answer
and click OK. The user will know whether the answer is correct or wrong
as it will be shown in the third label. To end, just click on the Stop
button.
<br>In this program, you may need to know what is the&nbsp; <b>Rnd and
Int </b>function<b> </b>. <b>Rnd</b> return a random number between 0 and
1 while <b>Int</b> returns the next smallest integer of a number.
<p><font color="#FF0000">Example:&nbsp;&nbsp; if Rnd=0.7423</font>
<br><font color="#FF0000">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
100*Rnd=74.23</font>
<br><font color="#FF0000">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
Int(100*Rnd)=74</font>
<p><font color="#000000">By combining <b>Rnd </b>and <b>Int</b> functions,
you can generate random integers of any range.</font>
<br><font color="#000000">The <b>Str$ </b>function is to convert a number
to<b> text&nbsp;</b> while <b>Val</b> function is to convert <b>text</b>
to <b>number</b>.</font>
