                <div align="left"><!-- #BeginEditable "right col" --> <b><font face="Verdana, Arial, Helvetica, sans-serif" size="2">Q: 
                  When can I get the source code?</font></b> <br>
                  <br>
                  <b><font face="Verdana, Arial, Helvetica, sans-serif" size="2">A: 
                  </font></b><font face="Verdana, Arial, Helvetica, sans-serif" size="2">The 
                  source code is here and available for <a href="/download/download_licenses.html">download</a>!</font><br>
                  <br>
                  <br>
                  <b><font face="Verdana, Arial, Helvetica, sans-serif" size="2">Q: 
                  Will there be a Watcom C/C++ and Fortran 11.0c release?</font></b> 
                  <br>
                  <br>
                  <b><font face="Verdana, Arial, Helvetica, sans-serif" size="2">A:</font></b> 
                  Yes! As part of the process of preparing the source code for 
                  release under an Open Source license, SciTech Software and a 
                  select group of external developers will create and distribute 
                  a binary patch upgrade release (11.0c) on the Open Watcom web 
                  site. This binary patch will be available free of charge to 
                  existing Watcom C/C++ and Fortran customers and will include 
                  all updates and bug fixes developed by Sybase since the last 
                  commercial update of the product (11.0b). <br>
                  <br>
                  <br>
                  <b><font face="Verdana, Arial, Helvetica, sans-serif" size="2">Q: 
                  Is everything that was on the Watcom CD available for free download?</font></b> 
                  <br>
                  <br>
                  <b><font face="Verdana, Arial, Helvetica, sans-serif" size="2">A: 
                  </font></b> No. The commercial versions of the Watcom C/C++ 
                  and Fortran compilers contained licensed copies of the Windows 
                  SDK, Microsoft Foundation Classes, OS/2 Toolkit and other commercial 
                  tools. Due to licensing issues, those components cannot be made 
                  freely available. Hence in order to utilise the 11.0c binary 
                  patch release, you will need an official copy of the 11.0, 11.0a 
                  or 11.0b installation CD. <br>
                  <br>
                  <br>
                  <b><font face="Verdana, Arial, Helvetica, sans-serif" size="2">Q: 
                  What compilers will I need to compile the source code?</font></b> 
                  <br>
                  <br>
                  <b><font face="Verdana, Arial, Helvetica, sans-serif" size="2">A:</font></b> 
                  Initially the Open Watcom 1.0 compiler will require an official 
                  copy of the Watcom C/C++ 11.0b compiler, with the 11.0c binary 
                  patch release applied in order to successfully compile it. This 
                  is due to the need to rely upon proprietry SDK's for platforms 
                  such as Windows, and OS/2 that cannot be distributed along with 
                  the Open Source 1.0 compiler release. It is planned to eliminate 
                  this dependency in the Open Source 2.0 release, by allowing 
                  the compiler to work with freely downloadable SDK's for the 
                  supported platforms. <br>
                  <br>
                  <br>
                  <br>

                  <b><font face="Verdana, Arial, Helvetica, sans-serif" size="2">Q: 
                  What does the Watcom compiler give us that we don't already 
                  get with other free compilers?</font></b> <br>
                  <br>
                  <b><font face="Verdana, Arial, Helvetica, sans-serif" size="2">A:</font></b> 
                  The Watcom compiler has a number of advantages of other free 
                  compilers, such as full support for 16-bit DOS and Windows development, 
                  powerful commercial level debugger with full remote debugging 
                  capabilities (serial, parallel, NetBIOS, TCP/IP etc). The Watcom 
                  compiler has always been known as producing some of the fastest 
                  code around, and even today the Watcom C/C++ 11.0b product produces 
                  code as good as the best comercial compilers available. The 
                  Watcom C/C++ compiler is also the only freely available compiler 
                  for 16-bit Windows and OS/2 device driver development. <br>
                  <br>
                  <br>
                  <br>
                  <b><font face="Verdana, Arial, Helvetica, sans-serif" size="2"> 
                  Q: I really want to contribute! What can I do! </font></b> <br>
                  <br>
                  <b><font face="Verdana, Arial, Helvetica, sans-serif" size="2"> 
                  A:</font></b> Until the Open Watcom 1.0 compiler release is 
                  ready, all development will be closed due to proprietary source 
                  code licensing issues. Until the Open Watcom 1.0 source code 
                  is ready for release, we are not accepting any new members on 
                  the Open Watcom development team. <br>
