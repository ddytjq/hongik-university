<p><font face="Arial, Helvetica" size="+1"><b>Beginning databases</b></font></p>
<p><i>By <a href="/php-bin/authorbio.php?authorid=31&associd=58&assoctype=2">Sam Huggill</a></i></p>
<p><font face="Arial, Helvetica" size="+1"><b>Introduction</b></font></p>

<p>
There are many different ways in which you can access
databases. The latest, and probably best is ADO (Active X Data Objects). I prefer using
DAO, as you will see in this HowTo.
</p>

<p>
When using DAO you must remember that a table is called a
'recordset' and is declared in that way. This recordset exposes several methods that we
can use to manipulate a database. The Edit, AddNew and Delete methods will be used in this
HowTo for the main functions.
</p>

<p>
Before you start make sure that you have a reference to
Microsoft DAO Object Library. Do this by clicking Project, References. Scroll down until
you see Microsoft DAO 3.x Object Library and check the box next to it. Then click ok.
</p>

<p>
Add a new form (frmTest). Add four command buttons, cmdAdd,
cmdDelete, cmdGet and cmdUpdate. Add three text boxes, txtName, txtPhone and txtNotes.
Copy this code into the form:
</p>

</font><pre>
<font color="#007F00">'Private declarations for the database and recordset </font>
<font color="#00007F">Private</font> DB <font color="#00007F">As</font> Database 
<font color="#00007F">Private</font> RS <font color="#00007F">As</font> Recordset 

<font color="#00007F">Function</font> EditRecord(sName <font color="#00007F">As</font> <font color="#00007F">String</font>) 
  <font color="#007F00">'Open the Contact table  </font>
  <font color="#00007F">Set</font> RS = DB.OpenRecordset(&quot;SELECT * FROM Contact WHERE <font color="#00007F">Name</font> _
           = <font color="#007F00">'&quot; &amp; sName &amp; &quot;'&quot;, dbOpenDynaset) </font>

  <font color="#00007F">With</font> RS 
    <font color="#007F00">'Set it in edit more  </font>
    .Edit 

    <font color="#007F00">'Update the following fields  </font>
    .Fields(&quot;Name&quot;).Value = txtName.Text 
    .Fields(&quot;Phone&quot;).Value = txtPhone.Text 
    .Fields(&quot;Notes&quot;).Value = txtNotes.Text 

    <font color="#007F00">'Update the table  </font>
    .Update 

    <font color="#007F00">'Close it  </font>
    .<font color="#00007F">Close</font> 
  <font color="#00007F">End</font> <font color="#00007F">With</font> 

<font color="#00007F">End</font> <font color="#00007F">Function</font> 

<font color="#00007F">Function</font> GetFromDatabase(sName <font color="#00007F">As</font> <font color="#00007F">String</font>) 
  <font color="#007F00">'Open the Contact table  </font>
  <font color="#00007F">Set</font> RS = DB.OpenRecordset(&quot;SELECT * FROM Contact WHERE <font color="#00007F">Name</font> _
           = <font color="#007F00">'&quot; &amp; sName &amp; &quot;'&quot;, dbOpenDynaset) </font>
  <font color="#007F00">'Put the data into the text boxes </font>
  <font color="#007F00">'Use the &quot;&quot; &amp; RS! so we don't get an error </font>
  <font color="#007F00">'if one of the fields is blank  </font>
  txtName = &quot;&quot; &amp; RS!Name 
  txtPhone = &quot;&quot; &amp; RS!Phone 
  txtNotes = &quot;&quot; &amp; RS!Notes 
  <font color="#007F00">'Close the table  </font>
  RS.<font color="#00007F">Close</font> 
<font color="#00007F">End</font> <font color="#00007F">Function</font> 

<font color="#00007F">Function</font> AddToDatabase(sName <font color="#00007F">As</font> <font color="#00007F">String</font>, sPhone <font color="#00007F">As</font> <font color="#00007F">String</font>, sNotes <font color="#00007F">As</font> <font color="#00007F">String</font>) 

  <font color="#007F00">'Open the Contact table  </font>
  <font color="#00007F">Set</font> RS = DB.OpenRecordset(&quot;SELECT * FROM Contact&quot;, dbOpenDynaset) 

  <font color="#00007F">With</font> RS 
    <font color="#007F00">'Set it to <font color="#00007F">Add</font> mode  </font>
    .AddNew 

    <font color="#007F00">'Enter the field values </font>
    .Fields(&quot;Name&quot;).Value = sName 
    .Fields(&quot;Phone&quot;).Value = sPhone 
    .Fields(&quot;Notes&quot;).Value = sNotes 

    <font color="#007F00">'Update it  </font>
    .Update 

    <font color="#007F00">'Close it  </font>
    .<font color="#00007F">Close</font> 

  <font color="#00007F">End</font> <font color="#00007F">With</font> 

<font color="#00007F">End</font> <font color="#00007F">Function</font> 

<font color="#00007F">Function</font> DeleteFromDatabase(sName <font color="#00007F">As</font> <font color="#00007F">String</font>) 
  <font color="#007F00">'Open the Contact table  </font>
  <font color="#00007F">Set</font> RS = DB.OpenRecordset(&quot;SELECT * FROM Contact WHERE <font color="#00007F">Name</font> _
            = <font color="#007F00">'&quot; &amp; sName &amp; &quot;'&quot;, dbOpenDynaset) </font>

  <font color="#00007F">With</font> RS 
    <font color="#007F00">'Set it to delete mode  </font>
    .Delete 

    <font color="#007F00">'Close it  </font>
    .<font color="#00007F">Close</font> 

  <font color="#00007F">End</font> <font color="#00007F">With</font> 

<font color="#00007F">End</font> <font color="#00007F">Function</font> 
 
<font color="#00007F">Private</font> <font color="#00007F">Sub</font> cmdAdd_Click() 

  <font color="#007F00">'Declare variables  </font>
  <font color="#00007F">Dim</font> sName <font color="#00007F">As</font> <font color="#00007F">String</font> 
  <font color="#00007F">Dim</font> sPhone <font color="#00007F">As</font> <font color="#00007F">String</font> 
  <font color="#00007F">Dim</font> sNotes <font color="#00007F">As</font> <font color="#00007F">String</font> 

  <font color="#007F00">'Ask for the data  </font>
  sName = InputBox$(&quot;Please enter your name:&quot;, &quot;Database&quot;) 
  sPhone = InputBox$(&quot;Please enter your phone number:&quot;, &quot;Database&quot;) 
  sNotes = txtNotes 

  <font color="#007F00">'Call the function  </font>
  <font color="#00007F">Call</font> AddToDatabase(sName, sPhone, sNotes) 

<font color="#00007F">End</font> <font color="#00007F">Sub</font> 
 
<font color="#00007F">Private</font> <font color="#00007F">Sub</font> cmdDelete_Click() 
  <font color="#007F00">'Declare the variable  </font>
  <font color="#00007F">Dim</font> sName <font color="#00007F">As</font> <font color="#00007F">String</font> 

  <font color="#007F00">'Ask for the data  </font>
  sName = InputBox$(&quot;Please enter your name:&quot;, &quot;Database&quot;) 

  <font color="#007F00">'Call the function  </font>
  <font color="#00007F">Call</font> DeleteFromDatabase(sName) 

<font color="#00007F">End</font> <font color="#00007F">Sub</font> 

<font color="#00007F">Private</font> <font color="#00007F">Sub</font> cmdGet_Click() 
  <font color="#007F00">'Declare the variable  </font>
  <font color="#00007F">Dim</font> sName <font color="#00007F">As</font> <font color="#00007F">String</font> 

  <font color="#007F00">'Ask for the data  </font>
  sName = InputBox$(&quot;Please enter your name:&quot;, &quot;Database&quot;) 

  <font color="#007F00">'Call the function  </font>
  <font color="#00007F">Call</font> GetFromDatabase(sName) 
<font color="#00007F">End</font> <font color="#00007F">Sub</font> 
 
<font color="#00007F">Private</font> <font color="#00007F">Sub</font> cmdRecord_Click() 
  <font color="#007F00">'Call the function  </font>
  <font color="#00007F">Call</font> EditRecord(txtName) 
<font color="#00007F">End</font> <font color="#00007F">Sub</font> 
 
<font color="#00007F">Private</font> <font color="#00007F">Sub</font> Form_Load() 
  <font color="#007F00">'Use the OpenDatabase function to open our database  </font>
  <font color="#00007F">Set</font> DB = OpenDatabase(App.Path &amp; &quot;\demo.mdb&quot;) 
<font color="#00007F">End</font> <font color="#00007F">Sub</font> 
 
<font color="#00007F">Private</font> <font color="#00007F">Sub</font> Form_Unload(Cancel <font color="#00007F">As</font> <font color="#00007F">Integer</font>) 
  <font color="#007F00">'Set the variables to Nothing  </font>
  <font color="#00007F">Set</font> RS = Nothing 
  <font color="#00007F">Set</font> DB = Nothing 
<font color="#00007F">End</font> <font color="#00007F">Sub</font>
