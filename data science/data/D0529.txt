<SPAN Class="smbl">
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;A lot of Java applets pop up seperate "applet windows".
There is often the message "Warning: Java Applet Window" or 
something like that at the bottom of an applet window. These
"applet windows" are actually Frames. This tutorial will teach
you the basics of frames so that you can use them in your
applets. First, import the gui components (java.awt.*)
Then, put the following line after the beginning of your 
applet (the first bracket), but before the first  method 
(usually before init() ).<br><br>

Frame name;
<br><br>
	&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Replace "name" in the line above with what you want
the name of your Frame to be. Then put the following line in
init() :<br><br>

name=new Frame("   ");<br><br>

	&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Replace "name" with the name you used in the line discussed
above. Also put the title of your Frame (the text in the bar
at the top of the Frame) in between the quotes in the line 
above. Now you have a frame, but it is invisible, and it
has no set size. So in init(), after the name=new Frame line,
put:<br><br>

name.setSize(x,y);<br>
name.setVisible(true);<br><br>

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;	Again, replace name with the name of your Frame. In the
first line, replace x with how wide you want your Frame to be
in pixels, and y with how tall you want your frame to be in
pixels. If you compiled and ran your applet now, the frame
would come up. However, it wouldn't close.....which is a
problem. The first step is to implement WindowListener. To do
this, put this after "extends Applet" :<br><br>

implements WindowListener<br><br>

	&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Now add the following lines (they are methods that go
with WindowListener) to your applet:<br><br><pre>


    public void windowDeactivated(WindowEvent e) {
    }
    public void windowClosed(WindowEvent e) {
    }
    public void windowDeiconified(WindowEvent e) {
    }
    public void windowOpened(WindowEvent e) {
    }
    public void windowIconified(WindowEvent e) {
    }    
    public void windowActivated(WindowEvent e) {
    }
    public void windowClosing(WindowEvent e) {
        name.setVisible(false);
    }</pre>

	&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;The code in the windowClosing method is what closes
the window (by making it invisible). Replace "name" with
the name of your Frame. Then, in init, after everything
else in init() mentioned before, put:<br><br>

name.addWindowListener(this);<br><br>

	&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;That's it! Now your Frame is fully functional, and
you can add things to it! Just do name.add(whateveryouwanttoadd);
It's great! Enjoy!
<br><br>
<applet code="frametut.class" width=200 height=200></applet><br><br>
<pre>
//-------------------------------//
//Copyright 2000 FreeJavaHelp.com//
//-------------------------------//

//the next line imports the GUI components
import java.awt.*;
// the next line imports the event classes
import java.awt.event.*;
//the next line imports the Applet class
import java.applet.Applet;
//the next line begins the applet named frametut
public class frametut extends Applet implements WindowListener{
    //the next line declares a frame
    Frame main;
    //the init() method occurs when the applet is initialized
    public void init () {
        //the next line makes main into a new Frame 
        //with the title "Window Title Here"
        main=new Frame("Window Title Here");
        //the next line makes the frame sets the size
        //to 500 pixels wide and tall
        main.setSize(500,500);
        //the next line adds a WindowListener to main
        main.addWindowListener(this);
        //the next line makes the Frame named main visible
        main.setVisible(true);
    }
    //the next six methods are needed to implement WindowListener
    public void windowDeactivated(WindowEvent e) {
    }
    public void windowClosed(WindowEvent e) {
    }
    public void windowDeiconified(WindowEvent e) {
    }
    public void windowOpened(WindowEvent e) {
    }
    public void windowIconified(WindowEvent e) {
    }    
    public void windowActivated(WindowEvent e) {
    }
    //the next line makes it so that when the X in the
    //corner of the Frame named main is clicked, the
    //Frame is not visible anymore
    public void windowClosing(WindowEvent e) {
        main.setVisible(false);
    }
}
