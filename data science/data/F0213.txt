<h4>
7.1&nbsp; Conditional Operators</h4>
To control the VB program flow, we can use various conditional operators.
Basically, they resemble mathematical&nbsp; operators. Conditional operators
are very powerful tools, they let the VB program compare data values and
then decide what action to take, whether to execute a program or terminate
the program and etc. These operators are shown in Table 7.1.

<h4>
<font color="#000000">7.3&nbsp; Using&nbsp; If.....Then.....Else&nbsp;
Statements&nbsp; with Opreators</font></h4>
<font color="#000000">To effectively control the VB program flow, we shall
use If...Then...Else statement together with the conditonal operators and
logical operators.</font>
<br>The general format for the if...then...else statement is
<blockquote><b><font color="#3366FF">If</font>&nbsp; </b>conditions <b><font color="#3366FF">Then</font></b>
<p>VB expressions
<p><b><font color="#3366FF">Else</font></b>
<p>VB expressions
<p><b><font color="#3366FF">End If</font></b></blockquote>
* any If..Then..Else statement must end with End If. Sometime it is not
necessary to use Else.
<p><font color="#000000">Example:</font>
<p><font color="#000000">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
Private Sub OK_Click()</font>
<blockquote><font color="#000000">firstnum = Val(usernum1.Text)</font>
<br><font color="#000000">secondnum = Val(usernum2.Text)</font>
<br><font color="#000000">total = Val(sum.Text)</font>
<br><font color="#000000">If total = firstnum + secondnum And Val(sum.Text)
&lt;> 0 Then</font>
<br><font color="#000000">correct.Visible = True</font>
<br><font color="#000000">wrong.Visible = False</font>
<br><font color="#000000">Else</font>
<br><font color="#000000">correct.Visible = False</font>
<br><font color="#000000">wrong.Visible = True</font>
<br><font color="#000000">End If</font>
<p><font color="#000000">End Sub</font></blockquote>
For more example on If...Then...Else, Click on the <a href="sample_lesson7.html">Sample</a>
program here.
