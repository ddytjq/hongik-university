<Table width='100%' border=0 cellpadding=5><TR><TD id=RevImage rowspan=6 valign=top width=5% ><img src='netreviewpics\0672321432.jpg' border='0' align=top ></TD><TD  id=RevItemName><FONT color=navy size=6>ASP .NET Tips, Tutorials and Code</FONT></TD></TR><TR><TD id=RevItemAuthor >By:  Scott Mitchell</TD></TR><TR><TD id=RevItemISBN>ISBN: 0672321432</TD></TR><TR><TD id=RevItemISBN>Publisher: <A href='http://www.samspublishing.com/detail_sams.cfm?item=0672321432' target='_blank'>Sams Publishing</A></TD></TR><TR><TD id=RevItemISBN>&nbsp;</TD></TR><TR ><TD >&nbsp;</TD></TR></Table><Table width='100%' border=1 cellpadding=5><TR><TD  id=RevTitle><FONT color=navy size=6>Jump right in</FONT></TD></TR><TR><TD id=RevAuthor >Reviewed By:  <a name=KunjalKaramshi><A href='Reviewer.asp?Rid=5&ISBN=0672321432' >Kunjal Karamshi</a></TD></TR><TR><TD id=Review><P>This book turned out to be an extremely interesting read. 
I liked the way it just jumps right into common form and code techniques for 
using ASP.NET. However the one thing that stands out about this book is its 
breadth of coverage of ASP.NET which I found to be excellent.</P>
<P>The book covers a lot of the common programming problems that are faced by 
ASP programmers and presents sample code to solve these problems. As far as the 
coverage of the book goes, the book goes from validating data on forms, 
manipulating data with ADO.NET and data presentation using the various controls 
that are present in the ASP.NET environment. Error handling and debugging in 
ASP.NET is covered quite well. The section on ASP.NET mobile controls was also 
very interesting and gives a brief introduction to developing applications for 
mobile devices. </P>
<P>I found the appendices very useful. They give advice on how to migrate 
existing ASP code to .NET. They also give common field validation examples and 
SQL stored procedure templates that can be used as a quick reference to solve 
common problems. </P>
<P>All said, this is a great book for a programmer who already has some ASP 
experience and is looking for a way to upgrade to ASP.NET. </P></TD></TR></Table>
