<p>This is a basic tutorial on HTML and building a web page from scratch. Although most people will say it is simpler to use an HTML editor to create a web page, if you don't understand what is going on behind the web page, it is like not knowing that certain variations of the alphabet create words, just that words are words. You need to learn the alphabet to create words. You also need to learn what numbers create new numbers. If you know what each HTML object does and how to make it work for you, then you'll understand what the editors are actually doing for you. Evenmore, some editors are limited to what they can do. With your knowledge of HTML, you will be able to create a simple page with an editor, and then turn it into a page you can be proud of. This tutorial will take you from a simple template with demos you can see as you go through each lesson and guide you to create a somewhat professional looking Web page.
