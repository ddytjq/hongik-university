<h2>
Lesson: User Interfaces that Swing: A Quick Start Guide
</h2>
<blockquote>
<!--User Interfaces that Swing: A Quick Start Guide-->


<!--Up to this point, most of the programs we've written haven't 
been anything pretty to look at. More important, our programs 
haven't enabled user interaction. We have come to expect 
software programs to have buttons and text areas and things 
for us to interact with.-->

This lesson is a quick start guide to using
Swing components to build user interfaces. This lesson is built 
around several progressively complicated examples. 
 
<blockquote><hr><strong>Note:</strong>&nbsp;This material is covered in depth in the full Swing trail: 
<a target="_top" href="../../uiswing/index.html">Creating a GUI with JFC/Swing</a><a target="_top" href="../../uiswing/index.html"><img src="../../images/tutorialIcon.gif" width=11 height=11 border=0 align="ABSMIDDLE" alt="(in the Creating a GUI with JFC/Swing trail)"></a>. This lesson is designed for readers who want to learn
the basics quickly by examining some simple examples. 
<hr></blockquote>

<P>
We cover many standard graphical user interface (GUI) components that 
the Java<sup><font size=-2>TM</font></sup> 2 platform 
provides, such as buttons, labels, 
and text areas. The handling of events is also discussed, 
as are layout management and accessibility.  

<blockquote><hr><strong>Note:</strong>&nbsp;You can download all the examples (including images) used in this trail 
from the 
<A HREF="example-1dot3/index.html">Example Index</A>.
<hr></blockquote>

<dl>
<dt><B>
<a target="_top" href="overview.html">Overview of the Swing API</a><a target="_top" href="overview.html"><img src="../../images/tutorialIcon.gif" width=11 height=11 border=0 align="ABSMIDDLE" alt="(in the Creating a GUI with JFC/Swing trail)"></a></B></dt>

<dd>
This page gives a quick overview of the history of the Swing 
package and how to compile and run Swing programs. 
<P></dd>

<dt><B>
<a target="_top" href="firstexample.html">Your First Swing Program</a><a target="_top" href="firstexample.html"><img src="../../images/tutorialIcon.gif" width=11 height=11 border=0 align="ABSMIDDLE" alt="(in the Creating a GUI with JFC/Swing trail)"></a></B></dt>

<dd>
This small example has all the code that every Swing application  
must have.
<P></dd>

<dt><B>
<a target="_top" href="secondexample.html">Example Two: <code>SwingApplication</code></a><a target="_top" href="secondexample.html"><img src="../../images/tutorialIcon.gif" width=11 height=11 border=0 align="ABSMIDDLE" alt="(in the Creating a GUI with JFC/Swing trail)"></a></B></dt>

<dd>
This section takes you through the code for a small Swing application.
The topics covered include: look and feel, setting up buttons and labels, 
handling events, and adding borders. 
<P></dd>

<dt><B>
<a target="_top" href="thirdexample.html">Example Three: <code>CelsiusConverter</code></a><a target="_top" href="thirdexample.html"><img src="../../images/tutorialIcon.gif" width=11 height=11 border=0 align="ABSMIDDLE" alt="(in the Creating a GUI with JFC/Swing trail)"></a></B></dt>
<dd>
The third example, a program that converts a given temperature 
from Celsius to Fahrenheit, illustrates how to add HTML and icons
to components.
<P></dd>

<dt><B>
<a target="_top" href="fourthexample.html">Example Four: <code>LunarPhases</code></a><a target="_top" href="fourthexample.html"><img src="../../images/tutorialIcon.gif" width=11 height=11 border=0 align="ABSMIDDLE" alt="(in the Creating a GUI with JFC/Swing trail)"></a></B></dt>

<dd>
The fourth example covers how to set up multiple panels, 
compound borders, combo boxes, and how to load multiple images.
<P></dd>

<dt><B>
<a target="_top" href="fifthexample.html">Example Five: <code>VoteDialog</code></a><a target="_top" href="fifthexample.html"><img src="../../images/tutorialIcon.gif" width=11 height=11 border=0 align="ABSMIDDLE" alt="(in the Creating a GUI with JFC/Swing trail)"></a></B></dt>

<dd>
You can vote for your favorite political candidate in the final example,
<code>VoteDialog</code>. This application illustrates the use of radio 
buttons and dialogs.
<P></dd>

<dt><B>
<a target="_top" href="layout.html">Layout Management</a><a target="_top" href="layout.html"><img src="../../images/tutorialIcon.gif" width=11 height=11 border=0 align="ABSMIDDLE" alt="(in the Creating a GUI with JFC/Swing trail)"></a></B></dt>

<dd>
This section discusses how to position the components in a container.
<P></dd>

<dt><B>
<a target="_top" href="threads.html">Threads and Swing</a><a target="_top" href="threads.html"><img src="../../images/tutorialIcon.gif" width=11 height=11 border=0 align="ABSMIDDLE" alt="(in the Creating a GUI with JFC/Swing trail)"></a></B></dt>

<dd>
Thread issues are discussed in this section, including the "Single-Thread Rule."
<P></dd>
