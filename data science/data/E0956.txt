<H2>Notes</H2>
<DL>
<DT><B><TT>NULL</TT></B>
<DD><TT>NULL</TT> is a defined constant used to express null pointers, that is,
an unassigned pointer or a pointer that points nowhere. It is defined as:
<BLOCKQUOTE><TT>#define NULL 0</TT></BLOCKQUOTE>
<DT><B><TT>size_t</TT></B>
<DD>
Defined type used as arguments for some functions that require sizes or counts
specifications. This represents an unsigned value generally
defined in header files as <TT>unsigned int</TT>:
<BLOCKQUOTE><TT>typedef unsigned int size_t;</TT></BLOCKQUOTE>
</DL>
