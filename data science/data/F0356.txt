<h2>
<b><font color="#000000">Lesson 3 : Writing the Codes</font></b></h2></center>
<font color="#000000">Now we shall attempt to write the codes for the cylinder
program.</font><font color="#000000"></font>
<center>
<p><img SRC="CYLINDER.gif" height=253 width=397></center>
<b><font color="#800080"></font></b>
<p><font color="#000000">Now, doubleclick on the O.K button and enter the
codes between Private Sub OK_Click( ) and End Sub</font>
<ul>
<dd>
<b><font color="#FF00FF">Private Sub OK_Click( )</font></b></dd>

<dd>
<b><font color="#FF00FF">r = Val(radius.Text)</font></b></dd>

<dd>
<b><font color="#FF00FF">h = Val(hght.Text)</font></b></dd>

<dd>
<b><font color="#FF00FF">pi = 22 / 7</font></b></dd>

<dd>
<b><font color="#FF00FF">v = pi * (r ^ 2) * h</font></b></dd>

<dd>
<b><font color="#FF00FF">volume.Text= Str$(v)</font></b></dd>

<dd>
<b><font color="#FF00FF">End Sub</font></b></dd>
</ul>
<font color="#000000">when you run the program , you should be able to
see the interface as shown above. if you enter a value each in the radius
box and the height box, then click OK, the value of of the Volume will
be displayed in the volume box.</font>
<p>I shall attempt to explain the above source program to newcomers in
Visual Basic( If you are a veteran, you can skip this part) . Let me describe
the steps using pseudocodes as follows:
<dd>
<b><i><font color="#FF00FF">Procedure for clicking the OK button to calculate
the volume of cylinder</font></i></b></dd>

<dd>
<b><i><font color="#FF00FF">get the value of r from the radius text box</font></i></b></dd>

<dd>
<b><i><font color="#FF00FF">get the value of h from the height text box</font></i></b></dd>

<dd>
<b><i><font color="#FF00FF">assign a constant value 22/7 to pi</font></i></b></dd>

<dd>
<b><i><font color="#FF00FF">calculate the volume using formula</font></i></b></dd>

<dd>
<b><i><font color="#FF00FF">output the results to the Volume text box</font></i></b></dd>

<dd>
<b><i><font color="#FF00FF">End of Procedure</font></i></b></dd>

<br>&nbsp;
<p>&nbsp;
<p><font color="#400040">The syntax <b><i>radius.Text </i></b>consists
of two parts, radius is the <b>name </b>of text box while <b>Text</b> is
the textual contents of the text box. Generally, the syntax is: </font><b><font color="#000000">Object.Property</font></b>
<br><font color="#400040">In our example, the objects are<b><i> radius,
hght </i></b>and <b><i>volume</i></b>, each having <b><i>text </i></b>as
their <b><i>property</i></b>.Object and property is separated by a period(or
dot).The contents of a text box can only be displayed in textual form,
or in programming term,as string. To convert the contents of a text box
to a numeric value so that mathematical operations can be performed , you
have to use the function <b><i>Val. </i></b>Finally,<b><i> </i></b>In order
to display the results in a text box, we have to perform the reverse procedure,
that is, to convert the numeric value back to the textual form, using the
function <b>Str$.</b></font>
<p><font color="#400040">I shall also explain the syntax that defines the
sub procedure <b>Private Sub OK_click.</b> <i>Private Sub </i>here means
that the parameters , values and formulas that are used here belong only
to the OK subprocedure(an object by itself).They cannot be used by other
sub procedures or modules. OK_Click defines what kind of action the subprocedure
OK will response .Here, the action is mouse click. There are other kind
of actions like keypress, keyup, keydown and etc that I am going to due
with in other lessons.</font><font color="#400040"></font>
