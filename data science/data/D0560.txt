<font face="Arial">
<p><FONT face=Arial>Have you ever wondered what kind of 
HTTP server software a particular Web site is running?&nbsp; Now you can find 
out with Hoskinson's Web server query tool.&nbsp; This tool is written 
completely in Java, and will run in any JDK 1.1 - compliant VM.&nbsp; I have 
tested it using the Sun JDK 1.2, and the Microsoft VM Build 
5.00.3167.</FONT></p>
<p><FONT color=#000080 face=Arial size=4>How does this tool 
work?</FONT></p>
<p><FONT face=Arial>First, this tool confirms the identity 
of the host by doing a name server lookup.&nbsp; Second, it attempts to connect 
to an HTTP daemon running on port 80.&nbsp; If successful, it gets the HTTP 
"Server" header (which is the name of the Web server software), and displays 
it&nbsp;to the end user. </FONT></p>
<p align="center"><FONT face=Arial><IMG alt="Screen shot" src="80.jpg"></FONT></p>
<p><FONT face=Arial>If the server in question is running 
HTTP on another port, you can still query it using this tool.&nbsp; Simply 
append the port number to the host name (separated by a ":"), as shown 
below:</FONT></p>
<p align="center"><FONT face=Arial><IMG alt="Screen shot" src="8080.jpg"></FONT></p>

<p><FONT face=Arial>If you are interested in using this 
tool, read both the <A href="/java/disclaimer.asp">disclaimer</A> and the <A 
href="/java/license.asp">license</A>, and then <A 
href="qryHTTPd.class">download</A> the tool and the <A 
href="qryHTTPd.java">source code</a>.&nbsp; If you are a Java 2 user (JDK 1.2.x), you can download 
an <A href="qryHTTPd.jar">executable JAR</A> version, and if you are a Microsoft VM user, you can 
download a <A href="qryHTTPd.exe">Win32 EXE</A> version.  To run it, execute one of the 
following commands:</FONT></p>

<p><FONT face=Arial><FONT color=#000080 size=4>Sun 
JDK</FONT>:</FONT></p>
<blockquote>
<p><FONT face=Arial>java qryHTTPd</FONT></p>

<p><FONT face=Arial>---- or ----</FONT></p>

<p><FONT face=Arial>javaw qryHTTPd</FONT></p>
</blockquote>

<p><FONT face=Arial><FONT color=#000080 size=4>Microsoft 
VM</FONT>:</FONT></p>
<blockquote>

<p><FONT face=Arial>jview qryHTTPd</FONT></p>

<p><FONT face=Arial>---- or ----</FONT></p>

<p><FONT face=Arial>wjview qryHTTPd</FONT></p>
</blockquote>
