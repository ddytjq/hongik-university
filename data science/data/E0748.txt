<CENTER><H3><FONT color=#a0a099>
Platform independent extensible log class
</FONT></H3></CENTER>
<HR>

<!-- Author and contact details -->
This article was contributed by 
<A href="mailto:bloodchen@hotmail.com">Chen Mingjie</A>.

<P><U>Environment:</U> VC6, BC55, Win2000 SP2, g++, Redhat Linux 7.1


<h3>Introduction:</h3>
<P>The ability to log is commonly needed in every software project on every platform. I wrote this class to save time.</p>

<p>There are two basic log classes provided for easy use. One is CFileLog, which implements a file logging system. The other is CRegFileLog that implements a registry controled file logging system. The whole logging system is quit easy to extended for any propose.

<h3>How to use:</h3>

<p>Just include MjLog.h and construct an MjTools::CFileLog or MjTools::CRegFileLog instance. When you want to add a log message, call yourObject.AddLog("Log message").

<h3>Demo code:</h3>
<pre>
#include "MjLog.h"
int main()
{
  MjTools::CFileLog m_Log("test.log");
  std::string a="aaa";
  m_Log.Clear();
  m_Log.AddLog("Abc");
  m_Log.AddLog(a);
  MjTools::CFileLog m_Log1=m_Log;
  m_Log1.AddLog("From Log1");
#ifdef WIN32 <span class="codeComment">//RegistryLogControl only valid in Windows system</span>
  <span class="codeComment">// construct a registry key controled log object. If 
  // the specified registry key is found,the log is enabled</span>
  MjTools::CRegFileLog m_regLog("reglog.log",
         "HKEY_LOCAL_MACHINE\\Software\\YourControlKeyName");
  m_regLog.AddLog("reglog");
  m_regLog.Pause();
  m_regLog.AddLog("reglog1");
  m_regLog.Resume();
  m_regLog.AddLog("reglog2");
#endif
  return 0;
}
</PRE>

<h3>How to compile:</h3>
<p>The source code itself can be compiled and executed. You can use command line tool to compile it. </p>

<p><u>Under VC++:</u></p>
<pre>CL /D"_TEST_" MjLog.cpp </pre>
<p><i>This one may cause a link error. I don't know why.But if you use a win32 console project, no error occurs</i></p>

<p><u>Under BCC:</u></p>
<PRE>bcc32 /D_TEST_ mjlog.cpp</PRE>

<P><u>Under Linux:</u></p>
<pre>g++ /D_TEST_ MjLog.cpp</PRE>
