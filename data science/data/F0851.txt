<CENTER><B><FONT COLOR="#FF0000">
Adding a file to the most recent documents list
<BR><HR SIZE=1 WIDTH=200>
</FONT></B></CENTER><P>
<BR>
You may have noticed that when you click the 'Start' button in Windows, you can access a list of the most recently used documents. Well, if your application makes use of any sort of data file, you may want to add a document to that list, too.<BR>
You can do this by using the SHAddToRecentDocs API:<BR>
<BR>
<BR>
Declare Sub SHAddToRecentDocs Lib "shell32.dll" _<BR>
(ByVal uFlags As Long, ByVal pv As String)              <BR>
<BR>
  <BR>
Now that the function has been declared, the following code can be used to add a file to the list.<BR>
<BR>
Dim NewFile as String<BR>
NewFile="c:\newfile.file"<BR>
Call SHAddToRecentDocs(2,NewFile)   <BR>
<BR>
             <BR>
You can also clear the list of files by using the following code.<BR>
<BR>
Call SHAddToRecentDocs(2,vbNullString)<BR>
