<h3><font color="#000000">Review</font></h3><p>
Active Server Pages 3.0 is the newly released version of ASP. If you wish to create dynamic web content, then this should be the book of choice. This is not a beginner ASP book. Beginners to ASP should be looking at the <a href="http://www.amazon.com/exec/obidos/ASIN/1861001347/visualbasicbookr">Beginning Active Server Pages 2.0</a> book. <br>
Professional Active Server Pages 3.0 is designed to satisfy the needs of those developers who already have considerable ASP and server side web application development skills, along with a working knowledge of VBScript or Javascript.<br>
Section one of the book covers ASP basics. It covers the changes to ASP 3.0, the ASP Object Model, and all basic concepts required for using ASP script. Section two covers ActiveX Data Objects (ADO) and data access issues in general. ADO is now the standard Microsoft data access technology and is an integral part of ASP. Section three teaches how to build components for use with ASP using the Component Object Model (COM). Section four shows how you can integrate ASP with backoffice applications such as Microsoft Message Queue Server, Collaborative Data Objects and Microsoft Exchange, Active Directory, etc. Section five covers enterprise ASP issues. Security, performance, and scalability issues are covered. Finally, section six provides the necessary reference material.<br>
This book is a must have for anyone wanting to create dynamic web content using ASP.
