<P>Drive.dll is a dll that creates a drive to system directoryor drive similar to existing drives such as C:\. It contains seven classes:</P>

<PRE>class CContextMenu : public IContextMenu, IShellExtInit
class CDropTarget : public IDropTarget
class CEnumIDList  : public IEnumIDList
class CPidlMgr
class CQDriveClassFactory : public IClassFactory
class CShellFolder : public IShellFolder, public IPersistFolder
class CShellView : public IShellView</PRE>
