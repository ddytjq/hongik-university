<p>
When it’s your job to build applications that solve business problems, you need a reliable set of tools to help you develop code quickly and easily. You want these tools to maximize your efficiency, yet still maintain direct access and flexibility. And, if you want to deploy applications on the Internet, or are already doing so, the products you use must have integrated support for distributed computing.
</p>
<p>
If you’re building component-based code, Visual C++ provides native Component Object Model (COM) support, visual COM editing, and language-independent COM browsing. MFC templates reduce the time it takes to create lightweight, fast COM objects. And, you can use these software components in other COM-enabled applications, including those created with Visual C++ and the Microsoft Visual Basic&reg;programming system.
</p>
<p>
The latest C++ compiler technology in Visual C++ is finely tuned for performance, producing significantly faster and smaller 32-bit applications for the Windows&reg;95 and Windows NT&reg;Workstation operating systems. With new industry-leading optimizations and the latest ANSI features, your Visual C++–based applications run better than ever.
</p>
<p>
Up against a deadline? Take advantage of Visual C++ wizards and controls to develop dynamic Internet and intranet applications—and save time on virtually any project. For additional productivity, the integrated Microsoft Visual Basic, Scripting Edition (VBScript) language helps automate a variety of tedious coding tasks.
</p>
<p>
The newest version of the Microsoft Visual C++ develop-ment system is designed to improve your productivity not only with C++ applications, but also with Internet, intranet, and mixed-language systems. You can build applications with Visual C++ as distributed components and develop code that works efficiently across both your corporate network and the Internet
</p>
<p>
Visual C++ is part of Microsoft Visual StudioTM97, the first comprehensive suite of award-winning tools in an integrated development environment (IDE). Use Visual C++ with the Microsoft Visual J++TMdevelopment system and the Microsoft Visual InterDevTMWeb application development system (each sold separately) to manage and debug your Windows-based, client/server, and Internet applications. Now, you can work smarter using familiar tools to handle today’s complex projects.
</p>

<h2>Product Highlights</h2>
<h3>Visual C++&#151;get all the power you need to build advanced Internet and intranet applications.</h3>

<p>Get the performance you’ve been waiting for.</p>
<ul>
<li>Deliver fast, small, robust 32-bit applications for the Windows 95 and <br>
Windows NT operating systems.</li>
<li>The new compiler has built-in extensions for COM client support, making developing component-based software easier than ever.</li>
<li>New efficiency optimizations mean unparalleled speed for your applications (Professional and Enterprise editions only).</li>
<li>Create powerful database applications&#190;
Visual C++ provides fast, integrated database access, plus support for the built-in Microsoft Jet engine with Data Access Objects (DAO).</li>
<li>Develop client/server applications when working with multiple databases through support for the latest version of Open Database Connectivity (ODBC).</li>
<li>Quickly create Active Document servers, containers, and Automation objects, and build and use ActiveXTMControls through MFC and wizards.</li></ul>

<h3>Boost productivity to a new level.</h3>

<ul>
<li>Develop with reusable components from the Component Gallery, a central location for prebuilt objects including custom C++ classes, ActiveX Controls, and your own components.</li>
<li>Automate and configure the development environment to suit your needs with the customizable editor and the VBScript automation language and add-ins.</li>
<li>Build on the more than 150 professionally written classes in the Microsoft Foundation Classes and Templates (MFC&amp;T) Library.</li>
<li>Automatically create applications that meet your company’s or industry’s specific requirements by creating custom AppWizards using the included AppWizard wizard.</li>
<li>Use the Microsoft Developer Studio IDE to quickly access a wide range of Microsoft development tools while you are working.</li>
<li>Simplify object-oriented programming with Class View by managing projects as a set of related classes.</li>
<li>Speed up the edit-build-debug cycle on large projects with the incremental compiler, incremental linker, and minimal rebuild.</li>
<li>Professional and Enterprise editions include the award-winning Microsoft Developer Network (MSDN) Library compact discs with sample code, Microsoft Knowledge Base, bug lists, latest technology specifications, product documentation, books and periodicals, and conference papers.</li></ul>

<h3>Extend your development potential.</h3>

<ul>
<li>The highly adaptable Developer Studio IDE allows you to improve team development through integration with leading version control systems, including Microsoft Visual SourceSafeTMproject-oriented version control.</li>
<li>Developer Studio also integrates with the MSDN Library, Visual J++, Visual InterDev, and Visual SourceSafe.</li>
<li>Localize your applications through support for Unicode&reg;and Double-Byte Character Sets (DBCS).</li>
</ul>

<h2>Microsoft Developer Studio</h2>
<h3>General</h3>

<ul>
<li>A common development environment shared across Visual C++, Visual InterDev, Visual J++, the MSDN Library, and Visual SourceSafe.</li>
<li>Project Workspace, which integrates classes, COM interfaces, files, documentation, resources, and databases into one work area.</li>
<li>A &quot;scriptable&quot; (automated) development environment using VBScript.</li>
<li>Wizards supporting ActiveX Control container projects, so you can insert an existing ActiveX Control into your application.</li>
<li>The new InstallSHIELD&reg;5 Free Edition makes it easy to support install and uninstall (to comply with Windows 95 logo requirements).</li></ul>

<h3>Project Workspace</h3>

<ul>
<li>Class View. Now allows you to view COM interfaces, properties, methods, and the C++ classes that wrap them.</li>
<li>File View. See all your project files in an intuitive, familiar way. Now group your files into folders according to your preferences.</li>
<li>Info View. Switch between Books Online, the World Wide Web, and MSDN, using the new HTML Documentation viewer.</li>
<li>Data View. Review tables, schema, queries, and stored procedures in your project—now for any ODBC-compliant database.</li></ul>

<h3>Component Gallery</h3>

<ul>
<li>Choose from a gallery of prebuilt, fully shareable C++ and ActiveX Controls.</li>
<li>Expand the Component Gallery with components from third-party software developers.</li>
<li>Add ActiveX Controls directly to the Controls palette in the dialog editor.</li>
<li>Insert components from the Component Gallery into your development projects.</li></ul>

<h3>Wizards</h3>

<ul>
<li>ATL COM AppWizard. Easily develop location- and language-neutral code for Windows and the Internet, and get more versatility with less maintenance.</li>
<li>Custom AppWizards. Create a wizard based on an existing project, start from an existing AppWizard, or design your own from scratch.</li>
<li>ActiveX Control Wizard. Build ActiveX Controls quickly and effortlessly.</li>
<li>WizardBar. Rapidly navigate and add message handler and member functions.</li>
<li>DevStudio Add-In Wizard. Create an add-in to enhance the functionality of the development environment.</li></ul>

<h3>Easier resource editors</h3>

<ul>
<li>Reusable resource templates you can create.</li>
<li>Toolbar editor with drag-and-drop support.</li>
<li>ActiveX Controls can be used in any dialog resource.</li>
<li>Support for GIF and JPEG image formats.</li>
<li>Visual Basic 5.0 forms can be imported into the dialog editor.</li></ul>

<h3>A more powerful source-code editor</h3>

<ul>
<li>Interfaces exposed to Automation, so you can script your development.</li>
<li>HTML syntax coloring so you can edit your HTML.</li>
<li>Many new editing and navigation commands.</li>
</ul>

<h3>Debugging information at your fingertips</h3>

<ul>
<li>DataTipsTMpop-up information displays the value of the variable or expression under the mouse pointer.</li>
<li>New Variables window featuring AutoTab, which displays the value of all variables on the last line executed and on the current line.</li>
<li>New Watch window lets you group watch expressions on tabbed panes.</li>
<li>The QuickWatch dialog box allows you to view and change data and evaluate expressions, including invoking functions.</li>
<li>AutoExpand, which displays variables to show the most important data at the top level.</li>
<li>Remote Microsoft SQL ServerTMdatabase debugging using TCP/IP to remotely debug your SQL Server database (Enterprise Edition only).</li>
<li>Attach to process so you can start debugging a process that is <br>
already executing.</li>
</ul>

<h3>Source-code control integration</h3>

<ul>
<li>Supported by Microsoft Visual SourceSafe, Intersolv PVCS, Atria ClearCase, MKS Source Integrity, and other systems.</li></ul>

<h3>An intelligent build system</h3>

<ul>
<li>Benefit from enhanced support for subprojects. Subprojects are now separable from parts of the main project, allowing different team members to work on different parts of the project, then integrate for builds.</li>
<li>Project files now use a more compact format for faster loading and saving. They can be exported as make files on command.</li>
<li>Define multiple configurations within a single project. Load external make files and take advantage of the new project views.</li>
<li>Specify pre-link and post-build steps in addition to custom build steps for further customization of your project builds.</li></ul>

<h3>A customizable environment</h3>

<ul>
<li>Add or remove buttons, including those for macros, from any of the Developer Studio customizable toolbars, or create your own unique toolbars.</li>
<li>Assign your own shortcut keys to a Developer Studio menu command or macro.</li>
<li>Customize the menu by adding, deleting, or modifying commands or macros.</li></ul>

<h3>Info Viewer technology and sample applications</h3>

<ul>
<li>Books Online is in HTML format and integrated into the development environment.</li>
<li>Print entire sections or books in one step.</li>
<li>Includes MSDN Library in Professional and Enterprise editions—load either Books Online or MSDN Development Library into the Project Workspace.</li>
<li>Drag text from Info Viewer topics to the editor.</li>
<li>Use sample applications—view or copy files, or run sample programs, all from Books Online.</li>
<li>Drag code from samples to your project files.</li>
<li>Click the shaded text in a Books Online topic to see pop-up glossary definitions.</li>
<li>Browse the World Wide Web.</li>
</ul>

<h3>New features in Microsoft Foundation Classes and Templates (MFC&amp;T)</h3>
<p>Internet-aware MFC classes</p>

<ul>
<li>Server-side classes support Microsoft Internet Information Server API (ISAPI) for creating high-performance ISAPI extensions and filters. These classes include: CHttpServer, CHttpFilter, CHttpServerContext, CHttpFilterContext, and CHtmlStream.</li>
<li>WinInet classes support direct access to ftp, http, and gopher protocols. These include CInternetSession, CInternetConnection, CFtpConnection, CGopherConnection, CHttpConnection, CInternetFile, CGopherFile, CHttpFile, CFileFind, CFtpFileFind, CGopherFileFind, CGopherLocator, and CInternetException.</li>
<li>ActiveX support classes. These classes support asynchronous download of properties or other data over the Internet. They include CAsyncMonikerFile and CDataPathProperty.</li></ul>

<h3>ODBC performance improvements</h3>

<ul>
<li>ODBC transactions for protecting data integrity in the event of failure.</li>
<li>Bulk row fetching for improvement of large-query performance.</li>
<li>Support for Microsoft DTC (Distributed Transaction Coordinator) transactions.</li></ul>

<h3>Active Document server support</h3>

<ul>
<li>Wizard and programmatic support for creating Active Document servers that can be invoked in any Active Document container (such as Microsoft Office 95 or Office 97, Microsoft Internet Explorer 3.0, and Microsoft Visual Studio 97).</li></ul>

<h3>Active Template Library (ATL) 2.1</h3>

<ul>
<li>Complete library for creating COM objects—both visual (controls) and &quot;UI-less.&quot;</li>
<li>ATL COM Wizard for creating minimal components, including self-registration support.</li>
<li>ATL Object Wizard for adding objects to an existing ATL project.</li>
</ul>

<h2>Project productivity</h2>
