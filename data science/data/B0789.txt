      <p align="center"><font face="Arial, Helvetica, sans-serif"><b>Instructions 
        for owners of PC computers (running Windows 95/98/ME/NT/2000)</b></font></p>
      <p><font face="Arial, Helvetica, sans-serif" size="2">The interactive mortgage 
        application form is available as a self extracting file which, when run 
        will unpack the program onto your computer. The process is fairly straight 
        forward and the following steps will guide you through all you need to 
        know.</font></p>
      <p><b><font face="Arial, Helvetica, sans-serif" size="2">Note: </font></b><font face="Arial, Helvetica, sans-serif" size="2">The 
        screen shots below are from the <b>Internet Explorer</b> browser. If using 
        Netscape Navigator 4 please skip the first screenshot.</font></p>
      <p><font size="3" face="Arial, Helvetica, sans-serif"><b><font size="2">Step 
        1: Download the installer</font></b></font></p>
      <p><font face="Arial, Helvetica, sans-serif" size="2">The link below allows 
        you to download an automatic installer which will install the mortgage 
        application form on your system. </font></p>
      <p><font face="Arial, Helvetica, sans-serif" size="2">If you feel confident 
        in downloading an executable file we recommend you download the installer 
        to your <b>desktop</b>. When you have done this move on to <a href="#step2">Step 
        2</a>.</font></p>
      <p align="center"><font face="Arial, Helvetica, sans-serif"><b>OR</b></font></p>
      <p><font face="Arial, Helvetica, sans-serif" size="2">If you are unsure 
        how to download and run an executable file, please follow the instructions 
        below <b>before </b>clicking on the link.</font></p>
      <p><font face="Arial, Helvetica, sans-serif" size="2"><a name="download"></a></font></p>
      <div align="center">
        <table border="0" cellspacing="0" cellpadding="3">
          <tr> 
            <td align="center"><a href="Claycross Mortgage Application.exe"><img src="images/siticon.gif" width="31" height="31" border="0"></a></td>
            <td><font face="Arial, Helvetica, sans-serif" size="2"><a href="Claycross Mortgage Application.exe">Claycross 
              Mortgage Application.exe</a> (415k)</font></td>

      </div>
      <font face="Arial, Helvetica, sans-serif" size="2">When you click on the 
      link above you will see the following window.</font> 
      <p align="center"><img src="images/saveas_small.gif" width="350" height="234"><br>
        <font face="Arial, Helvetica, sans-serif" size="1"><i>This will not appear 
        if using Netscape Navigator 4</i></font></p>
      <p><font face="Arial, Helvetica, sans-serif" size="2">Select the &quot;Save 
        this program to disk&quot; option and click &quot;OK&quot;. This will 
        lead to a box similar to the following:</font></p>
      <p align="center"><img src="images/saveas_dialogue_small.gif" width="350" height="197"></p>
      <p align="left"><font face="Arial, Helvetica, sans-serif" size="2">We recommend 
        you select to save the installer on your desktop so that it is easy to 
        locate and run. Once you have specified where you want the installer to 
        download to, click on &quot;Save&quot;.</font></p>
      <p align="left"><font face="Arial, Helvetica, sans-serif" size="2">Now wait 
        for the download to complete.</font></p>
      <p align="center"><font face="Arial, Helvetica, sans-serif" size="2"><b><i>Having 
        done this you are free to disconnect from the Internet.</i></b></font></p>
      <p><font size="3" face="Arial, Helvetica, sans-serif"><b><font size="2"><a name="step2"></a>Step 
        2: Run the installer to unpack the files</font></b></font></p>
      <p><font face="Arial, Helvetica, sans-serif" size="2">Assuming that you 
        have downloaded the installer to your desktop, it is now simply a matter 
        of looking on your desktop and double clicking on the icon named &quot;<b>Claycross 
        Mortgage Application</b>&quot; (it may or may not have .exe at the end 
        - depending on how your system is configured). Look on your desktop for 
        an icon which looks like this (the icon may not be in this position on 
        your desktop):</font></p>
      <p align="center"><img src="images/icon.gif" width="113" height="149"></p>
      <p><font face="Arial, Helvetica, sans-serif" size="2">Double clicking the 
        file will cause the following window to open up:</font></p>
      <p align="center"><img src="images/unpack_small.gif" width="350" height="142"></p>
      <p align="left"><font face="Arial, Helvetica, sans-serif" size="2">Now click 
        on &quot;<b>Browse</b>&quot; to select where you want the mortgage application 
        to install. We again recommend the desktop as this will make it very easy 
        to access the form when you need to.</font></p>
      <p align="left"><font face="Arial, Helvetica, sans-serif" size="2">After 
        clicking on &quot;<b>Browse</b>&quot; a file selector window will appear. 
        Locate your desktop at the top of this window and click on &quot;<b>OK</b>&quot;. 
        You will now be returned to the window displayed above. Click on &quot;<b>Extract</b>&quot; 
        to install the form.</font></p>
      <p align="left"><font face="Arial, Helvetica, sans-serif" size="2">When 
        complete a message will appear stating that all the files have been sucessfully 
        installed.</font></p>
      <p align="left"><font size="3" face="Arial, Helvetica, sans-serif"><b><font size="2"><a name="step3"></a>Step 
        3: Load the application form program</font></b></font></p>
      <p align="left"><font face="Arial, Helvetica, sans-serif" size="2">With 
        the desktop visible you should now see a new folder icon appear called 
        &quot;Claycross Mortgage Application&quot;.</font></p>
      <p align="center"><img src="images/folder_small.gif" width="136" height="136" border="1"></p>
      <p align="left"><font face="Arial, Helvetica, sans-serif" size="2">Double 
        click to open the folder. Inside you will see a file called &quot;<b>Begin 
        Application</b>&quot; (this may have a .htm extension depending on how 
        your system has been configured).</font></p>
      <p align="left"><font face="Arial, Helvetica, sans-serif" size="2">Double 
        click this icon to begin filling out your application form. Your web browser 
        should open up and automatically load the application form start screen. 
        Follow the onscreen instructions from here.</font></p>
      <p align="left"><font face="Arial, Helvetica, sans-serif" size="2">If you 
        havent already downloaded the application form installer,</font></p>
      <p align="center"><font face="Arial, Helvetica, sans-serif" size="2"><a href="#download">Return 
        to the link above to start the download process</a></font></p>
