<font size=+1><b>Software Pool:</b> <br> <br>
</center>
A bunch of software packages and tools is installed on our group server.
The tools listed below are located on the machine "xray". You can go to the cross-mounted partitions of
this server <font color="#009900">/net/xray/usr/local</font> and <font color="#009900">/net/xray/software</font> 
where the software is installed. In order to start the packages, you need to include some startup scripts
in your <font color="#009900">.cshrc</font> file located in your home directory.
Here is the recommended <a href="http://xray.astro.umass.edu/intern/.cshrc"><font color="#009900">.cshrc</font></a> 
you may want to use. 
<blockquote><font color="#009900"><font size=+1><b>CIAO:</b> `Chandra
Interactive Analysis of Observations', current version: 2.1.1, old version 2.0 still installed</font></font>
<p><font color="#009900"><font size=+1><b>HEAsoft:</b> a unified Release
of the FTOOLS and XANADU Software Packages, including XSpec</font></font>
<p><font color="#009900"><font size=+1><b>FV:</b> `The Interactive FITS
File Editor'</font></font>
<p><font color="#009900"><font size=+1><b>ds9: </b> `SAOImager' image display tool</font></font>
<p><font color="#009900"><font size=+1><b>SAS:</b> `XMM-Newton Science Analysis Software'</font></font>
<p><font color="#009900"><font size=+1><b>XEmacs:</b> `X-based Emacs Editor'</font></font>
<p><font color="#009900"><font size=+1><b>PIMMS:</b> `Portable, Interactive Multi-Mission Simulator'</font></font>
<br>&nbsp;</blockquote>

<font size=+1><b>File Exchange Areas:</b> <br> <br>
</center>
In order to exchange files between group members and people outside our group, we created the directory 
<blockquote>
<font color="#009900">/net/xray/pub</font> <br>
</blockquote>
People outside our group can use the web-based interface for this 
<a href="http://xray.astro.umass.edu/public/"><font color="#009900">file exchange folder</font></a>
<br>
Just send your colleagues an email with the link to the files: 
<a href="http://xray.astro.umass.edu/public/"><font color="#009900">http://xray.astro.umass.edu/public/</font></a>
<br><br>
Non-public data shared between group members only are located in 
<blockquote>
<font color="#009900">/net/xray/data1</font>
</blockquote>
