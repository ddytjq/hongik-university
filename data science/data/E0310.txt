<H1> Introduction to C Programming</H1>

<!-- ~~~~~~~~ Page content ~~~~~~~~ -->


<P>

<B>So you want to learn <B>C</B>? </B>We hope to provide you with an easy step by step guide to programming in <B>C</B>. The course is split up into several sections, or lessons, which include <B>C</B> example programs for you to demonstrate what has been taught. Although the ordering of the sections does not have to be strictly followed, the sections become progressively more involved and assume background knowledge attained from previous sections. Good Luck!

<HR>

You've now reached the end of this online tutorial. We have covered a lot of ground - but this has been a first course in <B>C</B> and there is still plenty to learn. However, as long as you keep in mind that <B>C</B> is an essentially simple language and how new features are built from this simplicity you shouldn't have many problems.<P>

You also need to be aware of the fact that <B>C</B> is a very low-level language and as a result allows programmers to confuse data types and muck around with the bit patterns of the data in a way that higher level languages would disown! You probably need to make sure that you understand binary and the way that values are represented to get the best from <B>C</B>.<P>

The final question is, where should you go next? If you
have enjoyed learning <B>C</B> then you probably need to move on to <B>C++</B> the language most likely to reign supreme in the future! Watch this space for online tutorial!
