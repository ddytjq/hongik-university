------------------------------------------------------------
               THIS WEEKS ARTICLE: Frames
------------------------------------------------------------

	A lot of Java applets pop up seperate "applet windows".
There is often the message "Warning: Java Applet Window" or 
something like that at the bottom of an applet window. These
"applet windows" are actually Frames. This tutorial will teach
you the basics of frames so that you can use them in your
applets. First, import the gui components (java.awt.*)
Then, put the following line after the beginning of your 
applet (the first bracket), but before the first  method 
(usually before init() ).

Frame name;

	Replace "name" in the line above with what you want
the name of your Frame to be. Then put the following line in
init() :

name=new Frame("   ");

	Replace "name" with the name you used in the line discussed
above. Also put the title of your Frame (the text in the bar
at the top of the Frame) in between the quotes in the line 
above. Now you have a frame, but it is invisible, and it
has no set size. So in init(), after the name=new Frame line,
put:

name.setSize(x,y);
name.setVisible(true);

	Again, replace name with the name of your Frame. In the
first line, replace x with how wide you want your Frame to be
in pixels, and y with how tall you want your frame to be in
pixels. If you compiled and ran your applet now, the frame
would come up. However, it wouldn't close.....which is a
problem. The first step is to implement WindowListener. To do
this, put this after "extends Applet" :

implements WindowListener

	Now add the following lines (they are methods that go
with WindowListener) to your applet:


    public void windowDeactivated(WindowEvent e) {
    }
    public void windowClosed(WindowEvent e) {
    }
    public void windowDeiconified(WindowEvent e) {
    }
    public void windowOpened(WindowEvent e) {
    }
    public void windowIconified(WindowEvent e) {
    }    
    public void windowActivated(WindowEvent e) {
    }
    public void windowClosing(WindowEvent e) {
        name.setVisible(false);
    }

	The code in the windowClosing method is what closes
the window (by making it invisible). Replace "name" with
the name of your Frame. Then, in init, after everything
else in init() mentioned before, put:

name.addWindowListener(this);

	That's it! Now your Frame is fully functional, and
you can add things to it! Just do name.add(whateveryouwanttoadd);
It's great! Enjoy!
