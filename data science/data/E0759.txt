<H3>Keywords</H3>

<B>C</B> is a relatively small language which uses only 32 keywords. Many of these will be covered in the course but for reference purposes the list of keywords is as follow:<P>
<PRE>
<B><A HREF="ccccauto.html">auto</A>		double		int		struct
break		else		long		switch
case		<A HREF="ccccenum.html">enum</A>		register	typedef
char		<A HREF="ccccextn.html">extern</A>		return		union
const		float		short		unsigned
continue	for		signed		void
default		<A HREF="ccccgoto.html">goto</A>		sizeof		volatile
do		if		static		while</B>
</PRE>

<P>Many of the keywords listed above will be covered on this course. However,
some have either limited use or are only used in advanced programming. Such keywords
are highlighted above where a brief desciption of their use is offered.
