<td bgcolor="#d2cec4">
<SPAN Class="smbl">
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Loops, along with conditionals, and everything else talked about in the Introduction section, are essential. Basically what a loop does is repeat a statement or block of code many times. If I wanted a command to happen while a certain variable is less than ten, I would use a loop.<br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;One type of loop is called a while loop.. This is
how it works. While a certain condition is true, it executes a certain block of
code. Here is the syntax:<br><br>
<i>while(condition){<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;statements<br>}</i><br><br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Ok, now here is an example that will show how it works. Just to let you know, System.out.println() prints the text in the parentheses in the Java console. We will not be using it much, but it is useful in this example.<br><br>
int
x=0;<br>while(x<10){<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;System.out.println(x);<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;x=x+1;<br>}<br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Allright. So what does this code do? I will
explain it line by line. First of, a integer called x is declared, and its
value is set to zero. This is done with int x=0; The next line, while(x<10){
,says that while x is less than ten, it will execute the code inside the
braces. The next line, System.out.println(x); , simply prints out the value of
x. The next line, x=x+1; , adds one to the value of x, so the next time around,
the loop prints the next number. When x=10, the loop stops running. So this is
the complete output (in the Java
Console):<br><br>0<br>1<br>2<br>3<br>4<br>5<br>6<br>7<br>8<br>9<br><br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;The other important loop you will need to know is
the for loop. For loops are more complicated, but are more useful than while
loops in certain conditions. I will explain it after I show you the syntax.
Here is the syntax:<br><br><i>for(starting
value;condition;increment){<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;statements;<br>}</i><br><br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;I know, this looking insanely hard in comparison to the while loops. The starting value is just the initial value of the variable that is part of the conditional. In this case, we will use an integer called x. While the condition is true, the block of code executes. The increment tells what we are supposed to do to the variable after the block of code executes. It can be anything, but usually it adds one. Here is an example:<br><br>
for(x=0;x<10;x=x+1){<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;System.out.println(x);<br>}<br><br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;The
output for this is the same as the output in our example while loop above. They
do the same thing, they check to see if x is less than ten, and if it is, the
value of x is printed in the Java console. Then one is added to x, and it goes
back to the condition.<br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;You may not see the usefullness of loops right
now, but take my word for it, you will use them a lot. One thing that I did not
mention is the purpose of the increment, x=x+1. The point of it is so there can
be a time when the condition is false. If we didn't have an increment in there,
the value of the variable would stay the same, and the condition would always
be true. This means that the block of code would continue to execute forever,
resulting in a frozen screen on your computer. These are called endless loops.
They are very annoying, so make sure you always make it possible for the
condition to be false so the loop can stop.
