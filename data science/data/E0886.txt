<h2 class=pagetop>Visual C++: the Fast Track for C++ Development</h2>
<p>Make the most of your development efforts with Microsoft Visual C++ development version 4.0. Create and code reuse and components, all with the latest Windows&#174;-based 32-bit technology.  Do this in a highly productive development environment designed to boost performance: yours.</p>

<h3>Overview</h3>
<p>
Just because you're starting a new application, doesn't always mean starting from scratch. We're talking about  reusing code and components from other applications to build new solutions.<p>The best way to do this is with Microsoft Visual C++ development system version 4.0, the tool designed to help you accelerate application development for the Microsoft Windows 95&#153; and Windows NT&#153; Workstation operating systems.
</p>
<p>
New to Visual C++ is the Component Gallery. It's a one-stop location for you to store and manage your reusable components, including C++ classes, OLE controls, and dozens of third-party components that enable you to quickly add advanced features.
</p>
<p>
Microsoft Developer Studio provides an interactive development environment to simplify object-oriented programming with new class navigation tools--such as ClassView and WizardBar--and seamless integration with other Microsoft tools.
</p>
<p>
And you can now create custom AppWizards for a fast start to building your applications.
</p>
<p>
Another way to accelerate development is with the popular Microsoft Foundation Class (MFC) library version 4.0. MFC contains more than 150 classes and 120,000 lines of code you don't have to write or test, and it includes support for creating applications that take advantage of the interface for Windows 95. 
</p>
<p>
For database applications, we've included Data Access Objects (DAO) classes for fast object-oriented access to the Jet database engine. And ODBC support for creating database-independent, client/server applications.
</p>
<p>
You can also bring your applications to a range of new users with the cross-platform support of Visual C++.
</p>
<p>
With the Microsoft Visual C++ Subscription, you get the latest version of Visual C++, plus three more releases over a year containing the latest tools and technical information. Automatically delivered to your desktop. 
</p>
<p>
It's everything you need to create applications that can display the &quot;Designed for Windows 95&quot; and &quot;Windows NT Compatible&quot; logos. No more wondering if you have the newest Visual C++ technology.
</p>
<p>
With this kind of convenience, it's no surprise that more and more developers who are looking for real performance gains are subscribing every day.
</p>

<h3>Microsoft Visual C++ 4.0</h3>
<ul>
<li>32-bit application development for Windows 95, Windows NT Workstation, and Windows 3.1 using Win32s&#174;.
<li>Latest Microsoft Foundation Class (MFC) library version 4.0, including support for new Windows 95-based user interface controls.
<li>Component Gallery that includes C++ classes and  components, your own code, and OLE controls.
<li>OLE controls creation and use in the dialog editor.
<li>Database support through Data Access Objects (DAO); includes Jet database engine.
<li>Built-in ODBC connectivity to popular databases, including Microsoft SQL Server&#153;, Microsoft Access, and ORACLE.
<li>Custom AppWizards: Create your own AppWizards to customize your applications.
<li>Developer Studio, integrated with Microsoft Developer Network (MSDN) Developer Library, Microsoft Visual SourceSafe&#174;, Microsoft Fortran PowerStation, and Microsoft Visual Test.
<li>ClassView feature to navigate through C++ classes as a set of C++ classes.
<li>WizardBar for navigating and adding message handlers and other member functions.
<li>Namespaces, Run-Time Type Information (RTTI), C++ templates, exception handling, and Standard Template Library (STL) support.
<li>Incremental compiler and linker to speed build time.
<li>Minimal rebuild feature that only compiles the files that need it.
<li>Support for complex projects including sub-projects (DLLs, EXEs, OCXs, and more).
<li>Just In Time debugging with memory window and DataTips&#153; pop-up information, variables window, and more.
<li>Full-featured editor: compatible with BRIEF&#153; and Epsilon&#153;, incremental search, full-screen editing, and fully mappable keyboard.<li>Resource editor includes toolbar editor and support for conditional resources.<li>InfoView provides more than 10,000 pages of Books Online, with integration to the MSDN Development Library.
<li>Support for porting code to multiple platforms (through optional products), including Intel, PowerPC&#153;, MIPS&#174;,  Alpha , Apple&#174; Macintosh&#174;, and Power Macintosh&#153;.
<li>UNIX&#174; platform portability through third-party products such as Bristol and MainSoft.<li>Support for Unicode and Double Byte Character Set (DBCS) for globalizing applications.
<li>Includes Visual C++ 1.52 for developing 16-bit applications for MS-DOS&#174; and Windows 3.1
</ul>

<h3>Microsoft Visual C++ Subscription</h3>

<ul>
<li>One full year of  updated development system and tools delivered automatically in three timely releases.
<li>Includes enhancements to the MFC library, new features, C++ components,  code samples, and technical notes.
<li>New technical notes on Visual C++  and MFC.
<li>Includes major releases as they become available during subscription period.
<li>The Subscription Renewal edition offers an easy way for current users of Visual C++ Subscription to continue receiving new technology throughout the year.
</ul>

<h3>Microsoft Visual C++ Cross-Development Edition<br>for Macintosh</h3>
<ul>
<li>Add-on tool for Microsoft Visual C++ version 4.0 for creating Macintosh-and Power Macintosh-based applications from source code written to Microsoft Foundation Classes and the Microsoft Win32 API.
<li>Reuse up to 90% of your Intel-based code with a simple recompile.
<li>Includes all of the  tools needed to build and debug Macintosh executables.
<li>Windows Portability Library for porting Windows-based features to the Macintosh, including OLE and ODBC.
<li>True native Macintosh programming in applications.
<li>Target applications have familiar Macintosh interface.
<li>Fast executables using the Visual C++ optimizing C/C++ compiler.
</ul>

<h3>Microsoft Visual C++, RISC Edition</h3>
<ul>
<li>Separate edition available to support Windows NT Workstation version 3.51 on PowerPC, Digital Alpha AXP, and MIPS processors.
<li>Identical toolset as for Intel processors.
<li>Full source code compatibility across processors (does not include DAO and Jet database engine on RISC platforms).
</ul>

<h3>The new word for productivity: reusability.</h3>
<ul>
<li>Deliver fast, small, robust 32-bit applications for the Windows 95 and Windows NT operating systems.
<li><i>Component Gallery</i> helps improve productivity through reusability, serving as a central location for prebuilt objects including custom C++ classes, OLE Controls, and your own components.
<li>Build, use, and share <i>OLE Controls</i> across several Microsoft products, such as the Microsoft Visual Basic&#174; programming system and Microsoft Access database management system.
<li><i>Latest Microsoft Foundation Class library</i> (MFC) offers over 150 professionally written classes for you to build on.
<li>The <i>Custom AppWizard</i> allows you to automatically create applications that meet company- or industry-specific requirements.
<li><i>Microsoft Developer Studio</i> provides seamless, quick access to a range of complementary Microsoft tools.
<li>Simplify object-oriented programming with <i>ClassView</i> by managing projects as a set of related classes.
<li>Speed up the edit-build-debug cycle on large projects with the <i>incremental compiler, incremental linker, </i>and <i> minimal rebuild.</i>
</ul>

<h3>The extra performance you and your users are looking for.
</h3>
<ul>
<li><i>Fast, integrated database access and support</i> with DAO and the Jet engine for creating powerful database applications and components.
<li>Develop client/server applications when working with multiple databases with support for the latest version of Open Database Connectivity (ODBC).
<li>Quickly create OLE compound document server containers and OLE Automation objects, and build and use OLE Controls through MFC and wizards.
<li>Take advantage of full C++ language support with Namespaces, RTTI, C++ templates, exception handling, and full support for Standard Template Library (STL).
</ul>

<h3>Expand your development horizons.</h3>
<ul>
<li>The highly adaptable Developer Studio enables you to <i>improve team development</i> through integration with leading version control systems, including Microsoft Visual SourceSafe&#174; project-oriented version control.
<li>Developer Studio also integrates with Microsoft Developer Network, Microsoft Visual Test, and Microsoft Fortran PowerStation.
<li>Globalize your applications through support for Unicode and Double Byte Character Sets (DBCS).
<li>With the Cross-Development Edition for the Apple Macintosh, you can <i>port your code with minimal effort</i> to Macintosh platforms including the Power Macintosh.
<li>Recompile applications using Microsoft Visual C++, RISC Edition, to run natively on Alpha, MIPS, or PowerPC systems.
<li>Use third-party tools to port code to UNIX platforms.
</ul>

<h3>Microsoft Visual C++ Subscription: Welcome to the latest technology.</h3>
<ul>
<li>Receive three additional releases of Visual C++, including major releases as they become available during your subscription year.
<li>Latest tools and information include new components, OLE controls, sample code, recent Visual C++ and MFC technical notes, and updated online documentation.
</ul>
