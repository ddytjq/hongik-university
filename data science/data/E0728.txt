<p>Windows 2000 is designed to provide best of breed networking, application, communication, and Web services with increased reliability, scalability, and manageability.  Together these capabilities significantly reduce IT costs and enable a new generation of enterprise web solutions that help organizations realize real competitive benefit.  The Visual Studio 6.0 family of tools enables developers to harness the capabilities of Windows 2000 to deliver compelling Web solutions to customers.</p>

<h3>Windows 2000 Server</h3>

<p>The Microsoft Windows&#174; 2000 Server family of products is the next generation of the Windows NT® Server series of operating systems.  Microsoft is delivering several editions of Windows 2000 Server. These include Windows 2000 Server for workgroup and department file/print servers, Windows 2000 Advanced Server for application hosting, and Windows 2000 Datacenter Server for organizations’ most mission-critical server systems.</p>

<p>Windows 2000 Server offers higher levels of overall system uptime and performance compared to Windows NT Server 4.0. In addition to enhancements to the core operating system that makes it intrinsically more reliable and scalable, the Advanced Server & Datacenter Server versions of Windows 2000 provide clustering and load balancing system services offering even higher levels of application and data availability as well as the ability to support larger user populations on either single, symmetric multiprocessing (SMP) servers or distributed, clustered servers.  As a result, the Windows 2000 Server family offers better price/performance and costs significantly less to deploy and support than comparably priced Unix mini and mainframe based systems.</p>  

<p>Windows 2000 Server also provides a best of breed platform for building and deploying large-scale enterprise web applications.  Everything needed for constructing enterprise web applications is included, such as Internet Information Server (IIS), Microsoft Transaction Server (MTS), Microsoft Message Queue (MSMQ), Active Directory Services (ADSI), Kerberos security, and much more.  Because these services are integrated into the core operating system they offer a common programming model and extremely fast performance.  Windows 2000 also offers the best support for web standards, including DHTML, HTTP compression, enhanced TCP/IP, LDAP and a world class XML implementation.  With these features, Windows 2000 has the capabilities required to build and host any Enterprise web application.</p>
