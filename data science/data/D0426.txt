<p><font face="Arial" size="2">This applet is a simple text animation which
&quot;grows&quot; a user specified string from a minimum font point size to a maximum
point size, both specified by the user as APPLET tag parameters. If the user clicks on the
applet container, it restarts the animation. </font></p>

<p><font face="Arial" size="2">This is what the mark-up looks like to implement the
growing text applet in your HTML page (Note: this assumes that <i>growing_text_andyhosk.class</i>
is in the same directory as it's parent HTML page). The parameters, by the way, correspond
to the instance that is running on this page right now:</font> </p>

<h2><font face="Arial" size="2">Background and Foreground Color Attributes </font></h2>

<p><font face="Arial" size="2">Most people will find the six basic colors sufficient.
However, you can set the background or foreground color of the applet to any one of the
16+ million colors in the 24 bit RGB (Red-Green-Blue) spectrum. But first, you must
calculate the value of your custom color. To facilitate this, please use <a href="../rgb/">this
calculator</a>. </font></p>
