<CENTER><B><FONT COLOR="#FF0000">
Displaying/ Changing Port Settings
<BR><HR SIZE=1 WIDTH=200>
</FONT></B></CENTER><P>
You can use the windows API to call a dialog box with information of any port in your computer. Also, you'll be able to alter this ports settings. To accomplish this task, use the <I>ConfigurePort</I> function as below (add the code to a form - when you click the 'Command1' button, the system will display information for COM4 and LPT1 ports):

<BR>
<BR>
<B>Private Declare Function</B> ConfigurePort _<BR>
Lib "winspool.drv" Alias "ConfigurePortA" _<BR>
(ByVal pName As String, ByVal hwnd As Long, _<BR>
ByVal pPortName As String) <B>As Long</B><BR>
<BR>
<B>Private Sub</B> Command1_Click()<BR>
<I>Call</I> ConfigurePort("", Me.hwnd, "COM4")<BR>
<I>Call</I> ConfigurePort("", Me.hwnd, "LPT1")<BR>
<B>End Sub</B><BR>
<BR>
<I>'Note that if the function fails, it returns a value of 0</I><BR>
