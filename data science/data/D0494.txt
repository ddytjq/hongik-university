<p><font face="Arial" size="2">This applet allows web authors to create a custom web site
navigation &quot;pick list&quot; on their HTML pages. This applet allows HTML authors to
specify an array of URLs, targets, and descriptive titles to populate a choice widget
which, when selected, will load the currently selected shortcut. This applet is written to
run on all major browser/operating system combinations. Here's a sample implementation,
including an example of what the HTML APPLET tag mark-up should look like to implement
this applet in a web page:</font></p>

<p align="center">
<applet align="absMiddle" code="choiceMenu_andyhosk.class" height="25"
name="Web Site Shortcuts" width="270" VIEWASTEXT>
  <param name="urls"
  value="/java,http://www.javasoft.com/,http://www.gamelan.com/,http://www.jars.com/,http://www.microsoft.com/java/,/default.asp">
  <param name="choices"
  value="Hoskinson's Java Resources,JavaSoft's Web Site,Gamelan -- The Official Java Directory,JARS -- The Java Applet Rating Service,Microsoft's Java Web Site,My Home Page">
  <param name="targets" value="_top">This tool requires Java
</applet>
&nbsp;</p>
