<h2>
Trail: JavaBeans(TM)
</h2>
<blockquote>
</blockquote>
<h3>
<em>by <a href="../information/bios.html#quinn">Andy Quinn</a></em>
</h3>

<p>
JavaBeans brings component technology to the Java
platform.

With the JavaBeans API you can
create reuseable, platform-independent components. Using
JavaBeans-compliant application builder tools,
you can combine these components into applets,
applications, or composite components.

JavaBean components are known as <i>Beans</i>.

<blockquote>
JavaBeans is a core JDK1.1 capability. Any JDK1.1-compliant
browser or tool implicitly supports JavaBeans. 
</blockquote>

This document is a hands-on guide to learning JavaBeans and
the Beans Development Kit (BDK).
The 
<a target="other" href="http://java.sun.com/beans/spec.html"><font color="#009bbb">JavaBeans API Specification</font></a><a href="http://java.sun.com/beans/spec.html"><img src="../images/otherIcon.gif" width=11 height=11 border=0 align="ABSMIDDLE" alt="(outside of the tutorial)"></a>  provides a complete JavaBeans description.Print out a copy of the JavaBeans specification, and keep it
handy when going through this document.


<p>
The BDK is  available free on the web.
In addition to the 
<a target="other" href="http://splash.javasoft.com/beans/bdk_download.html"><font color="#009bbb">BDK</font></a><a href="http://splash.javasoft.com/beans/bdk_download.html"><img src="../images/otherIcon.gif" width=11 height=11 border=0 align="ABSMIDDLE" alt="(outside of the tutorial)"></a> ,  you will need the 
<a target="other" href="http://java.sun.com/products/jdk/1.1/index.html"><font color="#009bbb">Java Development Kit</font></a><a href="http://java.sun.com/products/jdk/1.1/index.html"><img src="../images/otherIcon.gif" width=11 height=11 border=0 align="ABSMIDDLE" alt="(outside of the tutorial)"></a>  (JDK).
<p>
<!--    What is a JavaBean    -->
<a href=whatis/index.html>
<img src="../images/jdk1_1.gif" width=20 height=20 align=left
 border=0><b>JavaBeans Concepts and the Beans Development Kit</b></a>
describes what makes a Bean, and describes the
Beans Development Kit (BDK).

<p>
<!--    BeanBox Basics    -->
<a href=beanbox/index.html>
<img src="../images/jdk1_1.gif" width=20 height=20 align=left
 border=0><b>Using the BeanBox</b> </a> describes basic BeanBox
operation, and explains the BeanBox menus.

<p>
<!--    Writing a Simple Bean    -->
<a href=writingbean/index.html>
<img src="../images/jdk1_1.gif" width=20 height=20 align=left
 border=0><b>Writing a Simple Bean</b></a> walks you through creating
a rudimentary Bean, saving the Bean, adding the Bean to the ToolBox,
placing the Bean in the BeanBox, inspecting the Bean's properties and
events, and generating a Bean introspection report.

<p>
<!--    Properties    -->
<a href=properties/index.html>
<img src="../images/jdk1_1.gif" width=20 height=20 align=left
 border=0><b>Properties</b></a> explains how to give your Beans properties:
Bean appearance and behavior characteristics <i>customizable at design time</i>.

<p>
<!--   Events   -->
<a href=events/index.html>
<img src="../images/jdk1_1.gif" width=20 height=20 align=left
 border=0><b>Manipulating Events in the BeanBox</b></a> describes the
BeanBox's event manipulating capabilities. If you are not familiar 
with event handling, you might want to read up on
<a href= "../uiswing/overview/event.html">
Event Handling</a> to prepare for this material.

<p>
<!--    BeanInfo    -->
<a href=beaninfo/index.html>
<img src="../images/jdk1_1.gif" width=20 height=20 align=left
 border=0><b>The BeanInfo Interface</b></a> describes how to
write Bean information classes: Separate classes you can use to explicitly
advertise your Bean's properties, methods, and events to builder tools.

<p>
<!--    Customizing Beans  -->
<a href=customization/index.html>
<img src="../images/jdk1_1.gif" width=20 height=20 align=left
 border=0><b>Bean Customization</b></a> introduces you to property
editors, and the <code>Customizer</code> interface.

<p>
<!--    Persistence    -->
<a href=persistence/index.html>
<img src="../images/jdk1_1.gif" width=20 height=20 align=left
 border=0><b>Bean Persistence</b></a> explains how to save and retore
your Beans, and their customized state.

<p>
<!--    Beancontext    -->
<a href=beancontext/index.html>
<img src="../images/jdk1_1.gif" width=20 height=20 align=left
 border=0><b>Using the BeanContext API</b></a> 
is a <font color="red"><b>NEW</b></font>
lesson by
<a href="../information/bios.html#hommel">Scott Hommel</a>
that covers
The Extensible Runtime Containment and Services Protocol,
also known as beancontext.


<p>
<!--    Glasgow    -->
<a href=glasgow/index.html>
<img src="../images/jdk1_1.gif" width=20 height=20 align=left
 border=0><b>New Features</a></b> lists upcoming BDK and 
Beans-related features.

<h3>Additional Documentation</h3>
<p>
The BDK's <code>beans/docs</code> directory
contains documentation for
<ul>
  <li>The Beans API
  <li>The BeanBox API
  <li>The demo Beans
  <li>The <code>java.util</code> API
  <li>Java Archive (JAR) files and manifests
  <li>Makefiles for gnumake (Unix) and nmake (Windows)
</ul>
A good starting point is the file <code>beans/README.html</code>.

<p>
The 
<a target="other" href="http://java.sun.com/beans/docs/index.html"><font color="#009bbb">JavaBeans Documentation</font></a><a href="http://java.sun.com/beans/docs/index.html"><img src="../images/otherIcon.gif" width=11 height=11 border=0 align="ABSMIDDLE" alt="(outside of the tutorial)"></a>  page contains current JavaBeans API definitions,
upcoming JavaBeans feature descriptions,
and related Java documentation such as the
Java Core Reflection API, Object
Serialization, Remote Method
Invocation (RMI), and a 
third-party JavaBeans book list.
</ul>

</blockquote>
