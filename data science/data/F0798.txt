<p><font face="Arial, Helvetica" size="+1"><b>Tips</b></font></p>



<p><b><a href="http://www.vbsquare.com/api/tip462.html">
Invert the desktop</a></b><br>
Simple API calls to invert the desktop <font size="1" color="#666666">01/29/2000</font></p><p><b><a href="http://www.vbsquare.com/api/tip449.html">
How to hook VB callback functions into the API</a></b><br>
Visual Basic's AddressOf operator, introduced in version 5, lets
you pass a pointer for a user-defined sub, function, or property
to an API procedure. The API function can then pass various data
to the callback procedure, which the callback can manipulate as
it deems appropriate. <font size="1" color="#666666">01/8/2000</font></p><p><b><a href="http://www.vbsquare.com/api/tip359.html">
Copy and move using the API</a></b><br>
Use the CopyFile API to do lots of file operations. <font size="1" color="#666666">12/30/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip289.html">
Extract Icons</a></b><br>
Extract icons from files. <font size="1" color="#666666">12/30/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip307.html">
Get the exe name from its handle</a></b><br>
 <font size="1" color="#666666">12/30/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip420.html">
Is active desktop turned on?</a></b><br>
IE 4/5 provide the user with the active desktop. Your app can manipulate this, but only if it is turned on. <font size="1" color="#666666">12/30/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip343.html">
Is you program running?</a></b><br>
Using the FindWindow API is a much better way of determining whether your app is already running. Now you can be sure that your application is not running when you execute. This example assumes the window caption of your form is calculator, so change it as necessary. If you want, you can omit the classname completely. <font size="1" color="#666666">12/30/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip344.html">
Launch a program and wait for it to end</a></b><br>
 <font size="1" color="#666666">12/30/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip386.html">
Minimize & Restore all windows</a></b><br>
It may be useful to quickly minimize all the windows currently open, and then restore them. You can do this by using the PostMessage API to send a MIN_ALL and a MIN_ALL_UNDO message to all the windows. <font size="1" color="#666666">12/30/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip287.html">
Port Settings</a></b><br>
Port Settings Note: Type the name of the port into text1 e.g. LPT1 <font size="1" color="#666666">12/30/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip413.html">
Reposition the desktop icons</a></b><br>
Resort the desktop icons programatically. <font size="1" color="#666666">12/30/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip340.html">
Retrieve the current printer name</a></b><br>
This tip uses the registry to get the current printer name. <font size="1" color="#666666">12/30/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip366.html">
Return the handle of the desktop</a></b><br>
Get the hWnd of the desktop window. <font size="1" color="#666666">12/30/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip367.html">
Returns the classname from the window handle</a></b><br>
If you have an hWnd, then use this tip to get the classname. <font size="1" color="#666666">12/30/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip368.html">
Returns the handle of the active window</a></b><br>
Get the hWnd of the active window. <font size="1" color="#666666">12/30/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip369.html">
Returns the handle of the window that has the mouse captured</a></b><br>
Get the hWnd of the window where the mouse is. <font size="1" color="#666666">12/30/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip385.html">
Show the shutdown dialog</a></b><br>
In other tips, we have shown you how to forcefully shutdown, reboot and logoff the computer. In this tip, we show you how to display the shutdown dialog, using a function in the shell32 dynamic link library. <font size="1" color="#666666">12/30/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip355.html">
Shows the Win95 About Box</a></b><br>
Display the standard Win95 about box. <font size="1" color="#666666">12/30/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip400.html">
Subclassing: Changing the default popup menu</a></b><br>
Intercept the default popup menu on a form and add your own item. Warning: Do not break otherwise VB will crash. <font size="1" color="#666666">12/30/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip277.html">
Using control panel</a></b><br>
Launch control panel from your program. <font size="1" color="#666666">12/30/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip235.html">
Close a DOS prompt</a></b><br>
Terminate a DOS prompt from your program. <font size="1" color="#666666">12/22/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip233.html">
Close an application</a></b><br>
Close an application using the API. <font size="1" color="#666666">12/22/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip234.html">
Delay a program</a></b><br>
Delay a program using the API. <font size="1" color="#666666">12/22/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip224.html">
Change the computer name</a></b><br>
Change the name of the users computer. <font size="1" color="#666666">12/22/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip230.html">
Find out how long windows has been running</a></b><br>
Get the number of miliseconds that Windows has been running. <font size="1" color="#666666">12/22/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip221.html">
Find out the user name</a></b><br>
Find out the user name of the person currently logged on. <font size="1" color="#666666">12/22/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip264.html">
Show/Hide the taskbar</a></b><br>
Toggle the visibility of the Windows task bar. <font size="1" color="#666666">12/22/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip160.html">
Check if your program is running</a></b><br>
Visual Basic provides you with the App.PrevInst that allows you to detect another instance of your program. <font size="1" color="#666666">12/11/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip183.html">
Create a hotkey for your application</a></b><br>
Allow users to create a hotkey for your application without using hooks or subclassing. <font size="1" color="#666666">12/11/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip184.html">
Change the wallpaper</a></b><br>
Change the Windows wallpaper using the SystemParametersInfo API. <font size="1" color="#666666">12/11/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip186.html">
Get the battery information</a></b><br>
Use the SysInfo control to get info about a laptop battery. <font size="1" color="#666666">12/11/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip191.html">
Get the OS Build & Version</a></b><br>
Get the OS Build number and its version number. <font size="1" color="#666666">12/11/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip190.html">
Get the OS Platform</a></b><br>
Detect whether the user is running Windows 95, Windows NT or another version of Windows. <font size="1" color="#666666">12/11/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip185.html">
Make a memory monitor</a></b><br>
Calculate how many free and used bytes of memory you have. <font size="1" color="#666666">12/11/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip178.html">
Put your program in the system tray</a></b><br>
Add your own icon to the system tray (found next to the clock on the taskbar) <font size="1" color="#666666">12/11/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip179.html">
Shut down windows</a></b><br>
Use the ExitWindowsEx API call to reboot, logoff and shutdown the computer. <font size="1" color="#666666">12/11/1999</font></p><p><b><a href="http://www.vbsquare.com/api/tip187.html">
Start menu speed</a></b><br>
Change the start menu speed. <font size="1" color="#666666">12/11/1999</font></p>
