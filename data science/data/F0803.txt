<B>Text Boxes</b>
</font>
</p>

<p align="left">
<font face="tahoma">
<B>Text boxes</b> are used for user input and to show output. Text boxes are easy, useful, and used often.
</p>
<p align=center>
<img src="http://www.angelfire.com/ny3/visualbasic/images/vbtext.jpg"
</p> 


<p align="left">
<font face="tahoma">
<font color="blue">
Text Box properties:
</font>
</p>

<p align="left">
<font face="tahoma">
<font color="red">
Alignment</font> - Determines whether the text in the text box is centered, toward the right or left<Br>
<Br>
<font color="red">
BorderStyle</font> - Determines what the border of the text box looks like<Br>
<Br>
<font color="red">
Enabled</font> - Determines if the text box is useable or not<Br>
<Br>
<font color="red">
Font</font> - Determines what the font is<Br>
<Br>
<font color="red">
ForeColor</font> - Determines the Text box's font color<Br>
<Br>
<font color="red">
Locked</font> - Determines if the user can write in the text box or not<Br>
<Br>
<font color="red">
MaxLength</font> - Determines how much text the text box can hold, you won't be able to type more then the max value<Br>
<Br>
<font color="red">
Multiline</font> - Determines if the text box has a single line or multiple lines to write in<Br>
<Br>
<font color="red">
PasswordChar</font> - Determines if the text is masked with the character (letter/number/symble) you specify
so no one can see what the user is typing<Br>
<Br>
<font color="red">
ScrollBars</font> - Determines if it has scrollbars so you can scroll<Br>
<Br>
<font color="red">
TabStop</font> - Determines if the control gets focus by pressing tab<Br>
<Br>
<font color="red">
TabIndex</font> - The order the control gets focus<Br>
<Br>
<font color="red">
Text</font> - What the text will be in the text box, which is usually left blank<Br>
</p>
