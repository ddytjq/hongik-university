      <h2><a name="help"></a>Online Help and Documentation</h2>
      <p>&nbsp;&nbsp;&nbsp;<a href="http://www.dinkumware.com/htm_cpl/index.html" target="new">Dinkum 
        Standard C++ Library Reference</a> -- You can also buy an offline copy. 
        This is the first place to go for help with the iostreams classes.</p>
      <p>&nbsp;&nbsp;&nbsp;<a href="http://msdn.microsoft.com/library/" target="new">Microsoft 
        Developer Network Library</a> -- First place to go for help with DirectX 
        or the Win32 API, also has documentaion on the standard C and standard 
        C++ libraries, old and new. Hard to navigate but almost anything you ever 
        would want is here.</p>
      <p>&nbsp;&nbsp;&nbsp;<a href="http://www.sgi.com/tech/stl/" target="new">SGI 
        STL Reference</a> -- Although the previous two sites have documentation 
        on the entire Standard C++ Library, the SGI site provides the clearest 
        documentation, and also provides examples and explaination.</p>

      <h2><a name="Programming">Programming</a></h2>
      <p>&nbsp;&nbsp;&nbsp; <a
href="http://alleg.sourceforge.net/" target="new">Allegro</a> -- This is, IMO, 
        by far the best independent game programming library for DJGPP/MSVC/Windows/GCC/Linux.&nbsp; 
        It has graphics support up to 1600x1200 at any color depth.&nbsp; It also 
        has FLI support to play movies, and SFX and midi support.&nbsp; And for 
        cards without good midi, it has software wavetable emulation, also.&nbsp; 
        Also included are keyboard, joystick, and other very useful routines all 
        written in C and assembly.</p>
      <p>&nbsp;&nbsp;&nbsp;<a
href="http://www.ecst.csuchico.edu/~beej/guide/net/" target="new">Beej's Sockets</a> 
        -- Learn to program internet applications in UNIX BSD socket style.<br>
      </p>
      <p>&nbsp;&nbsp;&nbsp; <a
href="http://www.concentric.net/~Skis/" target="new">BGUI</a> -- The original 
        library which I ported to Allegro WIP. Check out the <a
href="bgui2/index.htm" target="new">BGUI2 page</a> for more information on my 
        port.</p>
      <p>&nbsp;&nbsp;&nbsp; <a
href="http://www.careerexchange.com/" target="new">CareerExchange.com</a> -- CareerExchange.com 
        is a Technical Internet career site providing employment and resume information 
        services to job candidates, employers and recruitment firms.</p>
      <p>&nbsp;&nbsp;&nbsp; <a
href="http://www.delorie.com/djgpp/" target="new">DJGPP</a> -- This is the well-known 
        GNU C++ complier.&nbsp; It is free to download and use, and on the website 
        it A LOT of useful info to get started, as well as links to many other 
        additions and libraries to DJGPP. </p>
      <p>&nbsp;&nbsp;&nbsp; <a
href="http://www.fly3d.com.br/index.htm" target="new">Fly 3D Engine</a> -- Although 
        I haven't used it first-hand, it looks like a very good quality and promising 
        free 3D engine SDK.</p>
      <p>&nbsp;&nbsp;&nbsp; <a
href="http://www.prgjobs.com/" target="new">Jobs!</a> -- You want a job? Jobs 
        for programmers offers the web's top employment site for programmers. 
        Search for jobs in C++, Visual Basic, Java, Oracle, WWW, Cobol, even Telecommute! 
        Thousands of resumes online.</p>
      <p>&nbsp;&nbsp;&nbsp; <a
href="http://www.JustCJobs.com/" target="new">JustCJobs.com</a> -- Part of the 
        jobsite collection from JustComputerJobs.com. A jobsite for IT professionals, 
        but with a twist. Rather than being a single monolithic jobsite that contains 
        hundreds of thousands of postings for every discipline within the technology 
        industry, JustComputerJobs.com is your gateway to a collection of 40 technology 
        niched jobsites.</p>
      <p align="center"><a href="#TOC">Back to Top</a></p>
      <p>&nbsp;&nbsp;&nbsp; <a
href="http://www.ultimapage.f2s.com/" target="new">Ultima Page</a> -- Tchernobill's 
        page which has a lot of content and more on the way. Topics include programming, 
        cryptology, HTML, retro games and RPGs.</p>
      <p>&nbsp;&nbsp;&nbsp; <a
href="http://libsocket.tsx.org/" target="new">Libsocket</a> -- A winsock access 
        library for DJGPP in the middle to later stages of development. Winsock 
        1 support is good although Winsock 2 support is not very workable now.</p>
      <p>&nbsp;&nbsp;&nbsp; <a
href="http://webempire.netfirms.com/" target="new">THe_MaSTeR'S Webempire</a> 
        -- A site cataloging information for beginners such as tutorials, books, 
        links and other resources. If you are a beginner and are looking to search 
        for new sites for help this site should help.</p>
      <p>&nbsp;&nbsp;&nbsp; <a
href="http://www.progsharehouse.com/" target="new">The Programming Sharehouse</a> 
        -- This website offers a large searchable directory of programming tutorials, 
        source code files, programs, tools, links to other sites, games, articles, 
        tips, and FAQs--covering many programming languages and topics. Our site 
        also provides some forums, where you can post programming questions or 
        comments.</p>
      <p>&nbsp;&nbsp;&nbsp; <a
href="http://www.spriteworks.com/index.html" target="new">Spriteworks Developments</a> 
        -- Spriteworks v1.0 is sprite, graphics library for game programmers5,000 
        icons and includes a viewer. I have viewed this and it looks very good 
        and very . Icon bank v2.0 contains professional!</p>
      <p>&nbsp;&nbsp;&nbsp; <a
href="http://members.xoom.com/escp/tut.htm" target="new">QBASIC Programming Tutorial</a> 
        -- from the same author of &quot;The Programming Sharehouse,&quot; a good 
        tutorial for beginning programmers.</p>

      <h2><a name="Emulators">Emulators</a></h2>
      <p>&nbsp;&nbsp;&nbsp; Emulators are computer programs that emulate other 
        computers, like Atari's, Commodores, calculators, and even console systems 
        like Nintendo SNES and Sega Genesis.</p>
      <p>&nbsp;&nbsp;&nbsp; <a href="http://www.snes9x.com/"
target="new">SNES9X</a> -- This emulator has been totally continued back again 
        and is in full support. Super FX support is added and it's much better 
        now, check it out!</p>
      <p>&nbsp;&nbsp;&nbsp; <a
href="http://www.zsnes.com/" target="new">ZSNES</a> -- This is, IMHO, the best 
        SNES emulator out there.&nbsp; It has by far THE BEST graphics and it's 
        speed and sound are MUCH better. Very recently they have done something 
        that people have been waiting a long time for -- now some games actually 
        play BETTER than on the SNES, due to much higher resolutions capable by 
        computers.</p>
