<H3><FONT color=#a0a099>
STL Compatible Base64 Encoder 
</FONT></H3></CENTER>
<HR>
<!-- Author and contact details -->
This article was contributed by <A 
href="mailto:lostd@ukr.net">Konstantin Pilipchuk</A>. 


<P><U>Environment:</U> Any C++ compiler with STL support

<P>This is compatible with STL base64 encoder. 
<P>It can encode/decode data from/to i(o)streams, streambufs, memory buffers, etc. Encoder uses iterator concept. Iterator must overload the *, ==, !=, and prefics ++ operators. 

<P>See encoder using: </p>

<P>1) Declare encoder 

<PRE>#include &lt;base64.hpp&gt;

int _State = 0;
base64&lt;char&gt; encoder; <span class="codeComment">// base64 output/input in chars 
                      // (may be wchar_t, ...) or base64&lt;&gt; encoder;</span>
</PRE>

<P>2) Encode data form cin stream to output file 

<PRE>#include &lt;iostream&gt;
#include &lt;fstream&gt;
<span class="codeComment">
//
   ...
//
// encode from cin
</span>
istream&amp; istr = cin;
istreambuf_iterator&lt;char&gt; _From(istr.rdbuf());
istreambuf_iterator&lt;char&gt; _To(0);
<span class="codeComment">
//_From, _Last, _To may be any iterator class or pointers

// encode to output file</span>
ofstream ostr("test_out.txt");
ostreambuf_iterator&lt;char&gt; _Out(ostr);
<span class="codeComment">
// Encode data with CRLF ta the end of line.
// You can use base64&lt;&gt;::noline() or base64&lt;&gt;::crlfsp()
// or other classes.</span>
encoder.put(_From, _To, _Out, _State, base64&lt;&gt;::crlf());
</PRE>


<P>3) Decode data form file stream to console output stream 

<PRE><span class="codeComment">// decode from file ...</span>
ifstream istr("test_out.txt");
istreambuf_iterator&lt;char&gt; _From(istr.rdbuf());
istreambuf_iterator&lt;char&gt; _To(0);
<span class="codeComment">
// ... to console</span>
ostreambuf_iterator&lt;char&gt; _Out(cout);

encoder.get(_From, _To, _Out, _State);
</PRE>
