<p>Visual C++ is Microsoft’s development solution when high-performance, low-level control, flexibility, and scalability are essential. These strengths have helped Visual C++ become the most popular C++ development tool on the market: today 78 percent of professional C++ developers use Visual C++.  In addition, 70 percent of developers using Visual Studio also use Visual C++, making it a cornerstone of the Microsoft Visual Studio suite of development tools.</p>

<p>Visual C++ 6.0 is designed to make developers more productive in creating the highest-performance and most scalable solutions. Thus, developers must first have a tool that contains the best core development functionality in the industry, from compiler to debugger. Then, to expedite development of these solutions, developer productivity must be as high as possible, making it as easy as possible to exploit the power of Visual C++ and the C++ language without sacrificing control. Within a productive, integrated development environment, developers must be able to write code that takes advantage of the latest technologies and standards from the Internet, emerging and existing platforms, and beyond. Finally, the code must be reusable, scalable, and comprehensive, able to access data, and converse with old and new systems.</p>

<p>Visual C++ 6.0 is available in three editions to provide individuals and development organizations with the functionality they need at the most reasonable price:</p>
<ul>
<li><b>Visual C++, Standard Edition.</b> A full-featured version of Visual C++ ideal for those familiar with languages like Visual Basic, or for C/C++ developers new to the Windows platform and Win32-based programming. Extensive samples, included tutorials from Wrox Press, and thorough documentation ensure that developers at any level can get a jumpstart on Windows and Internet development.</li>
<li><b>Visual C++, Professional Edition.</b> Gives professional developers everything they need to develop full-featured, high-performance applications, including an optimizing compiler, Professional Visual Database Tools, and a new version of InstallShield. Moreover, developers can build sophisticated Windows and Internet applications with full access to the Microsoft Foundation Classes, the most popular class library available, and the Active Template Library, the best choice for fast, lightweight component development.</li>

Everything in the Professional Edition is included as well as Enterprise Visual Database Tools, remote SQL debugging, extended stored procedure creation, Oracle support, and a notable set of enterprise tools: Microsoft SQL Server™ 6.5, Visual SourceSafe, Visual Modeler, Microsoft Transaction Server, SNA Server, AS/400 data access drivers, Internet Information Server 4.0, and more.</li>

<p>Visual C++ has evolved from a development system for 16-bit applications under Windows 3.x into a tool that can target the Windows 95, Windows NT&#174;, Windows 98, and now the Windows CE operating system. Two add-on toolkits for Visual C++ support development for Windows CE:</p>
<ul>
<li><b>Windows CE Toolkit for Visual C++.</b> The Windows CE Toolkit for Visual C++ allows developers to apply their existing programming skills and investment in Visual C++ to easily create Windows CE-based applications for devices ranging from the Auto PC to the Palm-sized PC. After installation, the Windows CE development technology is combined with the Visual C++ IDE.  Development can be performed completely on desktop systems using the provided emulators and custom tools. When the code is ready for the Windows CE-based device, developers can use any of the cross-compilers appropriate for the processor they are targeting: MIPS, PPC, x86 and more.</li>
<li><b>Windows CE Embedded Toolkit for Visual C++.</b> Once installed, the Windows CE Embedded Toolkit for Visual C++ is combined with the Visual C++ IDE.  The tools and technologies required to custom configure Windows CE for the needs of embedded systems are provided in this toolkit, including a ROM image maker and Win32 API access.</li>
