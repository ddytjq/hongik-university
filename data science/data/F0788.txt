<Table width='100%' border=0 cellpadding=5><TR><TD id=RevImage rowspan=6 valign=top width=5% ><img src='netreviewpics\0735711127.jpg' border='0' align=top ></TD><TD  id=RevItemName><FONT color=navy size=6>XML and SQL Server 2000</FONT></TD></TR><TR><TD id=RevItemAuthor >By:  John Griffin</TD></TR><TR><TD id=RevItemISBN>ISBN: 0-7357-1112-7</TD></TR><TR><TD id=RevItemISBN>Publisher: <A href='http://www.newriders.com/books/title.cfm?isbn=0735711127' target='_blank'>New Riders</A></TD></TR><TR><TD id=RevItemISBN>&nbsp;</TD></TR><TR ><TD >&nbsp;</TD></TR></Table><Table width='100%' border=1 cellpadding=5><TR><TD  id=RevTitle><FONT color=navy size=6>To the point but easy to understand</FONT></TD></TR><TR><TD id=RevAuthor >Reviewed By:  <a name=KunjalKaramshi><A href='Reviewer.asp?Rid=5&ISBN=0-7357-1112-7' >Kunjal Karamshi</a></TD></TR><TR><TD id=Review><P>This is a must read book for anyone who wishes to create XML based solutions 
with SQL Server 2000. Not only is it to the point but easy to understand. </P>
<P>As is to be expected, the book begins with a detailed introduction to XML. 
The structure of an XML document is discussed in great depth and serves as a 
great introduction for anyone who has little idea about XML. The book then 
carried on to explain XSLT stylesheets and how they are used to transform XML 
documents into various output formats. I found this part of the book to be 
excellent in that there are clear explanations of the various XSL elements with 
examples to help you understand. The setup of IIS to access SQL Server 2000 is 
also covered in order to help the reader understand what is to come in the 
latter chapters. </P>
<P>The second part of the book looks at how to execute SQL statements over the 
HTTP protocol and how to process the data that is returned. XDR schemas are then 
discussed in detail. There is also a brief discussion of Biztalk and SOAP with 
relevance to XDR schemas. </P>
<P>The final part of the book focuses on the XML extensions present in SQL 
Server 2000. This was quite a technical discussion but it helped me to 
understand the XML extensions better. The appendices contained a brief 
description of IBM’s XSL editor and Microsoft’s SQL Server XML View Mapper which 
I found to be a good introduction to two very useful tools. </P>
<P>Overall, a great book which you cannot go wrong with especially as a quick 
reference. </P></TD></TR></Table>
