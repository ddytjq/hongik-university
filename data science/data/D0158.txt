<h2>How to start programming Java applets?</h2>
Now it is time to take the Contact applet and make it your own.  To start programming in Java you'll need to get the <a href="http://www.javasoft.com:80/products/jdk/1.0.2/index.html"> JDK 1.0.2</a> from JavaSoft.  Follow the installation instructions for your particular hardware,  found on the download page under the heading "How to Download the JDK", or <a href="http://www.javasoft.com:80/products/jdk/1.0.2/installation-win32-x86.html">here </a> for Windows NT and Windows 95 platforms.  Then use your favorite text editor to modify the <a href="Contact.java">Contact.java</a> source code.  You will compile the source using the command "javac Contact.java" which creates the bytecode in "Contact.class".  You will also need to copy the <a href="Contact.html">Contact.html</a> document in order to execute the Java applet.
<p>Once you gain confidence with the Contact applet you will want some more examples of Java source code.  One of the highly regarded reference books on Java programming is the O'Reilly &amp; Associates book "Java in a Nutshell";
here are some <a href="http://www.ora.com/catalog/javanut/examples"> examples</a> from that book.


<h2>Conclusion</h2>
<P>The Contact applet has introduced you to some of the Java statements, the Java objects and their methods. It has shown you how the browser controls the execution of the applet.  It illustrates the simplicity and power of Java. Many say that Java's popularity is based on its "Write Once Run Anywhere<sup><font size=1>(TM)</font></sup>" portability.  However, the ability to easily write interactive graphical user interfaces, animation and interesting multimedia effects has also significantly added to its popularity with programmers around the world.
