	<font size=4><div class=cutline>Product Review:</div></font><BR>
	<font size='-1'><h4>Word in a Box</h4>  Remember the last time you wanted to provide editing functionality in your VB application but gave up? The Rich Text control doesn't cut it, because it doesn't give users all the features they expect from big-brand names like Microsoft Word. What once sufficed in terms of formatting options now also needs OLE embedding, headers and footers, drag and drop, tables, and spell checking. Ready to pull out that VB Control Creator book and start programming 24 hours a day for the next five years? With TX Text Control you can create a "My Word" application, rich with advanced word processing features, in a matter of minutes. What's more, TX Text Control comes with great sample programs, so you don't need to be a VB whiz to get your first application up and running.  <h4>Text Layout</h4>  TX Text Control can be used as an OCX or directly, using the functions exported by its DLL. As an OCX, it can be dropped onto a VB form (or any other ActiveX host) and used almost immediately as a powerful RTF editor. What's more, it accepts images, OLE objects, formatted text, and lots of other features you would expect to find only in the best off-the-shelf word processing packages.
