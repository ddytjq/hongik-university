<h4><a href="comeback.asp?URL=http://msdn.microsoft.com/VBASIC/">Microsoft Visual Basic Home Page</a></h4>

<a name="DEVELOPER"><H2><B>DEVELOPER SITES OFFERING Visual Basic INFO & ARCHIVES</B></h2>

<h4><a href="comeback.asp?URL=http://www.ms-vb.com">Add to Your Programming Knowledge
</a></h4>
This site includes tutorials for both VB and HTML, as well as sample code for VB5 (SP 3), links, and a forum in which to ask your VB questions.

<h4><a href="comeback.asp?URL=http://vbwire.com/advanced/">Advanced Visual Basic</a></h4>
Advanced Visual Basic (AVB) contains information on Visual Basic for the advanced Visual Basic programmer. AVB is not sponsored by Microsoft, a book, or any product. It is a page by a developer written for developers. It assumes you already know about OLE automation, object-orientation, and hopefully have a few years of Visual Basic programming under your belt. The content of these pages is aimed at the experienced Visual Basic programmer and touch on topics found nowhere else on the web. 

<h4><a href="comeback.asp?URL=http://w3.one.net/~dreitz/">ATTAP: All Things To All People - Visual Basic Programming</a></h4>
This site is full of Visual Basic references and links! Go here to download Visual Basic files, get program tips, find a Visual Basic user group or training class, and more!

<h4><a href="comeback.asp?URL=http://www.avbug.com.au/">The Australian Visual Basic Page</a></h4>
This site contains information for all you Visual Basic programmers out there wanting to learn more about Visual Basic. There are heaps of hints, examples and even some unanswered questions lying around in here so, take your time, and have a scrounge around. 

<h4><a href="comeback.asp?URL=http://www.ax2net.com">AX2NET -- ActiveX/COM Component Directory</a></h4>
This site contains links to various categories of components and includes articles and tips for developers of various skill levels.

<h4><a href="comeback.asp?URL=http://www.codearchive.com/">CodeArchive</a><img src="/gifs/new.gif"></h4>

<h4><a href='http://www.codehound.com' target='new'>CodeHound</a></h4>
CodeHound is a powerful Internet search engine designed for VB developers.  If you have a seemingly unanswerable VB question, CodeHound will search web pages, books, ZIP Files and the Microsoft Knowledge Base to find the answers, quickly and without having to sift through non-VB search results.  And CodeHound's free add-in for VB lets you use CodeHound without ever leaving the comforts of VB!

<h4><a href="comeback.asp?URL=http://www.coolstf.com/">COOL.STF</a></h4>
The programmers at COOL.STF.com hand craft fine ActiveX™ Controls and design connectivity software that allows Windows® to connect to Unix and AMOS hosts.  With the idea that Visual Basic developers are often locked out of some of the cooler parts of Windows, the guys at COOL.STF developed the RasDial ActiveX control - probably the first to allow Visual Basic to communicate with RAS, the Remote Access Service.  Their product offering now includeds host connectivity software, additional ActiveX controls, AMOS utilities and some Java applets.

<h4><a href="comeback.asp?URL=http://www.beadsandbaubles.com/coolvb/index.shtml">Cool Visual Basic</a></h4>
This site is for writing really cool visual basic apps. Visual Basic information abounds on the net, but, unfortunately, not much has been written on the cool side of VB. Look on this site for information on GUI design, optimization, DDE, Net-DDE, OLE, Database access, and other cool stuff.
