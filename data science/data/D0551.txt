<SPAN Class="smbl">
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;BorderLayout is another simple layout that
can be easily understood. Basically what BorderLayout does is let you assign
objects to different areas of an applet. There are five areas, North, South,
East, West, and Center. North is the top part, South is the bottom part, East
is the right part, West is the left part, and Center is the middle part. The
size of North, South, East, and West are determined by the number of objects in
those sections. The more objects, the bigger the area. Any area left over is
taken up by the Center section.<br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;You do not always have to add things to all
of the areas, however. As you can see in the applet below, I only used North,
Center, and South. The main way to set up a BorderLayout is, in the init()
method, type:<br><br><i>
setLayout(new BorderLayout());</i><br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;To add objects to the North, South, East,
West, and Center sections, respectively, type:<br><br><i>
add("North",objectnamehere);<br>
add("South",objectnamehere);<br>
add("East",objectnamehere);<br>
add("West",objectnamehere);<br>
add("Center",objectnamehere);</i><br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Obviously in the lines above you have to
replace objectnamehere with the name of the object you want to add. Now that
you understand the basics, check out the applet:<br><br>
<applet code=borderlayout.class height=300 width=300></applet><br><br>
<pre><small>
//-------------------------------//
//Copyright 2000 FreeJavaHelp.com//
//-------------------------------//

//the next line imports the GUI components
import java.awt.*;
//the next line imports the Applet class
import java.applet.Applet;
//the next line make a public class called borderlayout
//that is an applet
public class borderlayout extends Applet{
    //The next line creates a panel called np which will be added to the
    //North part of the border layout
    Panel np=new Panel();
    //The next line creates a panel called cp which will be added to the
    //Center part of the border layout
    Panel cp=new Panel();
    //The next line creates a panel called sp which will be added to the
    //South part of the border layout
    Panel sp=new Panel();
    //the next line creates a button named nbutton which will
    //be added to np
    Button nbutton=new Button("This Button is in the North Section");
    //the next line creates a button named cbutton which will
    //be added to cp
    Button cbutton=new Button("This Button is in the Center Section");
    //the next line creates a button named sbutton which will
    //be added to sp
    Button sbutton=new Button("This Button is in the South Section");
    //the init method is executed when the applet is initialized
    public void init(){
        //the next line sets the layout to a BorderLayout
        setLayout(new BorderLayout());
        //the next line adds np to the "North" part of the BorderLayout
        add("North",np);
        //the next line sets the Background of np to blue
        np.setBackground(Color.blue);
        //the next line adds cp to the "Center" part of the BorderLayout
        add("Center",cp);
        //the next line sets the Background of cp to black
        cp.setBackground(Color.black);
		//the next line adds sp to the "South" part of the BorderLayout
        add("South",sp);
        //the next line sets the Background of sp to red
        sp.setBackground(Color.red);
        //the next line adds nbutton to np
        np.add(nbutton);
        //the next line adds sbutton to sp
        sp.add(sbutton);
        //the next line adds cbutton to cp
        cp.add(cbutton);
    }
}

</small></pre><br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;One thing shown in the applet that I did
not mention yet is the Panel class. I think of Panels almost as mini-applets.
You can add them to a grid in a GridLayout or one of the Sections in a
BorderLayout. The nice thing about Panels is that you can add objects to them.
Not only that, but you can also use panelname.setLayout(new BorderLayout()); to
make a BorderLayout within a Panel. You can also do panelname.setLayout(new
GridLayout(rows,columns)); to create a GridLayout on a Panel.<br><br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Panels allow users to have more than one
layout in a given applet. I could add a Panel to a grid, then make another
GridLayout or a BorderLayout on that Panel if I wanted to. I could add a Panel
to a section of a BorderLayout and then make another BorderLayout or a
GridLayout within that Panel. Panels are <b>very</b> useful. Learn them. Use
them.
