// Invoke it once to activate the clock; it will call itself from then on.
function display_time_in_status_line()
{
    var d = new Date();                // get current time;
    var h = d.getHours();              // extract hours: 0 to 23
    var m = d.getMinutes();            // extract minutes: 0 to 59
    var ampm = (h >= 12)?"PM":"AM";    // is it am or pm?
    if (h > 12) h -= 12;               // convert 24-hour format to 12-hour
    if (h == 0) h = 12;                // convert 0 o'clock to midnight
    if (m < 10) m = "0" + m;           // convert 0 minutes to 00 minutes, etc.
                                       // put it all together
    var t = 'The current time is ' + h + ':' + m + ' ' + ampm;

    defaultStatus = t;                 // display it in the status line

    // arrange to do it all again in 1 minute.  
    setTimeout("display_time_in_status_line()", 60000); // 60000 ms in 1 minute
}

<H5>
These web pages are for people interested in foreign languages, computers,
and certain mainly US organizations. Links to various sites on these
topics are provided here, in that general order. Language sites are divided 
into general/multilingual resources; general/multilingual dictionaries;
specific languages; and machine translation &amp; natural language processing.
I'm also offering several resources of my own creation. 
<A HREF="http://www.dma.org/~deand/potm.html">&quot;C&quot; Program of the
Month</A> is the source code for a &quot;C&quot; language computer program
that I'm making freely available for personal use. No warranty or guarantee
is offered or implied. Feel free to use at your own risk. I'll place a new
program here every month. The source code will also often appear in the
USENET group alt.sources.<P> 

Also, I'm providing listings of a Vietnamese-English education-related
vocabulary glossary that I've put together. Here are links to several formats:
<OL>
<LI><A HREF="http://www.dma.org/~deand/educatn.html"
>Vietnamese Education Terms (VIQR text format)</A>
<LI><A HREF="http://www.dma.org/~deand/educatv.html"
>Vietnamese Education Terms (VISCII format)</A> 
<LI><A HREF="http://www.vietmate.com/scripts/viv.pl?from=VISCII&to=VPS&site=http://www.dma.org/~deand/educatv.html"
>Vietnamese Education Terms (VPS format)</A>
<LI><A HREF="http://www.vietmate.com/scripts/viv.pl?from=VISCII&to=VNI&site=http://www.dma.org/~deand/educatv.html"
>Vietnamese Education Terms (VNI format)</A>
<LI><A HREF="http://www.vietmate.com/scripts/viv.pl?from=VISCII&to=VNU&site=http://www.dma.org/~deand/educatv.html"
>Vietnamese Education Terms (VNU format)</A>
</OL>

<A HREF="http://www.vietmate.com/"
>VietMate</A> generates the VPS, VNI, &amp; VNU versions. This glossary will be periodically expanded. Please send me suggested
updates or additions, if you have any. However, I have very limited time for requests
for information.
