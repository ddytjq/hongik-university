<center><strong><font size="+1" color="red">Neighbor TreeMaker</font></strong><P>
<TABLE width=95% cellpadding=10><TR><TD>
<strong>Neighbor TreeMaker</strong> takes a sequence alignment,
	converts it to <A HREF="http://evolution.genetics.washington.edu/phylip.html" target=_top>PHYLIP</a> format, runs it through the PHYLIP programs <strong>Dnadist</strong>
(Distance Matrix program), and <strong>Neighbor</strong> (treefile generator),
then displays a tree.    
<P>
The <strong>Dnadist</strong> program reads in  nucleotide  sequences  and  writes  an  output
file containing  the  distance  matrix. PHYLIP gives options concerning
the model used to calculate distances. This interface uses ML, the model used
in PHYLIP's maximum likelihood phylogeny program DNAML. This model
incorporates  different  rates  of transition and transversion, and also
allows for different frequencies of the four nucleotides.
<font size="-1"><A HREF="JoeCredit.html">PHYLIP copyright/reference info</A>.</font>
</TD></TR><TR><TD>
<I><font color="red"><B>Disclaimer.</B> This interface only offers very basic, 'quick-and-dirty' phylogenetic analysis.
More in-depth analysis is usually needed. For more information see the <A HREF="../../TREE_TUTORIAL/Tree-tutorial.html">Treemaker Tutorial</A>. </font></I>
</TD></TR></TABLE></center>
