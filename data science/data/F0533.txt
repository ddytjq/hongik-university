    <td valign="top" width="100%"><font size="2" face="Arial"><strong>How to display the
    &quot;Browse for Folder&quot; dialog box</strong> <br>
    <br>
    An example that shows how to display the &quot;Browse for Folder&quot; dialog and return a
    user selected folder.<br>
    <br>
    The zip file Includes translated definitions and documented source for the Browse dialog's
    C structures:<br>
    <br>
    BROWSEINFO</font><ul>
      <li><font size="2" face="Arial">ITEMIDLIST and </font></li>
      <li><font size="2" face="Arial">SHITEMID</font></li>
    </ul>
    <p><font size="2" face="Arial">and the API calls: </font><ul>
      <li><font size="2" face="Arial">SHBrowseForFolder()</font></li>
      <li><font size="2" face="Arial">SHGetPathFromIDList() and </font></li>
      <li><font size="2" face="Arial">SHGetSpecialFolderLocation(). </font></li>
    </ul>
    <p><font size="2" face="Arial">The demo also shows two of the many ways to call the
    SHGetFileInfo API.<br>
    <br>
    This is the 2nd update to the original version released to the public domain on 3/24/96.
    The code has been made more readable by simplifying the procedures. Updates include
    checkbox options for Browse dialog return types and demonstrates how to retrieve the
    selected folder's icon from the return value.<br>
    <br>
    For a free and full featured Browse dialog implementation wrapped into a Visual Basic
    ActiveX server, visit the Common Controls Replacement Project (CCRP) site at <a href="http://www.mvps.org/ccrp">http://www.mvps.org/ccrp</a>.<br>
    <br>
