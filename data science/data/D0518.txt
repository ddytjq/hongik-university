<H2 ALIGN="CENTER">Companies partner to enable Web services interaction </H2><P>
<strong>July 12</strong> &#8212; BEA Systems, Intalio, SAP AG, and Sun Microsystems have published an  XML-based Web services interface that bridges the gap between Web services and business process management. The Web Services Choreography Interface (WSCI) enables Web services to interact with each other, fulfilling business process needs such as delivering automated, application-to-application collaboration. <P>
<a href="http://dev2dev.bea.com/techtrack/wsci.jsp">http://dev2dev.bea.com/techtrack/wsci.jsp</a> <P>
<a href="http://titan.intalio.com:8080/intalio/wsci/">http://titan.intalio.com:8080/intalio/wsci</a> <P>
<a href="http://ifr.sap.com/wsci/">http://ifr.sap.com/wsci</a> <P>
<a href="http://wwws.sun.com/software/xml/">http://wwws.sun.com/software/xml</a> <P>
<A HREF="#index">Back to index</A><P>
<A NAME="1"</A><P>
<H2 ALIGN="CENTER">Zion launches IM products </H2><P>
<strong>July 12</strong> &#8212; Zion Software has introduced JSuperChat, a cross-service, cross-platform instant-messaging (IM) client for Java 2 Platform, Standard Edition (J2SE)-compliant PCs and devices. The company has also announced JMessageServer, a presence and IM real-time communications platform available for Java 1.4 computing environments.
<P>
<a href="http://www.zion.com/">http://www.zion.com</a> <P>
<A HREF="#index">Back to index</A><P>
<A NAME="2"</A><P>
<H2 ALIGN="CENTER">SolarMetric teams up with Borland </H2><P>
<strong>July 12</strong> &#8212; As a new participant in Borland Software's partner program, SolarMetric will include Kodo JDO (Java Data Objects) for object/relational mapping in the Borland JBuilder 7.0 companion CD.  Kodo JDO relieves developers of hand coding data access. <P>
<a href="http://www.solarmetric.com/">http://www.solarmetric.com</a> <P>
<a href="http://www.borland.com/jbuilder/">http://www.borland.com/jbuilder</a> <P>
<A HREF="#index">Back to index</A><P>
<A NAME="3"</A><P>
<H2 ALIGN="CENTER">Nazomi licenses Sun's CLDC and J2ME </H2><P>
<strong>July 12</strong> &#8212; Nazomi Communications will incorporate Sun Microsystems' Connected Limited Device Configuration (CLDC) and Mobile Information Device Profile (MIDP) into its current JA108 Universal Acceleration chip for Java architectures and multimedia, and its JStar and JSmart acceleration technologies. Nazomi's JA108, JStar, and JSmart solutions are optimized for mobile wireless environments.
<P>
<a href="http://www.nazomi.com/ ">http://www.nazomi.com</a> <P>
