<CENTER><H3><FONT COLOR="#AOAO99">
Static Parameter Class
</FONT></H4></CENTER><HR>

<!-- Author and contact details -->

This article was contributed by
<A HREF="mailto:Kerry@Barnes.net">Kerry Barnes</A>.

<!-- For which environment is this code specific??? -->
<p><u>Environment:</u> Should run under any OS

<P><H3>The problem:</H3>
The other day I was caught with a dilemma.  You see I had a group of
several threads that required access to parameters that are currently
stored in a file.  My first instinct was to just make a class to load
the data so that each thread could instantiate a copy of this.  This
however just sort of rubbed me the wrong way.  There had to be a
better way.</P>
<P><H3>My Solution:</H3>
I came up with an idea, which I don't think is all that new, but it
did catch a few people off guard.  I put a little twist on the old
Singleton object.  You see I decided to create a Singleton object with
a series of static member functions and member variables.  This way I
never have to either instantiate or cleanup the object.  And all of my
threads have access to the data without obtaining a referance to it,
because all of the data is availabile through static methods.</P>
<P>My example has all of the thread synchronization removed for clarity,
also I used only the Standard Template Library objects to make
this class portable.</P>

#include &lt;iostream>
using namespace std;

#include "Parameters.h"

void main(int argc, char* argv[])
{
  <span class="codeComment">//**************************************************************</span>
  <span class="codeComment">//* The parameters are accessed without an instance of the class</span>
  <span class="codeComment">//**************************************************************</span>
	cout << "Name: " << CParameters::getName().c_str() << endl;
	cout << "Version " << CParameters::getVersion() << endl;
}
</PRE>
<H4>The Header File:</H4>
<PRE>
class CParameters
{
public:
	static string getName();
	static int getVersion();
	static void load(string fileName);
	virtual ~CParameters();
private:
	CParameters();
	static string m_ProgramName;
	static int m_Version;
	static CParameters instance;
};
</PRE>
<H4>The .cpp File</H4>
<PRE>
#include &lt;string>
#include &lt;fstream>
using namespace std;

#include "Parameters.h"

<span class="codeComment">////////////////////////////////////////////////////////////////</span>
<span class="codeComment">// Construction/Destruction</span>
<span class="codeComment">////////////////////////////////////////////////////////////////</span>

<span class="codeComment">//**************************************************************</span>
<span class="codeComment">//* Here I set default values to my static variables</span>
<span class="codeComment">//**************************************************************</span>
string CParameters::m_ProgramName = "Undefined";
int CParameters::m_Version = 0;

<span class="codeComment">//**************************************************************</span>
<span class="codeComment">//* Here I invoke my constructor and load my variables</span>
<span class="codeComment">//**************************************************************</span>
CParameters CParameters::instance = CParameters();

<span class="codeComment">//**************************************************************</span>
<span class="codeComment">//* Since this is a private method it can only be invoked by the </span>
<span class="codeComment">//* previous line</span>
<span class="codeComment">//**************************************************************</span>
CParameters::CParameters()
{
	load("default.dat");
}

<span class="codeComment">//**************************************************************</span>
<span class="codeComment">//* No cleanup required since we didn't make anything on the heap</span>
<span class="codeComment">//**************************************************************</span>
CParameters::~CParameters()
{

}

<span class="codeComment">//**************************************************************</span>
<span class="codeComment">//* I put this in to change the values after the program has </span>
<span class="codeComment">//* started.  Mind you in my case this was unnecessary.</span>
<span class="codeComment">//**************************************************************</span>
void CParameters::load(string fileName)
{
	<span class="codeComment">// Open the parameters file</span>
  ifstream in(fileName.c_str(), ios::in); 

	<span class="codeComment">// Did it open?</span>
  if (in)                                 
	{
		<span class="codeComment">// Yes, load up our data</span>
    in >> m_Version >> m_ProgramName;   

		<span class="codeComment">// now, close the file</span>
    in.close();                         
	}
}

<span class="codeComment">//**************************************************************</span>
<span class="codeComment">//* Access methods incase you don't want to make your static </span>
<span class="codeComment">//* variable public</span>
<span class="codeComment">//**************************************************************</span>
int CParameters::getVersion()
{
	return m_Version;
}

string CParameters::getName()
{
	return m_ProgramName.c_str();
}
