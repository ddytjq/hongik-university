			<TD COLSPAN=3 WIDTH=86% BGCOLOR="#ffffff">
				<P>&nbsp;<BR>This site is dedicated to advancing the
				understanding of Borland's C++ Builder, the Borland Visual
				Component Library (VCL), and related Microsoft Windows API
				issues.</P>
				<P>In particular, I have spent an inexcusable amount of time
				working through many of the problems associated with Rich Edit
				controls.&nbsp; Here you will find information on using OLE with
				Rich Edit controls, using the relatively unknown Rich Edit
				version 2.0, using the even newer Rich Edit version 3.0, answers
				to frequently asked questions about Rich Edit controls, and other
				information that I hope you will find useful.</P>
				<P>To save others a little effort, I have made most of the code
				that I use available for downloading.&nbsp; Unless otherwise
				stated, you may use it freely with the following limitation:&nbsp;
				You may not distribute the code to others.&nbsp; Please send them
				here if they are interested.&nbsp; If you find the code useful, a
				little credit would be appreciated, but is not required.<BR>&nbsp;</P>

		<TR>
			<TD COLSPAN=3 WIDTH=86% VALIGN=TOP>
				<P>The TaeRichEdit Component is a full-featured TRichEdit
				analogue.&nbsp; It supports Rich Edit 2.0 &amp; 3.0 character and
				paragraph formatting; provides OLE support (including embedded
				graphics); expands printing options to include headers, footers,
				and border lines; and much more.&nbsp; The Borland C++ Builder
				(no Delphi) source is included.&nbsp; The component is free for
				non-commercial use.<BR>&nbsp;</P>
